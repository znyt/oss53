　　“我道心会坏。”叶少阳实话实说。
<br/>　　“不光是你的道心。”林三生道，“星月奴这一手太狠了，你不是什么天选之子吗，你是人类法术界复兴的支柱，一旦你背上杀死长辈的罪名，你就成了法术界最恶的人，这污名一生都洗不掉，你还怎么样领导法术界，甚至于茅山都会受到连累，两大派互相攻伐，人间法术界再无团结……”
<br/>　　说到这，林三生自己也倒吸了一口凉气，纯粹从计谋的角度说，星月奴这一手，真的是把阴谋诡计发挥到极致了。
<br/>　　林三生突然意识到，星月奴一直都在韬光养晦，一直以来，她给人的感觉是虽然谋略不错，但是性格暴躁，感情用事，喜怒无常，但仅仅通过眼前这件事来看……她实在是谋略过人。
<br/>　　叶少阳失魂落魄地站了半天，喃喃说道：“那怎么办，不动手的话，我们都得在这困着，永远都出不去……”
<br/>　　这简直就是个死结！
<br/>　　而且，星月奴眼看着就要完成修炼，一旦完成，她就证道了，这世界又是她开创的，规则是她定的，到时候她随便拉一批手下过来，分分钟吊打他们这些人啊。
<br/>　　叶少阳咬了咬牙，说道：“没办法了，再说现在不光是我们自己的事，这么多人被困在这，如果不赶紧破阵，大家都得死，所以……”
<br/>　　“所以什么？”
<br/>　　叶少阳耸了耸肩。
<br/>　　林三生道：“少阳，如果你想道心无愧，如果你真是为了众生，你就换个角度想一下，如果九星阁在我身体里，或者是小马、小青他们身上，你能不能做出这个决定……如果你做不到，那你将来必然会被愧疚纠缠到死！”
<br/>　　叶少阳好像挨雷劈了一下，怔在当场，半晌，颓然地吐出了一口气，冲林三生冷笑道：“军师，我有时候挺不爽你的，你这人，能把什么都说尽，让人无处可藏，想找借口都找不到。”
<br/>　　刚他按照林三生的提示，试着想了一下，做不到，真的做不到……
<br/>　　林三生笑道：“你应该庆幸有我这么一个兄弟，有些事一旦做了，就再回不了头了，但只要没做，你就还有选择的机会。”
<br/>　　“可是我现在真的没法选。”
<br/>　　“那就不选。”
<br/>　　“不选？”
<br/>　　林三生拍了拍他的肩膀，“也许会有人帮你选。”
<br/>　　“谁？”
<br/>　　林三生没有回答，然后安慰他，自己还要再调查一下，确定是不是怀疑错了，于是先让叶少阳回去休息。
<br/>　　自己也的确需要想想，林三生已经说得那么透了，不能再让他替自己选择，叶少阳回到房间，把自己关在里头，静下心来，好好地从头思考起来……
<br/>　　林三生仍旧站在湖边，一个人想着心事，等了一会，瓜瓜和包子过来，没找到叶少阳，问他去哪了。
<br/>　　“他有事情要思考，你们自己玩一会去，不要去打扰他。”
<br/>　　瓜瓜吐了吐舌头，道：“那个，我们是来汇报的，刚有人在不远处偷听……”
<br/>　　“我知道，你们去玩吧。”林三生打发他们离开，一个人在湖边站了许久，回到后面的宫殿去。
<br/>　　宫殿很大，房子也很多，都是给太监宫女什么的住的，之前全被他赶了出去，住进了自己这帮人。
<br/>　　林三生想来想去，也不知道那人房间在哪儿，于是回到自己的房间。
<br/>　　门外站着士兵，看他回来，一个去点香，另一个去准备云雾茶，恭恭敬敬地交到他手中。
<br/>　　“你们去外边吧，一会有人来找我，你们守在外门那里，除了第一个来找我的，任何人都别放进来。”
<br/>　　两个士兵一头雾水，但还是乖乖离开了。
<br/>　　林三生盘膝坐在床上，一只手托着香炉，一只手托着云雾茶，静静发呆，等了有半小时左右，门外果然响起脚步声，然后是敲门。
<br/>　　“进来吧。”
<br/>　　门被推开，进来的是龙虎山弟子曲波，林三生一点也不意外，从床上下来，道：“你怎么来了。”
<br/>　　曲波恭敬地笑了笑，道：“大元帅好啊。”
<br/>　　林三生笑道：“别客气，什么事找我。”
<br/>　　“那个，我师父让我来的，说他找到了一坛好酒，想找你去聊聊。”
<br/>　　林三生跟着去了。
<br/>　　到了张无生房间外面，曲波敲了敲门，然后推开门，表示林大帅请来了。
<br/>　　“林大帅请啊，曲波你也进来吧。”
<br/>　　曲波有点惊讶，只好跟了进去。
<br/>　　张无生真的在喝酒，面前摆着一个酒坛子，还有几盘菜，只见他双手抱着个大肘子，正在啃着，油都沾到胡须上了，张无生完全无视，自顾自地吃着，指着面前一个蒲团，让林三生坐。
<br/>　　林三生坐下来，望着张无生，笑道：“祖师好胃口。”
<br/>　　“我敢保证，这些东西都是从人间弄来的，口味很正，酒也很好，我让曲波找内务府的人翻出来的，好东西。”
<br/>　　“可惜我是鬼，喝不了这个。”
<br/>　　“所以你生前应该多喝点了，死了变成鬼，很多东西都享受不到了。”
<br/>　　“你现在对我说这个，太晚了，我死了几百年了。”
<br/>　　“对我来说，不晚啊。”
<br/>　　张无生大笑。
<br/>　　“怎么样，你羡慕了吧。”
<br/>　　林三生不说话。
<br/>　　曲波站在他身后，一脸的茫然，不知道他们在聊什么。
<br/>　　张无生大口吃肉，一边含混不清地说道：“你知道吗，我这人很会享受，很多戒律我都不放在眼里，但我修的是清静道，我一直不吃肉，但我一直没忘记吃肉的感觉……”说着用力咬了一口肉在嘴里，大嚼起来。
<br/>　　张无生一边吃着，一边让曲波给自己泡了杯茶过来，曲波恭恭敬敬地把茶递过去，有点尴尬地提出要出去守着，毕竟大佬对话，他这身份留在这听不合适。
<br/>　　林三生笑道：“你可不能走，将来只怕有很多事我还要指望你。”
<br/>　　曲波愣住，不明白他为什么这么说。林三生让他好好呆着，喝了茶，吃东西也没那么猛了，慢慢地在喝酒，也不说话。
<br/>　　林三生老实坐在他对面，也一样不开口。
