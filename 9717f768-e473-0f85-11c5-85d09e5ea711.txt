        看完这条信息后，连星儿便没再回复了。
<br/>
<br/>        对于这个师父，连星儿还是有种说不出来的陌生感和奇怪感，连带着游戏她都不想玩了。
<br/>
<br/>        到底是哪里出了问题？
<br/>
<br/>        为什么现实世界里的师父，跟网络世界里的师父给她的感觉不一样。
<br/>
<br/>        连星儿有些迷茫地挠了挠头发，直到上课铃声响起，她才收好手机，不再去想这个问题，而是拿出课本准备听课。
<br/>
<br/>        只不过，这听课的时候，她的思绪总会不经意间地开起小差，而且只会想到一个人，那就是明司寒那家伙。
<br/>
<br/>        想到他抱着她的感觉，想到他牵着她的感觉，还想到他身上那……六块腹肌？！
<br/>
<br/>        说到六块腹肌，到了下午没课的时候，学校的健身馆里，张子晋、邢乐游和仲正浩，被明司寒叫了过来一起陪练。
<br/>
<br/>        “什么？六块腹肌不够？要练出八块来！”邢乐游一脸惊悚地看着明司寒。
<br/>
<br/>        而张子晋和仲正浩正躺在仰卧起坐机上，表情扭曲，硬是起不来。
<br/>
<br/>        “寒哥，你是不是为了那个连星儿才想要练八块腹肌的啊？”张子晋一脸苦瓜相。
<br/>
<br/>        仲正浩双手抱头，终于坐起了身，随声附和：“这还用问？自从那个连星儿出现后，咱们寒哥每天都在中毒。”
<br/>
<br/>        “阿寒，我跟你说啊！女人的话，不可信！而且，这女人吧！比男人还好色！”邢乐游举着哑铃，一本正经地说，“她要求你八块腹肌，那你怎么不去要求她胸大、臀翘、腰细、腿瘦？”
<br/>
<br/>        明司寒此刻也正做仰卧起坐，比起张子晋和仲正浩，他的往复动作要快许多。
<br/>
<br/>        但邢乐游的这句话，让明司寒突然坐起身停了下来：“费什么话，赶紧练！”
<br/>
<br/>        三人顿时语塞。
<br/>
<br/>        张子晋不禁感慨：“咱们寒哥恋爱，我们三个兄弟跟着遭殃。”
<br/>
<br/>        “明司寒，跟我们比一场篮球赛，如何？”就在这时，一个又高又壮的男生，带着三个小跟班，哦，不是，是三个室友，走到了他们四人的身旁问。
<br/>
<br/>        明司寒未吭声，邢乐游便直接答应了：“好啊！什么时候？”
<br/>
<br/>        “待会三点半，体育馆。”壮男说道。
<br/>
<br/>        邢乐游做了一个“OK”的手势。
<br/>
<br/>        随后，那壮男便带着三个小跟班离开了。
<br/>
<br/>        张子晋瘪了下嘴，问道：“你这么快就答应他们做什么？”
<br/>
<br/>        “你看不懂吗？我们要是不答应，他们就会说我们怕他们！”邢乐游振振有词道，又看了明司寒一眼，“是吧？阿寒？”
<br/>
<br/>        “你都答应了，我们只能硬着头皮上了。”明司寒勾了勾唇。
<br/>
<br/>        下午三点，连星儿她们有体育课要上。
<br/>
<br/>        如果不是看这群女学生们一个个娇滴滴的，女体育老师也不会选在体育馆里上体育课。
<br/>
<br/>        半个小时的课程上完后，女体育老师便宣布大家自由活动，然后下课前五分钟集合点名。
<br/>
<br/>        而正在这个时候，篮球场那边忽然热闹了起来。
<br/>
<br/>        连星儿被顾千凝挽着手臂，跟着路娅玟和赵安彤两个人一起去了篮球场那边。
<br/>
<br/>        场边围着的，大多数都是女生，但也有部分男生在捧场。
<br/>
<br/>        顾千凝牵着连星儿的手，好不容易挤到前排，结果被一个篮球砸到额头上。
<br/>
<br/>        球场上的张子晋见状，连忙疾步跑过来关问：“喂，千凝，你有没有事？”
<br/>
<br/>        连星儿连忙替顾千凝揉着额头，只见顾千凝对张子晋摆手道：“没事没事！”
<br/>
<br/>        “那我待会打完球后，请你喝奶茶赔礼道歉！”张子晋捡起球，说完后又返回到球场上，将球扔向明司寒。
<br/>
<br/>        而作为投手的明司寒，无意间在场外的人群里看到连星儿的时候，因为走神未接球，球被对手抢走，因此而错过了两分。
<br/>
<br/>        “喂！阿寒！你怎么搞得哦！”邢乐游抹了把汗，抱怨道。
<br/>
<br/>        明司寒回过神来，淡定自若：“别急！刚刚只是热身活动！现在，我们来猛的！”
<br/>
<br/>        四个人正式进入状态，默契地传球，有效的假动作，两分、三分轮着拿。
<br/>
<br/>        “星儿，你知道吗？男生在球场上打球的时候，是最帅的！”顾千凝兴奋不已地看着张子晋。
<br/>
<br/>        一旁的路娅玟和赵安彤，目光都在跟着自己心仪的男生走。
<br/>
<br/>        而连星儿的焦点，也自然而然地落在了明司寒的身上。
<br/>
<br/>        是从什么时候开始，她在不知不觉中，开始悄悄地关注起明司寒这家伙来？
<br/>
<br/>        只见他和张子晋、邢乐游还有仲正浩四个人相互之间的配合十分默契，很少不给对手碰到篮球的机会。
<br/>
<br/>        就那么十几分钟下来，他们一个个汗流浃背，身上的球服前后都湿了一大片。
