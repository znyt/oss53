　　“桀诺老爷让您去会客厅，有贵客拜访。”梧桐恭敬的站在苏重身前轻声道。
<br/>　　“嗯，我知道了。”苏重睁开眼睛，眼里闪过一丝疲惫。自从魔龙六段巅峰，魔龙意志突然变的狂暴。
<br/>　　压制暴虐的魔龙意志，消耗了他大量心神。强悍的身体，让他精力充沛。意志的碰撞，却让苏重精神疲惫。矛盾的感官，让苏重烦躁不已。
<br/>　　“黑暗奏鸣曲、魔龙意志，不是自主修炼来的力量，果然存在巨大隐患。”苏重愁眉不展，已感觉到不妥。
<br/>　　多日来思考解决办法，却毫无所获。但他有预感，等到魔龙七段大成，他必然会发生一次翻天覆地的蜕变。
<br/>　　到那时，他将会和魔龙意志决出胜负。
<br/>　　他胜，则完全融合魔龙意志，彻底改造自身。他败，则被魔龙意志吞噬心神，丧失自我。成为一个只知杀戮的魔王！
<br/>　　感受着苏重身上不时爆发的凶残气息，梧桐越发恭敬。苏重的实力，让他无法在以小辈的眼光看待。
<br/>　　只不过，这种不受控制的暴虐气势……梧桐眼中闪过不安。
<br/>　　“能这么快获得这一身恐怖实力，代价肯定非常大。这种不稳定，难道是誓约与制约的反噬？”他和桀诺、席巴保持着同一意见，认为苏重是通过誓约制约方式获得力量。
<br/>　　虽然对目前状况有所预料，但仍然忍不住担忧。
<br/>　　走进会客厅，苏重一眼看到一身运动装，神采奕奕的尼特罗。
<br/>　　察觉到尼特罗提聚至巅峰的念力，苏重心底升起一股威严被冒犯的感觉。卧榻之旁岂容他人鼾睡，这是一种本能的领地意识。强忍住攻击欲望，苏重皱着眉头道：“尼特罗，你来干什么？”
<br/>　　“当然是给你送报酬的，难道不欢迎。”尼特罗笑呵呵调侃。
<br/>　　“废话少说，到底找我干什么！”苏重眉头皱的更紧。
<br/>　　尼特罗眼睛眯起，凝重的和桀诺对视一眼。收起嬉笑直言道：“我打算和蚁王决战。需要有人帮忙扫清嵌合蚁护卫，断绝后患。”
<br/>　　明理有帕里斯通这个内奸，暗地里有五大帝国掣肘，猎人协会很难派出精英，帮他消灭嵌合蚁。
<br/>　　尼特罗去过黑暗大陆，清楚哪里的恐怖之处。各种诡异的生命，单体可能不强，却有着极其强悍的生存能力。
<br/>　　这种生存能力，往往又是在争夺其他生物的生存空间。
<br/>　　如果任由嵌合蚁发展，人类社会必将造成巨大损失。即便有脏弹大范围清洗，但肯定会造成人类社会元气大伤。
<br/>　　历史证明，每一次黑暗大陆灾祸，一旦处理不当，必然会造成社会退步！尼特罗不允许这种事情发生。
<br/>　　“决战蚁王？”压下魔龙意志的翻腾，苏重顿时恢复冷静。
<br/>　　沉吟半晌，苏重决然道：“我来消灭蚁王，你去清扫嵌合蚁残余势力。”
<br/>　　“让你的人找到蚁王位置，明天一早立刻出发！”然后不等尼特罗反对，直接转身就走。
<br/>　　魔龙七段到了瓶颈，魔龙意志时刻作乱。苏重要利用和蚁王决战的机会，彻底完成蜕变！
<br/>　　至于蚁王的实力，苏重并不担心，反而非常期待。
<br/>　　嵌合蚁实在是一个优秀的种族。一旦让蚁王知道自己不再无敌，必然会激起对方的雄心。
<br/>　　绝不会像原著中一样，因为实力强大没有敌手，进而不再刻意提升力量。只是一味的躲在皇宫里下棋，以此证明自己的种族优越性。
<br/>　　现在他刚出生就差点被杀。苏重相信，蚁王一定会争分夺秒、不择手段的提升实力。
<br/>　　因为这个世界不是他想象中的世界。想要带领嵌合蚁崛起，蚁王必须搬开苏重这块绊脚石。
<br/>　　“糜稽有问题。”尼特罗脸色凝重，他实力强大，清晰感觉到苏重精神上的暴虐。在念力感知中，苏重好似一头披着人皮的魔兽。精神层面全然是霸道凶残！
<br/>　　“力量越大，付出的代价就越大，誓约与制约，不是那么好订的。”桀诺揍敌客背着手，惯有的死鱼眼中，闪过一丝担忧，旋即消失不见。取而代之的是如刀一般的冷酷：“路是自己选的，结果要他自己去承担。揍敌客需要的绵延长存，不是一时称霸。”
<br/>　　这是当做弃子，放任不管啦？尼特罗心里一叹，却保持沉默。就像揍敌客不在乎他的处境，无视纠纷倾轧，只是单纯的收钱办事一样。他也不会插手，这是揍敌客的家事。
<br/>　　反倒是苏重的状态引起了他的主意。誓约与制约？那可不太像。反而更像什么恐怖东西，藏在对方体内！
<br/>　　……
<br/>　　东拓国边境城市，这里地处交通要道经济繁荣。
<br/>　　往日里人流不息，但现在整个城市一片寂静，一点儿而声音也没有，好似一座鬼城！
<br/>　　“蚁王通过各种手段，聚集东拓国全境国民，这座城市，半个月前就已经清空！”尼特罗皱着眉头道。
<br/>　　尼特罗和苏重两人，正站在一座钟塔上。它是整个城市最高建筑，站在这里，可以俯瞰城市全貌。
<br/>　　蚁王聚集人类，是为了筛选优秀个体，改造成强力战斗兵蚁。
<br/>　　这种涉及基因层面的改造非常残酷，甚至是恐怖的万不存一！
<br/>　　一旦蚁王的计划成功，整个东拓国，会瞬间成为历史，整个从大陆版图上抹去！这就是黑暗大陆灾祸的可怕之处！
<br/>　　“会长。”两人身边墙壁上，突兀出现一个黑色洞窟。诺布从里面跳出，一个翻身站稳在屋檐边缘。
<br/>　　“他是诺布，你见过。”尼特罗给苏重介绍：“东拓国的情报，就是他和几位优秀猎人调查提供的，哦，他们来了。”
<br/>　　苏重转头看向从屋顶飞奔过来的几个人，眉头顿时皱了起来。
<br/>　　“奇犽，你为什么会在这里。”意料之外又在情理之中的，苏重见到了奇犽。
<br/>　　“立刻回枯枯戮山，这里的事情，你不能掺和！”苏重斩钉截铁道。
<br/>　　见到苏重，刚刚还有些发憷的奇犽。被这一声呵斥，激起了火气：“我的事，不用你管！”
<br/>　　嗯！眼睛猛然一瞪，一股强悍的精神威压，骤然爆发。
<br/>　　整个世界好似变成了寂静的苍白画，奇犽脑海一片空白，瞬间失去思考能力。
<br/>　　苏重眉头微皱，冷哼一声，压下心中暴虐，收摄精神冲击。
<br/>　　好半晌，奇犽才反应过来，脸色惨白。刚才瞬间，他甚至以为自己已经死了。即便是在爷爷桀诺身上，他都没感觉到过这种压力。
<br/>　　“走吧，你们找的凯特没死。他有一项特殊能力，能够转世重生。去找那些投降的嵌合蚁吧。”苏重尽量放缓语气道，对着依旧倔强的奇犽和小杰道。
<br/>　　因为苏重的原因，嵌合蚁发展情势大变。东拓国成了一个恐怖漩涡，不知多少利益纠葛在其中。奇犽、小杰两人一头扎进去，渣子都不会剩下！
