　　　　今天是蛇王节，可谓就是涂小安自己的节日，长蛇形队伍最前方的蛇王雕像虽然还没有揭幕，但却是按着他现在的蛇形态样子雕刻。
<br/>　　到时候这座蛇王雕像入庙正堂，他的信仰之力将源源不绝，滚滚如黄河之水。
<br/>　　蛇王的影响力在白镇已经达到巅峰，街道上几十万的队伍跟着蛇王出巡，就已经说明一切，规模浩大，前所未有。
<br/>　　但他们不知道，真正的蛇王，就在这人海之中，跟着他们，一起庆祝蛇王节。
<br/>　　蛇王出巡，三小时后，下午13时许，蛇王出巡回到庙前。
<br/>　　此时时刻，无数人泛着激动的心情望着被大红布盖着的蛇王雕像，一个个都迫不及待的想亲眼目睹蛇王雕像的风采。
<br/>　　据可靠消息，这次的蛇王雕像不是人形，而是蛇王本尊的样子。
<br/>　　而且听说还是蛇王亲下传下口谕所制作。
<br/>　　看到这蛇王雕像，等于就是看到蛇王了。
<br/>　　这小道消息，也不知道哪里传出去，但可信度非常的高，让所有人都非常的期待。
<br/>　　蛇王雕像立于庙前，仪式即将开始，已蛇王庙为中心点，早就被包围的水泄不通，非常热闹拥挤，爆满，只要能站脚的地步，绝不会空着。
<br/>　　层楼上，大槐树上，哪怕是电线杆上，都能看到人的影子。
<br/>　　蛇王庙上空，几十架无人机悬空拍摄，人们不会放过任何一个细节。
<br/>　　白镇人民气氛高涨，就像炽热的火焰朝天燃烧一样，呐喊声，高呼声不绝于耳。
<br/>　　他们都在期待蛇王仪式的正式开始，蛇王的真面目。
<br/>　　涂小安悄然的隐匿拥挤的人群中，那张面容不禁流露出苦笑，人实在太多，他感觉自己这虚假的人形态要被挤爆了。
<br/>　　这人头攒动，纷至沓来，一个个趋之若鹜都想站在最前排，因为很显然只有最前排才能看的最为清楚。
<br/>　　“咚....”
<br/>　　忽然，此时庙里钟楼内，传出了一声洪亮的钟声，钟声古朴，肃穆，又绵长震撼，敲响在万千民众的心间。
<br/>　　霎时间，所有人都不自觉地屏息凝视着钟声响起的方向，原本还热火朝天的现场顿时鸦雀无声，整个空间变得寂静无比。
<br/>　　大家放眼抬头，面容肃穆，他们知道蛇王雕像的仪式真正要开始了。
<br/>　　此时，庙堂之中，一位满头的白发老者，手持一根蛇杖，佝偻着身子，步履蹒跚的缓缓走了出来。
<br/>　　一张饱经风霜的老脸引入大家的眼帘，顿时间，大家肃然起敬。
<br/>　　李得丰，他是白镇最德高望重的老者，地位崇高，活化石级别。
<br/>　　蛇王雕像仪式要开始，蛇王雕像上的大红布谁来揭开，这也是非常的讲究。
<br/>　　不能随便找一个人，这个人必须要德高望重，资深，让众人信服。
<br/>　　李得丰就是这样一位老人，在白镇无人不信服，捕蛇水平早就是殿堂级别，他的徒弟不单单是遍布白镇，可谓遍布天下，年轻的时候是白镇最优秀的捕蛇人，人中蛇王，老了，将自己的捕蛇技术一一传授。
<br/>　　最让人尊敬的是，老人传授的不单单是如何捕蛇，更是对蛇的一种态度跟文化知识。
<br/>　　老人是真正的敬畏蛇，崇拜蛇，不会将蛇当成是畜生，当成赚钱的工具，他像大多数传统手艺人那样，安于贫穷，虽有一技在手，却从来不显摆，默默已发扬传承文化为己任。
<br/>　　捕蛇人，是一门职业，捕蛇是一门技艺，也是老祖流传下来的文化瑰宝和智慧结晶。
<br/>　　他说的最多的一句话，便是你可以捕蛇，但你不可肆意的杀蛇，因为你已蛇为生，自然要敬重它，爱护它。
<br/>　　就好像经常被人挂在口头上的一句话，只有你热爱生活，生活才会热爱你。
<br/>　　你肆意的杀蛇，换来的不是财富，而是蛇的血光之灾。
<br/>　　很显然，李得丰是现在白镇最殿堂级别的捕蛇人，蛇王雕像由他老人家来揭开，最合适不过。
<br/>　　当初佘山之难，也正是大家请出了李得丰，由他去蛇王庙请蛇王出世，降妖除魔。
<br/>　　钟声响起，老人出现，现场鸦雀无声，不敢造次。
<br/>　　老者李得丰虽然年事已高，两只眼镜都深陷了，但他的眼睛还深邃而明亮，看上去很有神，步履阑珊间，透着浑厚有力。
<br/>　　他走到盖着大红布雕像面前，一边的道袍男子连忙为他献上三柱点燃着的香，然后乖乖退到一旁。
<br/>　　此时站于前排的民众，都是一些有身份有地位的人，他们早就持香等待，尊敬的看着老者。
<br/>　　李得丰高举三柱香，先是冲着蛇王雕像深深鞠躬一礼，然后准备跪下先行磕头礼拜。
<br/>　　可诡异的事情再次发生了，老者的膝盖如何都弯不下来。
<br/>　　人群中的一位清秀男子脸上透着惶恐跟无奈：“我的师公，您可别在继续跪了，小子承受不起”
<br/>　　这蛇王节众人下跪是正常现象，但是今年的蛇王节有所不同，涂小安成了蛇王，雕像用的是他的样子。
<br/>　　别的人，涂小安可以无视，但李得丰何等人物呀，涂小安怎能让他跪在自己的雕像面前。
<br/>　　李得丰半天跪不下来，很多人都看出了猫腻，想必是蛇王大人不忍让他跪下。
<br/>　　“老爷子，您别跪了，蛇王大人体恤您老年高，不让你行大礼”
<br/>　　道袍男子李顺连忙向前说了一句。
<br/>　　李得丰能感受到那股无形的浩然力量，心中有数，但仍然坚持的对着雕像三鞠躬，望向众人，郎朗高声：“我白镇，自古就受蛇王大人庇佑，风调雨顺，百姓安康，古有蛇王出世，消除霍乱，今有蛇王出世，降妖伏魔，得蛇王庇佑，乃我白镇大福也”
<br/>　　老人一句话说完，道袍男子一举手，高呼：“跪”
<br/>　　顷刻间，兵败如山倒般，密密麻麻，人山人海的群众全部跪下，场面无比的宏观，让人震撼非常。
<br/>　　一片片跪倒下来，只有一位年轻男子还站着，顿时间无数道目光齐刷刷的杀来，年轻男子吓的连忙跪了下来。
<br/>　　“哎，这成了什么事，自己还要对着自己的雕像下跪”
<br/>　　涂小安恶趣味的苦笑一声，自我调侃。
