　　第5章
<br/>　　土屋内亮着烛火，土炕上坐着一红衣女子。妖娆的火光衬得她脸蛋愈发红润，琼鼻玉脸，黑发如瀑布般垂落，披在肩头。
<br/>　　程大雷心中一荡，好安静的女子。怕是只有这古老的时代，才能出落如此干净的女人。她坐在那里，眼神澄清，干净，如水也如冰，让自惭形秽，也让人心生贪婪。
<br/>　　程大雷心底突然冒出一个邪恶的念头：这是我的，我的。在这山寨之中，没人可以阻拦自己，大家都已默认这个女子是自己的压寨夫人，在这间土屋，在这土炕上，自己想对她做任何事都可以。
<br/>　　想到这里，程大雷心底忽然一慌。人在没有约束的环境，道德与原则会迅速退化，前世自己从没做过坏事，可一旦到这不受拘束的环境，恶念便不可遏制的滋生。
<br/>　　人性，真的就比兽性高尚嘛？
<br/>　　嘟，正义的少年喔，因你心怀善念，触发『抢夺压寨夫人』第二阶段任务：用非暴力手段打动苏樱的芳心。
<br/>　　咦，程大雷睁大眼睛，自己怎么就心怀善念，你搞搞清楚好不好，道德从来都是批判别人的武器，而不是约束自己的工具。
<br/>　　嘟，真正的男人既要有取天下的雄心，也当有取女人心的柔情，正义的少年，要加油喔。
<br/>　　嘟你大爷的嘟，你一个破系统卖什么萌，真是受够这种满满中二病的台词。
<br/>　　这件事真切的提醒程大雷，伤春悲秋要不得，矫情永远都是一种病。
<br/>　　不过系统已经如此要求，看来暴力手段已经用不得，他深吸一口气，摆出一张自以为最英俊潇洒的表情。
<br/>　　忽悠，并不是暴力手段！
<br/>　　“长夜漫漫，无心睡眠，我以为只有我睡不着觉，原来晶晶——不，原来苏樱姑娘也睡不着？”
<br/>　　苏樱睁大眼睛：“我家是落叶城的四海商行，只要你肯放我离开，财或物都有人愿意给你，我还有一个身份，是黑石城未来城主的夫人，伤害我会有什么代价，不用我说你也明白？”
<br/>　　落叶城是除黑石城外，距离蛤蟆岭最近的一座城池。苏樱在第一时间就告诉程大雷自己的价值，不仅有利诱，还有威胁。
<br/>　　“误会啦，误会啦。”程大雷尽力挽回自己的形象：“苏樱姑娘肯定把我当成一般的山贼，但这次的事情，是因为我与黑石城主有杀父之仇，等这场事情结束，我会平安将苏樱姑娘送回。”
<br/>　　“假的。”苏樱平静道：“我从小就能感觉到一个人是否撒谎，我从你的眼睛中看到贪婪，欲望，和其他男人没有什么不同。我只是想说，不要伤害我，其他条件都可以谈。”
<br/>　　程大雷语结，都说胸大无脑，难道这位姑娘竟然是个例外。他把注意力集中在苏樱身上，查看着她的信息。
<br/>　　姓名：苏樱
<br/>　　年龄：17
<br/>　　技能：无
<br/>　　隐藏属性：明察秋毫的奇女子。
<br/>　　是因为明察秋毫，所以能分辨谎言？程大雷很是郁闷，这是连忽悠都不能了。
<br/>　　程大雷决定最后努力一把：“苏樱小姐，我知道在你眼里，我只不过是个粗鲁的蟊贼，但若能安稳度日，谁又想落草为寇，其实在很久以前，我也是一个有理想的少年。”
<br/>　　“理想，少年？”苏樱面露困惑，觉得这两个词和程大雷都扯不上关系。
<br/>　　程大雷调动面部肌肉，控制呼吸，决定开始自己最竭尽全力的一次表演。他在房间内来回踱步，时而挥舞着手臂慷慨激昂，时而面露悲色长吁短叹。
<br/>　　“……我这双手也曾捧过圣贤书，握过青锋剑，也曾想习得文武艺，货与帝王家，也曾说男儿何不带吴钩，收取关山五十州。可我游历四方，所见流民如猪狗，权贵如虎狼，我发愤于圣贤，只想问这世界为何如此，那时我有一个理想，救苍生于水火，解万民在倒悬，我有一个理想，打破旧世界，重建新乾坤，我有一个理想，让人人有其居，耕者有其田……”
<br/>　　最后，程大雷向苏樱伸出手，用满怀深情的目光注视着她的眼睛。
<br/>　　“现在，我这双手只想握紧一个姑娘的手，苏姑娘，你愿意和我共同完成这个理想么？”
<br/>　　“假的。”苏樱眨了眨大眼睛，又道：“可虽然你说的是假的，但我能感觉到，你不想伤害我，我不知道为什么。”
<br/>　　程大雷无奈的拍拍额头，自己演技不够好么，怎么就被看穿了，难道是情绪起得太高，前期铺垫得不够？
<br/>　　为什么不想伤害你，这个问题我无法回答，请把话筒交给系统。
<br/>　　“你想要什么，都可以告诉我，只要你不伤害我。”苏樱。
<br/>　　程大雷在苏樱对面坐下，终于说出进入房间后唯一一句真诚的话。
<br/>　　“你能给我讲讲外面的世界吗？”
<br/>　　不是亲身体会，程大雷根本无法想象这个世界的信息闭塞程度。唯一的信息传播渠道，可能就是流言。十里之外，就是两眼一抹黑，有人甚至一生都未走出过村庄，外面改朝换代，都无法知道。
<br/>　　假若有人肯花十几年时间，在各州转一圈，那就能配得上见多识广的评语。
<br/>　　现在程大雷也只是知道，帝国的国号是大武，由十三州一百单八城组成，除此之外，程大雷对整个世界一无所知。
<br/>　　他迫切需要知道关于这个世界的一切，人们吃什么，穿什么，读什么书，靠什么消遣，有怎样的文化，信仰如何的宗教……程大雷知道，这些看似微不足道的东西，其实无比重要。
<br/>　　而在这样一个信息闭塞的时代，游历四方的商人掌握着更多的信息。现在自己眼前坐着的，是一个商人之女。
<br/>　　这个机会，无比珍贵。
<br/>　　一番长谈之后，程大雷离开房间，在山寨内随便找个地方休息。
<br/>　　苏樱一人坐着房间内，屋内红烛照亮她的手，她用手指沾着水，在木桌上写下两行字。
<br/>　　『男儿何不带吴钩，收取关山五十州』
<br/>　　苏樱颦起眉头，认真看着这两行字：吴钩是何物，该是一种很锋利的兵器，关山又在何处，帝国只有一十三州，何来的五十州……
<br/>　　苏樱越想越不得其解，一个人呐呐自语。
<br/>　　“明明知道他的话是假的，可这些假话说得真漂亮啊，如果那在诗会上给那些所谓的才子看，估计会吓他们一跳吧……”
<br/>　　
<br/>(本章完)
