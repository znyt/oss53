　　或许是运气好，这一次遇到的猎妖队，并没有恶言恶语。
<br/>　　“易霄是吗？我们这支猎妖队刚好缺一人。只是，你那后天一重的修为...”
<br/>　　说话的是这支猎妖队的队长，是个中年大叔。
<br/>　　萧逸看不透他的修为，证明他是先天武者。
<br/>　　大叔也算给萧逸留了面子，后面的话没有说下去。
<br/>　　“唉，看来只能另想办法了。”萧逸叹了口气，准备离开。
<br/>　　这时，一个一身淡青色长衫的少女走来，手上拿着一大堆零食，嘴里还大口吃着。
<br/>　　“铁达大叔，招募够人了吗，可以出发了吧？”少女嘴里吃着东西，含糊不清地说道。
<br/>　　“还没呢。”大叔无奈地摇摇头。
<br/>　　萧逸停下脚步，问道，“这是你们的队员？”
<br/>　　“嗯。”大叔点点头。
<br/>　　萧逸瞪眼道，“我看她年纪比我还少，修为也不过是后天二重吧。”
<br/>　　少女约莫15、6岁。
<br/>　　事实上，这个年龄能有后天境以上修为的武者，已经是很强的天赋了。
<br/>　　但，当猎妖师，是要到各种危险之地完成任务的。
<br/>　　没人会管你有没有天赋，外头的妖兽更不会看你是个天才而不吃你。
<br/>　　所以，实力才是猎妖队招募队员的标准。
<br/>　　“是你？”少女见了萧逸，蹙眉瞪了眼萧逸。
<br/>　　“怎么，浅沫，你们认识？”大叔问道。
<br/>　　“不认识，哼，不礼貌的家伙。”少女撇撇嘴。
<br/>　　“不认识，哼，一个幼稚的家伙。”萧逸也在同一时间撇撇嘴，
<br/>　　这支猎妖队一共五人，除了少女和大叔外，还有三个年轻男子，年龄约莫三十出头。
<br/>　　其中一人，尖嘴猴腮，见少女似乎认识萧逸，立刻露出了不爽的神情。
<br/>　　“小子，哪凉快哪待着去，别浪费我们时间。”那人不屑地对萧逸说道。
<br/>　　这三人都是后天五重以上的修为。
<br/>　　大叔也对萧逸说道，“易霄，抱歉了，这次的任务难度很高，你确实不适合加入我们。下次吧，若是有缘，一起合作。浅沫她是药师，所以不需要实力，自有我们保护她。”
<br/>　　很多猎妖队都不是固定的。
<br/>　　由一个队长临时招募，完成任务后，便会各散东西，或者下次有缘再合作。
<br/>　　当然，也有的在任务中结下深厚情谊，便一直结伴完成任务。
<br/>　　“药师？”萧逸淡淡地说了一声。
<br/>　　药师，指的是在战斗中有治疗能力的武者，不一定是炼药师。
<br/>　　少女像是炫耀一般对萧逸道，“哼，我乃是长生藤武魂，天生具有治疗能力。”
<br/>　　之前敌视萧逸的男子不屑道，“有浅沫在，我们受再严重的伤势都不怕了，小子你有什么本事跟她比。”
<br/>　　萧逸耸耸肩，淡淡道，“我是炼药师。”
<br/>　　炼药师三字一出，那男子的话语戛然而止。
<br/>　　大叔和其他两名猎妖师更是脸色一变。
<br/>　　“炼药师？”铁达大叔瞪大了眼睛，立刻问道，“会炼制驱兽丹、隐气丹和巨力丹吗？”
<br/>　　“会。”萧逸点点头，道，“但不是很精通，三...”
<br/>　　铁达大叔立刻垂头丧气，打断道，“不精通吗？唉，还以为运气那么好，能遇到个一品炼药师呢。”
<br/>　　萧逸笑笑，道，“不是很精通，三品的练不出。二品的驱兽丹、隐气丹和巨力丹还是练得出的。”
<br/>　　“二品？”铁达立刻面露惊讶，一把捉住萧逸的肩膀，激动道，“你是二品炼药师？”
<br/>　　“二品炼药师？”其他队员同样面露惊讶。
<br/>　　连少女都直勾勾地盯着萧逸。
<br/>　　萧逸笑笑，道，“如何，能加入吗？”
<br/>　　“废话。”铁达大叔大笑道，“傻子才不让你加入。”
<br/>　　这支猎妖队的人立马态度一百八十度转变。
<br/>　　炼药师作为整个大陆最稀有也是最高贵的职业，待遇自然没得说。
<br/>　　若是之前萧逸早早亮出炼药师的身份，根本不会有人拒绝他的加入。
<br/>　　铁达大叔笑道，“本以为能有一个一品炼药师就不错了，没想到竟然是二品。走，易霄兄弟，我们到猎妖队去登记，在完成这个任务前，我们就是同伴了。”
<br/>　　铁达大叔连称呼都变了，直接称呼萧逸为易霄兄弟。
<br/>　　驱兽丹，武者服之，身上自动散发特殊气息，让妖兽不敢靠近。
<br/>　　隐气丹，武者服之，自动隐匿气息，方便逃生。
<br/>　　巨力丹，短暂时间内增加武者力量，增加战力。
<br/>　　丹药的品级越高，效果越大。
<br/>　　二品驱兽丹和隐气丹，足以让拥有后天境实力的二级妖兽不敢靠近，更闻不到武者的气息。
<br/>　　药师虽好，能让猎妖师受伤时立刻得到治疗。
<br/>　　可炼药师的各种神奇丹药，却直接能让猎妖师们免去不少战斗，出任务的安全性更高。
<br/>　　“慢着。”之前敌视萧逸的男子道，“队长，谁知道这小子是不是乱说的，要是骗我们怎么办。”
<br/>　　“不会吧。”铁达大叔道，“没人敢冒充二品炼药师的。”
<br/>　　“腾”的一声，萧逸手中猛地冒出一股炽热火焰。
<br/>　　“你想干什么？”那男子顿时一惊，摆出了战斗状态。
<br/>　　铁达大叔和另外两名队员也是神色一凛。
<br/>　　作为惊讶丰富的猎妖师，反应速度极快。
<br/>　　附近的猎妖队这时也注意到这边的情况，纷纷看过来，准备看戏。
<br/>　　“哼，区区后天一重，想和我战斗不成？”那男子满脸不屑。
<br/>　　“呵。”萧逸不屑笑笑，随意从乾坤袋中扔出一堆材料，投入掌心火焰中。
<br/>　　不多时，一粒洁白无瑕的淬体丹出现。
<br/>　　“原来是在炼药。”铁达大叔先是松了口气，而后惊呼道，“不用炼药炉，随手炼药，单靠真气凝聚的火焰就能结丹，厉害厉害。”
<br/>　　以萧逸如今的炼药水平，淬体丹作为最简单、最容易的丹药，自然是信手沾来。
<br/>　　萧逸掌中结丹，证明他绝对有二品炼药师的实力，否则绝不可能这么轻易炼制好一粒淬体丹。
<br/>　　“现在信了吗？”萧逸淡淡地说道，收起了火焰。
<br/>　　“信，当然信。”铁达大叔满脸笑容，而后面露抱歉之色，指了指那男子，道，“易霄兄弟，这是杜新，性子急了些，但绝对没有恶意的。”
<br/>　　萧逸点点头，没有多说什么。
<br/>　　“走，我们进猎妖殿等级。”铁达大叔和蔼地说道。
<br/>　　这时，周遭猎妖队的人肠子都悔青了。
<br/>　　“卧擦，那个少年不是刚才的小鬼吗？”
<br/>　　“吗的，竟然是个炼药师，早知道刚才就收了他。”
<br/>　　“草，被铁达他们捡了大便宜了。早知道那少年是炼药师，说什么也让他加入阿。”
<br/>　　刚才拒绝萧逸的猎妖队，纷纷后悔，暗骂自己眼力不足。
