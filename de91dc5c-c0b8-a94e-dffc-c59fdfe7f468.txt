　　钦陵十分认同黑齿常之的战略战术，既然敌人还没有反应过来自己的优势在什么地方，那就尽可能的抓紧时间，将己方的优势再扩大一些。
<br/>　　东、西、中大唐靺鞨铁骑三路大军齐出，进接向倭人的防线压了上去。
<br/>　　倭人似乎也有自知知明，知道以现有的力量根本无力阻止唐军南下，下于便不断的使用断尾战术，用一万或两万人的军队来拖延唐军的进攻步伐。
<br/>　　而钦陵这家伙与黑齿常之也的确够狠，本着蚊子再小也是肉的原则，就算是三、五千人也是来者不拒，统统杀光一个不留。
<br/>　　……
<br/>　　时间就这样又过了近半个月，半个倭岛已经变成了死地，到处都是野兽横行，那些死掉的倭人尸骨被拖的到处都是。
<br/>　　不过这此似乎都与唐军无关，钦陵的军队还在不断向南，十万大军拉开长长的战线，就那么平着推了过去，所到之处鸡犬不留。
<br/>　　倭人到了这个时候也终于不再退却，因为他们已经无路可退，再退他们将失去最后的后勤保障，到时候不用唐军来攻，他们自己就能把自己饿死。
<br/>　　百万倭军在平城京以北在的敦贺摆开阵式，扎下营地，看样子似乎颇有决一死战的味道。
<br/>　　钦陵自然不会在乎秦吉胜和山背大兄所谓的百万大军，透过望远镜，看着那些衣衫褴褛连裤子都没有的倭兵，他真的很难把这些人与士兵联系到一起。
<br/>　　如果不是李承乾的命令，钦陵是真的不想再打下去了，不是因为可怜那些倭人，而是因为他的对手太弱，弱到几乎就是一群拿着锄头的百姓。
<br/>　　但命令就是命令，就算钦陵不想打也必须打，在这个皇权至上的年代，敢于挑战大唐皇帝的人必须得到教训，用他们的鲜血来洗刷他们带给大唐的耻辱。
<br/>　　……
<br/>　　敦贺是倭国南北之间的一座重要关隘，也是平城京北面的最后一道防线，突破了这里接下来倭国将会处于无险可守的劣势。
<br/>　　所以不管是秦吉胜还是山背大兄，他们宁死都不会放弃敦贺，哪怕是战至最后一兵一座，这一仗也必须打下去。
<br/>　　不过，这座关隘因为此前在倭国内战的时候数次受到战火的洗礼，此时早就已经不具备成为关隘的条件，所以倭人只能用自己的生命来阻止大唐南下的脚步。
<br/>　　……
<br/>　　十万大唐靺鞨铁骑，装备精良，兵强马壮；百万倭军衣衫褴褛，武器不全。
<br/>　　两支完全不成比例的军队就这样在敦贺对峙着。
<br/>　　秦吉胜第一次出现在山背大兄的军营之中，为了表示自己的诚意，同时也为了商量下一步的行动。
<br/>　　“服部，这是你第一次来我的军营吧？”山背大兄毕竟是倭国的皇族，纵然已经被打的苟延残喘，但气度上却没有任何一点的瑕疵，亲自出营来接秦吉胜的作法让人无法挑剔。
<br/>　　“山背大兄王，这次我来是想要跟你商量一件事情。”秦吉胜在进入山背大兄的军帐之后，缓缓的说出自己的来意：“我希望你能将军队的指挥权交给我。”
<br/>　　“什么？这不可能！”物部氏族长想都没想，不等山背大兄表态便第一个跳了起来，大声呵斥道：“秦吉胜，你也不看看自己是个什么东西，如果不是你，我大倭帝国何至于走到今天这个地步，现在你竟然有脸来要军队的指挥权，你想干什么，直接投降么？！”
<br/>　　“物部，稍安勿躁。”山背大兄静静站在一边，等到物部氏族长将秦吉胜喷完了，这才站出来装好人：“或许服部君真的有什么办法也说不定，你说是吧，服部。”
<br/>　　但秦吉胜就像没有听到山背大兄话里的讽刺味道，只是淡淡看了他一眼：“我没有办法，此战只有拼命而已，胜则生，败则亡，百万大军已经是我们最后的力量，如果战败，除了引颈就戮之外，再也没有任何挽回的机会。”
<br/>　　“那你还来这里要兵权干什么？”物部氏族长冷声问道。
<br/>　　“鸟无翅不飞，蛇无头不走，一支队伍只需要有一个声音，现在我和山背大兄王合兵一处令出多头正是最大的弊端，若是打起仗来，下面的士兵到底要执行谁的命令？”秦吉胜这一次没有退缩，反而直视着物部氏族长，冷声反问道。
<br/>　　“这……”物部氏族长也有些不知应该如何回答这个问题，不过他更希望整只军队由山背大兄控制，因为这样多少会让他有些安全感。
<br/>　　“我知道你在担心什么，但是你可以放心，大是大非面前我秦吉胜还知道如何选择，绝不会做那亲者痛仇者快的事情，否则当如此桌”秦吉胜也不等物部氏族长说出个子午寅卯，“哗啦”一声抽出身边一个护卫的长刀，对着一边的桌子便砍了下去。
<br/>　　秦吉胜把事情做到如此地步，山背大兄还真就不好再说什么了，如果他还不答应就会显的有些不识大体，会让人看不起不说，下面那些当兵的也会对他有所不满。
<br/>　　更何况他现在手底下的这些士兵有一半是从秦吉胜的领地上招募来的，所谓的嫡系部队并不多，甚至就连他的后勤也都要依靠秦吉胜。
<br/>　　这样的情况更是让他有些骑虎难下，想不答应秦吉胜都难。
<br/>　　良久之后，山背大兄似乎有了决定，将自己的佩剑解了下来，‘啪’的一声放到桌上：“服部，希望你能记住你的话，倭国的命运就押在你身上了！”
<br/>　　物部氏族长在看到山背大兄的举动之后，顿时有种崩溃的感觉，噗通一声跪到地上，连声叫道：“陛下，不行啊，他连名字都改成姓秦了，能是什么好东西，您不能如此信任他啊。”
<br/>　　“朕意已决，物部休得多言！”山背大兄在有了决定之后，便再也没有悔意，淡淡看了物部氏族长一眼之后说道：“此次背水一战，若是胜了服部便是倭国的功臣，朕可以禅位于他，若是败了……那就万事皆休！”
