　　就在鬼修老人即将抓到刘哔头颅的时候，一柄闪烁着寒光的关刀，劈向了他的头颅。
<br/>　　察觉到头顶上方传来的凌厉风声，被迫无奈之下，鬼修老人只得作罢，转身用拐杖挡下了身后的攻击。
<br/>　　“林九，到底你还是过来了，怎么，要打搅老头儿我制作法宝不成？”挡下了头顶上方的攻击，鬼修老人看清了来人，阴森一笑，飞身退到了大厅中央。
<br/>　　滚动了一下眼珠，刘哔看见了半截黄色道袍，从声音上来听，应该是九叔赶来了。
<br/>　　“你杀的？”看了一眼地板上尸首分家的刘哔，九叔语气冰冷，一股强横的威压从九叔身上散发出来。
<br/>　　咧了咧嘴，鬼修老人没有回答，反手持着拐杖，向着九叔头顶攻击而去。
<br/>　　事实上，这次他来义庄，也并非只是为了那颗百年尸丹，更多的，还是为了九叔而来。
<br/>　　身为鬼修，能活到他这种岁数也极为不易，几乎他后半生所有的时间，都是在无尽的追杀中度过的。
<br/>　　而他与林九，年轻的时候也交过几次手，只是那时自己修为尚浅，没能在林九手上占到便宜。
<br/>　　恰好借着这次百年尸丹的机会，他也想再次讨教讨教，如果有机会的话，就顺手解决了这个强敌。
<br/>　　剑眉冷竖，九叔咬破手指，用鲜血在刀柄处画了一道符咒，随即，关刀伴随着一股金黄色的灵气，向鬼修老人迎了上去。
<br/>　　听见耳边传来的打斗声音，刘哔有些心急，他知道这个鬼修老头的实力不俗，也不知道九叔能不能应对。
<br/>　　如果九叔也被打败，那他就没有一点活命的机会了！
<br/>　　事实上，他也不是没有办法将头和自己的身体连接回来，但那样的话，自己的身份自然就会暴露了。
<br/>　　就算是九叔打赢了鬼修老头，那他同样不会有活命的机会。
<br/>　　以九叔的性格，绝对不会容忍他这样的异类活下去的。
<br/>　　一时之间，情况像是进入了死局，犹豫良久之后，刘哔将头颅滚到尸体旁边，侧看着九叔和鬼修老人的争斗。
<br/>　　他已经打定主意，只要找到机会，就立马逃走，至于那该死的任务，就让它见鬼去吧。
<br/>　　这个任务根本不是现阶段的他可以完成的，活命才是当下最重要的事情！
<br/>　　怀揣着这样的想法，刘哔安静的躺在一边，而九叔和鬼修老人的战斗，此时也进入了白热化。
<br/>　　光是看场上的情况，还看不出战斗的输赢，两人都是将实力爆发到了极致，许多刘哔之前没有见过的手段全都使用了出来。
<br/>　　那把普通的关刀，被九叔舞的虎虎生风，灿烂的金光不断从上面倾洒下来，如同天神下凡一般。
<br/>　　而鬼修老头也不再是开始那半死不活的样子，五颗狰狞的死人头颅，口中不断喷出一股股黑色的雾气，缠绕在九叔的身上。
<br/>　　这股雾气，像是带有极强的腐蚀性一般，将九叔的黄色道袍腐蚀出一个个大洞，露出了下面青紫色的皮肤。
<br/>　　躺在地上的刘哔猜测，九叔大概是中了什么毒。
<br/>　　两人打的越来越不可开交，黑金两色的灵气在半空中不停碰撞着，发出类似于金戈相交般的轰鸣声。
<br/>　　刘哔眼中带着焦急，他想要逃跑，但两人的战场离自己太近，他根本找不到合适的机会。
<br/>　　场中两人的身影不断腾挪着，像是两道人形闪电一般，很多时候，刘哔都捕捉不到他们的身影。
<br/>　　交手了几百回合，甚至天色都已经逐渐开始亮了起来，场中的战局，终于发生了一些变故。
<br/>　　秃顶老人明显是体力有些支持不住了，攻击的力道却带着不如先前那样凌厉了，察觉到这一点，九叔的攻击反倒是更加狂暴了。
<br/>　　一滴冷汗从鬼修老人的头顶滑落下来，他对林九的实力，也着实感到心惊，如果一直这样下去，迟早他都会因为灵气不足而丧命在林九手上。
<br/>　　手上的动作没有停顿，鬼修老人吃力的抵挡着九叔攻击，眼睛却是向停尸间门口瞥了一眼。
<br/>　　此时，他心中也已经有了退意。
<br/>　　虽然全力防御的话，他还可以支撑很长一段时间，但是他想要获得这场战斗的胜利，却已经没有什么希望了。
<br/>　　况且，天马上就要亮起来了，这边的战斗，已经持续了太久，万一引来其他的修炼者，到时候想要逃跑都成了问题。
<br/>　　心中这么琢磨着，鬼修老人开始有意识的将战场引到大门附近。
<br/>　　路过刘哔尸体的时候，鬼修老人眼中闪过一抹精光，随即操控五颗人头，爆发出一股强横之极的能量向九叔涌去。
<br/>　　一时没有准备，九叔被这突然袭来的爆发逼退，随后心中一沉，他知道鬼修老头这是想要逃走了。
<br/>　　虽然知道这鬼修的意图，但两人实力并没有相差许多，如果对方一心想要逃走，他也什么办法可以留住。
<br/>　　而鬼修老人这边，这一波凌厉的攻势，让他体内本来就所剩无几的灵气更加枯竭了。
<br/>　　逼退了林九，他也没有半点恋战的心思，伸手抓起刘哔的尸首，就从停尸间门口冲了出去。
<br/>　　这小子身上还有一颗百年尸丹，头颅也可以给徒弟炼制成法宝，这些东西都不能浪费。
<br/>　　在刚到义庄附近的时候，他就感应到了百年尸丹那股浓烈的尸气，否则的话，他也不可能这样精准的找上门来。
<br/>　　没有任何犹豫，鬼修老人抓着刘哔的时候，在房屋顶上疾驰着，之前留下断敌的五个死人头颅，不久后也重新回来，跟在了鬼修老人身后。
<br/>　　也幸亏是这个时间天还没有大亮，街上并没有什么行人，否则看到他这般模样，也是一个不小的麻烦。
<br/>　　眼睁睁的看着鬼修老人从义庄离开，九叔表情无比愤怒，纵身追了出去，根据房顶上留下的些许痕迹，远远的追踪了过去。
<br/>　　落在鬼修手中，他自然知道会有什么样的下场，自己徒弟的性命明显是救不回来了，但他可不想让刘哔死后都受到折磨。
