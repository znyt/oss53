        “非得去实验室吗？”童筱颖微微凝眉，有一丝不满这个见面地点的小情绪。
<br/>
<br/>        裴姨也不打算藏着掖着，而是十分坦白，开门见山地跟童筱颖说道：“解药在小澈身上的副作用太大了，已经到了没法控制的地步。所以，我需要抽筱颍的血，去救小澈，筱颍，裴姨求你了，好吗？”
<br/>
<br/>        “裴金澈他不是已经好了吗？”童筱颖感到纳闷。
<br/>
<br/>        当初是裴姨要她把解药给裴金澈，说是可以治疗他身上的怪病，怎么现在又到了病情没法控制的地步了呢？
<br/>
<br/>        “小澈的事情，我一时半会儿讲不清。总而言之，我继续你的血。筱颍，你就帮帮裴姨好不好？看在裴姨当初放你离开实验室的份上。”裴姨语重心长地哀求。
<br/>
<br/>        童筱颖犹豫了一会儿，弱弱地问道：“那我回去的话，那个连老头会放过我吗？”
<br/>
<br/>        “这段时间，老连他们出国了。要一个星期后才会回来，所以实验室里现在就我一个人。你可以放心的过来。”裴姨解释道。
<br/>
<br/>        童筱颖这才答应了裴姨的要求。
<br/>
<br/>        挂了电话后，童筱颖洗漱完，简单地吃了个早餐，然后去了实验室里赴裴姨的约。
<br/>
<br/>        再次回到这个实验室，童筱颖的心情很复杂。
<br/>
<br/>        当初，她还以为这是她的家，而所有的科研医生都是她的亲人。
<br/>
<br/>        后来……
<br/>
<br/>        当她得知真相的那一刻，心情也曾有那么一刻崩溃过。
<br/>
<br/>        童筱颖再次躺到了抽血台上，穿着白马褂、戴着白口罩的裴姨，手中端着抽血用的器具向她走了过来。
<br/>
<br/>        “这段时间不见你，你的气色好看多了。好像还比以前要胖了点。看来，傅夜沉把你养得还不错。”为了缓和气氛，裴姨打趣地说道。
<br/>
<br/>        这段时间，傅夜沉可是好吃好喝好玩地养着她，她当然会胖，只不过……
<br/>
<br/>        童筱颖抿了抿唇，讷讷地问道：“这次要抽很多血吗？”
<br/>
<br/>        “大概五百毫升左右。”裴姨一边回答道，一边抓住童筱颖的手臂，替她绑上橡胶带。
<br/>
<br/>        童筱颖抬将手伸直了，另一只手则伸到嘴边，贝齿咬住衣袖，偏过头去。
<br/>
<br/>        抽完血后，童筱颖起身的一瞬间，有些发黑眼晕，但很快就适应了过来。
<br/>
<br/>        “你离开实验室后，每天都还做梦吗？”裴姨问道。
<br/>
<br/>        “裴姨，你能告诉我，为什么我总会产生有人给我下达命令的幻觉？”童筱颖若有所思地问道。
<br/>
<br/>        “是什么命令？”裴姨不以为然地打趣道。
<br/>
<br/>        “让我去杀傅夜沉。”
<br/>
<br/>        “哐”地一声，童筱颖的话一落，裴姨手中的血袋掉在了托盘上。
<br/>
<br/>        “你也精神失常了？”裴姨匪夷所思地看向童筱颖。
<br/>
<br/>        童筱颖微微歪着脑袋，一脸茫然地对上了裴姨的目光。
<br/>
<br/>        “小灵跟我说，小澈最近总是精神失常。白天还好，但是，到了晚上，他总会做一些反常的事情来。比如，他要杀了小灵。再比如，他会自言自语。总之，他……”裴姨黯然神伤地垂下了眼帘，深深自责起来，“早知道解药副作用这么大，我就不应该要你拿给他去用。”
<br/>
<br/>        裴金澈有快速恢复的能力，因此没有痛觉，同时身体代谢也比常人要快，容易衰老。
<br/>
<br/>        但是，解药治好了他的怪病，却让他精神失常。
<br/>
<br/>        别说夏玄灵忍受不了，就连她这个代孕母亲也跟着很是心疼。
<br/>
<br/>        童筱颖忽然瞪大了眼睛，像是猛然间明白了什么，从抽血台上跳了下来。
<br/>
<br/>        “你去哪儿？”裴姨连忙问道。
<br/>
<br/>        童筱颖一边穿上鞋子，一边回答道：“我去找裴金澈。”
<br/>
<br/>        “你找他做什么？”
<br/>
<br/>        裴姨正为此事纳闷的时候，童筱颖已经穿上鞋往门口跑去。
<br/>
<br/>        童筱颖要离开，裴姨没理由拦着，就算她是去找小澈，她也很放心，因为，她知道，她不会害他。
<br/>
<br/>        这天傍晚，裴金澈的公寓里多了一个女人。
<br/>
<br/>        夏玄灵双手抱臂，瘪着嘴唇，充满敌意地盯着坐在对面沙发上的童筱颖。
<br/>
<br/>        裴金澈下班回到自己的公寓里时，看到客厅上两个女人之间透着一股火药味，冷不丁地打了个寒颤。
<br/>
<br/>        “阿澈，你回来了！”夏玄灵立即从沙发上起身，直奔裴金澈身边。
<br/>
<br/>        裴金澈看到童筱颖时，感到很诧异，但随即又恢复了淡定的神情，用兄长一样的语气，问道：“筱颍，你怎么在这里？”
<br/>
<br/>        “当然是来看你精神失常的样子啊！”童筱颖也跟着从沙发上起身，走到裴金澈的跟前，没心没肺地说道。
<br/>
<br/>        夏玄灵顿时被这个女人气得七窍生烟，忍不住大爆粗口：“童筱颖！你别太过分了！”
