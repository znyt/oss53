　　实际上，叶少阳倒巴不得他们都在下面，这三个小鬼虽然都是恶鬼等级，但也不能长时间在阳光下行动，目前正接近正午，太阳正是毒的时候，那三个恶鬼如果赶来搞事，简直就是送死。
<br/>　　叶少阳和张道长一起移动到树荫下呆着，望着那一群小道士挖井。
<br/>　　“哇……”一声婴儿的啼哭，从刘四媳妇怀里抱着的孩子口中发出。
<br/>　　刘四媳妇一听就慌了，急忙抱着孩子去给张道长看，问他是怎么回事。
<br/>　　张道长查看了一下那男孩，说道：“没什么，就是饿了。”
<br/>　　“饿了？”刘四夫妻都呆住了。刘四嗫嚅道：“以前都是半夜才饿……”
<br/>　　张道长两眼一翻，说道：“饿了是好事啊，以前那几个鬼一直缠着他，今天不缠着他，他魂力恢复，暂时正常了，赶紧去喂吧。”
<br/>　　刘四媳妇一听，慌忙背过身，喂孩子吃奶起来。
<br/>　　“多谢道长，多谢道长……”刘四欢喜地对着张道长作揖打躬。
<br/>　　张道长瞥了叶少阳一眼，淡淡说道：“谢他就好了，跟我没关系。”
<br/>　　刘四一怔，顿时就卡壳了，僵在那里不知道说什么好，主要是想起了自己之前对他的那种不信任的态度，还有在张道长面前说的那些损他的话，哪里想到张道长眼下却自己改变了对他的态度……
<br/>　　叶少阳道：“也先别谢我。斗法才刚刚开始，万一打不过那三个小鬼，他们还是回来骚扰你儿子的。”
<br/>　　刘四一听，陡然紧张起来，哀求叶少阳和张道长，救救自己的孩子。
<br/>　　叶少阳安慰了几句，让他在一边等着就好。
<br/>　　刘老汉也过去安慰他，拍了拍刘四的肩膀，说道：“我怎么跟你说的来着，我这个内侄……法力高强，我没有说错吧？”说完还得意地看了叶少阳一眼。之前的尴尬和歉意，早就一扫而光了。
<br/>　　张道长的几个弟子轮流上阵，把井口挖开了一大截，好在这是一座石井，四周的井壁是用石头砌的，只要上沿打开之后，下去往外搬石头就可以了。
<br/>　　搬石头是个体力活，几个小道士累的满头大汗，花了差不多一个小时，总算是把大部分石头搬出去，扩大了石井的空间，让一个人在下面可以自如地行动。
<br/>　　在这个过程中，叶少阳一点力气没有出，他需要节省力气。等到一切具备，他从树荫下站起来，问张道长：“你来？”
<br/>　　张道长看了一眼被挖开的石井，咽了口唾沫说道：“我年纪大了，不惯爬上爬下，还是你来吧。”
<br/>　　叶少阳笑了笑，找刘四要了几根准备好的蜡烛，提着一把铁锨，跳了石井下面。
<br/>　　先用火折子把几根蜡烛点着，插在被挖开岩壁的泥土上，照亮了地面，是一堆混合着瓦烁的泥土。也没什么好说的，就是挖。
<br/>　　为了防止一铲子下去，把尸骨插成两截，叶少阳从最边上开始小心翼翼地挖起，每一铲子下去，都要检查一下。
<br/>　　起初挖起来还挺容易，但是挖到后面，铲子已经难以下去，在泥土中间，到处都是盘根错节的树根，自然是属于地面上那两棵槐树的，树根蜷曲在一起，很难能够挖得动。
<br/>　　另外，叶少阳注意到，当自己用铁锹铲断树根的时候，这些根须的末端都会渗出像血一样的液体，比血要淡一些。
<br/>　　叶少阳见怪不怪，知道是那两棵槐树已经成精了，不过也是刚成精，还没有“入道”，也就是脱离木质的身体，暂时是不会动的。
<br/>　　这样的树根，其实反而好对付。叶少阳朝上面喊话，让张道长帮忙弄点公鸡血来。张道长立刻吩咐刘四去办。然后拉叶少阳先上去，听他说了下面的情况，也是惊叹不已。
<br/>　　“那三个恶鬼，不在下面？”张道长问。
<br/>　　“当然不在，他们应该是在外摄魂修炼，入夜才会回归肉身。”
<br/>　　张道长表情有些奇怪地看着他，说道：“你究竟是哪里来的？”
<br/>　　“我不是跟你说过，我是茅山弟子。”
<br/>　　张道长当然记得他之前的话，狐疑地说道：“茅山没有三十九代弟子。”
<br/>　　叶少阳耸了耸肩，没有说什么，因为无法解释清楚。
<br/>　　张道长只当他是有什么苦衷，需要隐瞒自己的出身，也没有再问下去。
<br/>　　等了不到半个小时，刘四回来了，手里提着一个葫芦瓢，里面装的是热腾腾的公鸡血。
<br/>　　叶少阳用鸡血画符，一口气画了十几张，张道长在一旁看着他画符的手法，简直纯熟到了极致，绝不是自己可比，当场震惊。
<br/>　　叶少阳带着这些鸡血画的符，再度跳下石井，尽数贴在那些根须上，念动咒语。灵符立刻燃烧起来，倾泻而出的灵力，像是某种具有腐蚀性的液体，将这些根须尽数熔断。
<br/>　　等到树根被腐蚀到足够程度之后，叶少阳铁锨将蜷曲在一起的树根扒开，中间还有一团没有被腐蚀的树根，层层叠叠地裹在一起，也看不清里面有什么，只能感受到丝丝缕缕的尸气从里面渗透出来。
<br/>　　叶少阳猜测自己想找的东西八成就在这里了，于是让上面放下绳子，自己动手，把这个东西从井下吊上去，摆在地上，看上去像是一个巨大的蚕茧。
<br/>　　大伙立刻都围了上来，好奇地围观。
<br/>　　叶少阳把葫芦瓢里剩下的鸡血对着这个“蚕茧”倒了下去，没过一会，蚕茧就被腐蚀了。
<br/>　　叶少阳兴致勃勃地蹲在这个蚕茧前面，说道：“各位观众，见证奇迹的时刻就要到了……”
<br/>　　一伸手，将最上面一层树根扯掉，下面露出的东西，让大伙霎时间震惊了。
<br/>　　“啊！”刘四媳妇尖叫了起来，刘四急忙将她拉到一边去，用身体挡住了他的目光，自己也浑身哆嗦起来，不敢再看。
<br/>　　“这……太吓人了！这怎么可能！”刘老汉吓的一屁股坐在了地上，想跟刘四夫妻一样躲避开，但目光却从那一堆东西上离不开了。
<br/>　　那是三个婴儿！
<br/>　　不是腐尸，也不是骷髅，而是三具完好的婴儿的尸体：皮肤白里透红，表情恬静，每一个身上都穿着肚兜，三人抱在一起，看上去像是在熟睡中。
