　　张晓寒抬起右手，左手摸出一枚五帝钱，在手掌上用力画了×，鲜血从掌心溢出。
<br/>　　在场的人看到这一幕，都有些动容，法师这么做，就代表要跟对方进行生死之战，不死不休。
<br/>　　“叶少阳，今天你与我只有一个人能活着离开，你敢是不敢？”
<br/>　　出乎意料的，叶少阳摇了摇头。
<br/>　　众人一片哗然。都到这份上了，你居然不敢跟人赌命？
<br/>　　“幼稚。”叶少阳叹了口气，说道：“今天到这一步，本来就是你死我活，还需要这种幼稚的程序吗，这么大的伤口，多久能长好啊。”
<br/>　　一边叹气，一边摸出铜钱，在自己掌心也画了个×。
<br/>　　“装逼。”四宝和吴嘉伟异口同声。
<br/>　　张晓寒什么也没说，直接扑了上来，左手结印，掌心向上，五道黑气，从五个指缝中飞出，围住叶少阳，猛攻起来。
<br/>　　“五朝元气！”围观的人群中，爆发出惊呼。
<br/>　　总是有得道高人，自己虽然不会，但是一眼就看出这是道门至强法术五朝元气，顿时骇然不已。
<br/>　　“居然学会了五朝元气！怪不得法术公会如此看重他。”
<br/>　　云春生、紫云真人等几大门派的宗师站在了一起，观看这场决斗，眼看张晓寒一上来就使出了五朝元气，几人交换了一下眼色，神色都是暗淡不已。
<br/>　　他们早就暗中调查过张晓寒，虽然知道他实力极强，但是对他所修炼的法术却是一无所知，今日张晓寒也是有心卖弄，一上来就使出了五朝元气这种失传的法术，自然也是有心立威。
<br/>　　单凭这一手，他就有资格成为第一弟子、号召同门。
<br/>　　“春春，你这传人危险了啊。”紫云真人拍了拍云春生的肩膀，说道。
<br/>　　云春生知道他指的是叶少阳，道：“休要胡言，他不是我传人，是凌波子那边的。”
<br/>　　紫云真人哼了一声，道：“你就不要掩饰了，凌波子早就封山自修了，哪会有这么了不起的亲传弟子。我刚才观察半天了，这小子用的就是正宗的茅山法术，你不承认不要紧，不过你不该让他来送死，他若真的天赋异禀，你再调教数年，不是更好，你呀，太着急了。”
<br/>　　云春生道：“你这什么意思？”
<br/>　　“这张晓寒已经修成五朝元气，人间二代弟子以下，不可能有对手，你这传人气势汹汹，但今日只怕要死在这里，到那一刻，你也不好救场吧。”
<br/>　　云春生怔了怔，道：“去你妈的，我倒希望他是我传人呢。”
<br/>　　紫云真人也不生气，把几个老家伙都叫过来，围成一圈，压低声音说道：“春春这个传人，天赋异禀，诸位不是一直都在寻找能跟法术公会抗衡的年轻弟子吗，我看这小子能当此大任，我们一起好生调教，假以时日，未必不能与张晓寒抗衡。”
<br/>　　了尘禅师道：“阿紫，你想的倒是容易，但这小子与张晓寒下了死斗之约，怕是今日要葬身此地，没得救了。”
<br/>　　这几个老家伙都是一派宗师，平时在自己的宗门里都是被各种供奉的，说话办事都是方圆有度，规矩的很，但是没有外人的时候，这些老家伙之间也会互相调侃，说话总没正经。
<br/>　　紫云真人道：“你就是不开窍，要不是这，我跟你们聊什么，我的意思，等这小子战败之时，我们抢先出手，把他救下来……”
<br/>　　九华山的大衍禅师道：“不成不成，说好了是赌斗，到时候我们出手救人，法术公会一定不会善罢甘休。”
<br/>　　“大不了就是打一架，也不会真怎么样，只要我们这些老家伙一条心，法术公会断然不会真撕破脸皮，到时候说点好话，张晓寒气也消了，双方就此罢手，岂不是更好。”
<br/>　　大伙商量片刻，都觉得这办法可行。
<br/>　　大衍禅师对云春生道：“这小子是你家传人，到那时候，你得先站出来说话，我们都挺你就是了。”
<br/>　　“我真的不认识他，我骗你们干什么呀！”云春生一头郁闷。
<br/>　　“你就别掩饰了，事关法术界未来大计，我们并不嫉妒他是你茅山弟子。”大衍禅师语重心长。
<br/>　　其余几人也纷纷点头，支持这个观点。云春生实在无奈，又解释不清，只好答应下来。几人分开，再度朝战场上看去。
<br/>　　五道黑气，各自形成旋流，吸附在了叶少阳身上不同的部位上：头顶，双肩，还有两个手腕，都是命门所在。张晓寒的意图很明显，只要一道穴位被攻破，叶少阳体内的三魂七魄都将不保，但是叶少阳作法布置结界，异常稳固地抵挡着五朝元气的攻势。
<br/>　　“这不是真的五朝元气，是从佛门五方中推演而出的，是盗版的，你唬的住别人，唬不住我。”叶少阳一边作法，一边说道。
<br/>　　张晓寒心中一怔，道：“你如何得知？”
<br/>　　“我师兄以前就这么干过，后来他学会真五朝元气，才放弃这玩意，你现在玩的这个，我早八辈子就看过了。”
<br/>　　“你师兄……你师兄是什么人？不可能，天下不可能有真五朝元气！”
<br/>　　“我师兄，是你问不起的人。”
<br/>　　叶少阳摸出八枚五帝钱，朝前方打去。
<br/>　　“钱定八门、罡气云生，急急如律令！”
<br/>　　八枚五帝钱落在自己身前，形成了一个五角星的形状。
<br/>　　八门金锁阵！
<br/>　　八枚铜钱，各自射出灵光，将叶少阳周身护住，瞬间将五朝元气从身上驱开三寸距离，形成对峙。
<br/>　　紫云真人用胳膊怼了云春生一下，道：“还说不是的传人，这八门金锁不是你家嫡传，凌波子会这个？”
<br/>　　云春生怔怔无语。
<br/>　　一道人影，掠到云春生的面前，正是鬼姨。
<br/>　　“云老祖，你教的好弟子。”
<br/>　　云春生干咳一声，硬着脖子说道：“不错，这是我茅山传人。”
<br/>　　鬼姨一笑，道：“那看来今日之事，却是你策划许久的了。”
<br/>　　云春生义正言辞，昂然说道：“人间法术界，总不能让你们公会全占了，我茅山总是天下担当。”
<br/>　　紫云真人抢着说道：“实话跟你说，不光是茅山，今日之事，我们大家都有份的。天下担当，宗派当先，我等自然不能坐视不管！”
