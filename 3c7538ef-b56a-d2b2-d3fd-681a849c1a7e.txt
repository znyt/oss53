　　唐清山道：
<br/>　　“叶先生，清雅被保送进入了京都大学，这件事您应该知道吧？”
<br/>　　叶尘点了点头，此事他的确听下面的人汇报过。
<br/>　　唐清雅在高考之前，就被保送进入京都大学，就连叶尘的好兄弟吴磊，也考到了外省读书，和前世的人生轨迹，已经大为不同。
<br/>　　当然，如果唐清雅早知道，叶尘会来天海大学，说不定也不会远赴京都读书。
<br/>　　“清雅到底出了什么事？”
<br/>　　叶尘在心里对唐清雅，多多少少还是有些愧疚的，而且唐清雅也是他在地球上，最为重要的朋友之一。
<br/>　　唐清山道：
<br/>　　“我接到消息，就在今天，有人企图绑架我妹妹，幸好神龙卫的人及时识破了阴谋，清雅这才逃过一劫！”
<br/>　　“什么！”
<br/>　　叶尘听到这话，顿时脸色一寒，
<br/>　　“可查到是什么人干的？”
<br/>　　唐清山摇了摇头，
<br/>　　“根据神龙卫那边的报告，想要掳走清雅之人，修为极高，恐怕至少也是神境级别！”
<br/>　　“若非他们动用了高科技武器，恐怕早就已经全军覆没了！”
<br/>　　“哦？还有这等事？”
<br/>　　叶尘眉头一皱，
<br/>　　唐家最近一年，虽然强势崛起，可势力范围，毕竟还只是在国内。
<br/>　　放眼整个华夏国，神境以上的高手，不过寥寥数人，而这些人，也不可能会找唐家的麻烦。
<br/>　　“难道此人是冲着我来的？可是经过米国那一战，这世上还有哪方实力，敢公然和我作对？”
<br/>　　叶尘思索了片刻，也没有想出一个所以然。
<br/>　　“看来只能亲自去京都一趟了！”
<br/>　　唐清山闻言，顿时大喜过望，
<br/>　　“叶先生肯亲自出马，自然再好不过了！我这就让人去准备私人飞机，我陪您一起去！”
<br/>　　叶尘点了点头，表示同意。
<br/>　　……
<br/>　　唐清山的办事效率很高，打了一个电话过后，不到十分钟，便已经准备就绪。
<br/>　　两人离开包厢之后，再次走出酒店，
<br/>　　让叶尘没有想到的是，不但李虎三人，一直等在外面，就算那位美女警官江飞燕，竟然也在外头等候着。
<br/>　　“老大，老三，老四，你们回去吧！我有急事，要去京都一趟！”
<br/>　　李虎三人此时也早已经看出来了，知道自己这位室友，身份绝对不简单，于是点头应了一声。
<br/>　　一旁的唐清山当即笑道：
<br/>　　“三位小兄弟，我让人开车送你们回去，以后你们如果有什么需要帮忙的，尽管来这里找我！”
<br/>　　说到这里，唐清山朝身后打了一个响指，一名气势凌厉的中年男子快步走了上来。
<br/>　　“传我的命令下去，以后这三位小兄弟，就是我们帝豪大酒店的顶级贵宾，若是有人胆敢怠慢，我唯你是我！”
<br/>　　那中年男子当即点头称是。
<br/>　　李虎三人听到这话，顿时个个大喜过望，
<br/>　　唐清山何许人也，那可是天南第一大家族的太子爷，有他这一句话，他们三个以后在整个天海，都可以横着走了！
<br/>　　当然，他们也都清楚，唐清山之所以会这么做，无非还是看着叶尘的面子。
<br/>　　这就让他们三人更加好奇，自己这位室友，究竟是何方神圣？
<br/>　　送走了李虎三人之后，一旁的江飞燕这才走了上来，
<br/>　　“叶先生，我……”
<br/>　　江飞燕才刚刚开口，叶尘直接摆了摆手道：
<br/>　　“江警官，我知道你想说什么，我既然已经放过你弟弟，那就不会再继续追究，你让他以后好自为之吧！”
<br/>　　说完这话，叶尘便直接弯腰坐进，唐清山的座驾之中，两人直奔机场而去。
<br/>　　望着飞驰而去汽车，江飞燕虽然达到了自己的目的，可是不知道为什么，心里却一点也高兴不起来。
<br/>　　她自问也算是天海市有名的大美女，没想到也会有被人完全无视的一天。
<br/>　　叶尘从头至尾连看都没看她一眼，让她第一次对自己的美貌产生了怀疑。
<br/>　　她又哪里知道，叶尘此时想着唐清雅的事情，哪里有时间和他多说。
<br/>　　……
<br/>　　叶尘和唐清山来到机场之后，直接乘坐私人飞机，连夜朝京都方向赶去。
<br/>　　大约只用了两三个小时，飞机便已经在京都降落。
<br/>　　神龙卫的人听说叶尘要来，早已经在机场等候多时。
<br/>　　叶尘大致问了一些情况，便又直奔京都大学而去。
<br/>　　很快，
<br/>　　车子来到京都大学附近，缓缓停了下来，唐清山道：
<br/>　　“叶先生，如今天色已晚，我们现在附近找一家酒店住下吧！”
<br/>　　叶尘摇了摇头，
<br/>　　“不！对方白天被你们击退，晚上说不定还会行动，我们不妨来了一个守株待兔！”
<br/>　　“你们让周围的兄弟都撤回去吧，这里有我一人足矣！”
<br/>　　说完这话，叶尘直接推开车门，下了车，然后身体微微一晃，便已经凭空消失不见。
<br/>　　车上的唐清山等人，顿时相顾骇然，当即开始向各自的手下传达命令，让守候在京都大学附近的人马纷纷撤离。
<br/>　　……
<br/>　　京都大学，女生宿舍，
<br/>　　唐清雅此刻正躺在床上，辗转反侧，难以入睡。
<br/>　　并不是因为，今天有人想要掳走她，而是因为另外一件事情。
<br/>　　她今天无意间，在网上看到天海大学的一则新闻，竟然看到了叶尘和萧若曦，两人卿卿我我的照片。
<br/>　　而且据说，两人现在已经是天海大学，最有名的一对情侣！
<br/>　　唐清雅得知这个消息之后，整个人顿时就不好了，一直到现在，都还有些难以接受。
<br/>　　就在唐清雅正暗自伤心之时，
<br/>　　“嘎吱~”
<br/>　　宿舍的窗户打开，继而一道黑影从外面窜了进来。
<br/>　　“谁！”
<br/>　　唐清雅顿时娇喝一声，霍然坐起身来，死死的盯着那道黑影。
<br/>　　奇怪的是，她的其他三名室友，却依旧在沉睡，居然没有被吵醒，显然已经被对方施展了什么手段。
<br/>　　“桀桀~”
<br/>　　那道黑影，顿时发出一道沙哑难听的笑声，
<br/>　　“唐小姐，我们又见面了！”
<br/>　　“是你！”
<br/>　　唐清雅听到这道声音，顿时感觉浑身充满了寒意，因为眼前这个人，正是白天企图要将她掳走的那个人！
<br/>　　虽然，唐清雅在叶尘的指导下，已经迈入修真行列，可是如今也不过才炼气五重境界而已，远远不是眼前这人的对手。
<br/>　　“你到底想要干什么？”
<br/>　　那黑衣再次桀桀一笑，
<br/>　　“唐小姐莫怕，我只是想请你去我们蓬莱仙界，去做几天客罢了！桀桀！”
