　　苏莫和吕罡两人来的较早，距离拍卖会开始，还有一段时间，两人闲聊了几句之后，苏莫便在包厢中闭目养神了起来。
<br/>　　这次拍卖会，他不为购买宝物而来，只是单纯的要售卖宝物。
<br/>　　当然，若是有什么适合他的奇珍异宝出现，他也会参加竞拍。
<br/>　　时间缓缓流逝，拍卖场中人越来越多，逐渐的热闹了起来，人声鼎沸。
<br/>　　约莫过去了大半个时辰之后，整个拍卖场已经座无虚席，这个时候，拍卖场的大门关闭，拍卖会即将开始了。
<br/>　　又过了片刻之后，一名老者大步的走到了拍卖台上。
<br/>　　这名老者身穿藏青色长袍，年过古稀，面容苍老，正是龚忠。
<br/>　　苏莫睁开了眼眸，见到龚忠上台，微微有些诧异，此人是六壬商会的长老，居然还兼任拍卖师。
<br/>　　“诸位，欢迎来到我六壬商会举行的拍卖会，老夫很荣幸担任此次拍卖会的拍卖师！”龚忠环视四方，朗声说道。
<br/>　　“龚长老乃是古桐城有名的拍卖师，参加您老的拍卖会，也是我等的荣幸！”
<br/>　　“龚长老，还是赶紧开始吧！”
<br/>　　“龚长劳，听说此次拍卖会，有重宝出现，不知是否为真？”
<br/>　　下方的坐席上，大量的武者纷纷开口，声音吵杂不已。
<br/>　　“诸位请安静！”
<br/>　　龚忠轻喝一声，声音如同雷霆炸响，直接在众人的脑海中炸开，整个拍卖场顿时安静了下来，变得落针可闻。
<br/>　　这种拍卖会，拍卖师必须修为强大，不然根本镇不住场。
<br/>　　“诸位，此次拍卖会宝物众多，很多都属于重宝的范畴，想要得到宝物，还请大家踊跃竞拍！”
<br/>　　说到此处，龚忠停顿了少许，又继续道：“既然大家都已经迫不及待，那老夫也不再多言了，拍卖会正式开始！”
<br/>　　龚忠话音一落，一名高挑的侍女端着长长的木质托盘，从后方走到了拍卖台上。
<br/>　　托盘上，盖着鲜红的丝绸，阻挡了气息和视线，让人无法看清，这是什么宝物。
<br/>　　唰！
<br/>　　龚忠二话不说，一把掀起丝绸，露出了其内的一柄金色长剑。
<br/>　　“第一件拍卖品，尊级中品宝剑，经过我六壬商会的鉴定师坚定，此剑品质上佳，在尊级中品宝器之中，也算是顶尖的存在！”
<br/>　　龚忠面色肃穆，话音一转，继续道：“现在开始竞拍，起拍价五十块中品灵晶，每次加价不得少于十块中品灵晶！”
<br/>　　龚忠话音一落，整个拍卖场中，立刻充斥着此起彼伏的竞价声。
<br/>　　“五十五块中品灵晶！”
<br/>　　“六十块中品灵晶！”
<br/>　　“六十五！”
<br/>　　“……！”
<br/>　　尊级中品宝器，虽然不是多么的珍贵，但对于一些小势力的武者而言，已经是不可多得的宝物了，所以竞拍的人也不少。
<br/>　　二十九号包厢中，苏莫面上露出了一丝笑容，因为这柄金色长剑，正是他的拍卖品。
<br/>　　这柄剑，乃是他之前的兵器，在得到会武奖励的石剑之前，他一直用这柄金色长剑。
<br/>　　让他更加惊喜的是，这里宝器的价格，比之古灵星河要高的太多了。
<br/>　　虽然苏莫不了解古灵星河之内，尊级中品宝器的具体价格，但他也知道绝对不会有这么高的起拍价。
<br/>　　或许是明虚星河的灵晶资源比较富足，价格才会远远高于古灵星河吧！苏莫心中暗想。
<br/>　　十号包厢之中，扬天成、杨天威，以及那名绿衣女子高师妹三人，正静坐其中，三人并没有参与竞拍，因为尊级中品宝器，还入不了他们的法眼。
<br/>　　他们三人财大气粗，即便没有请帖，也可以用三百中品灵晶，而获得一间包厢。
<br/>　　此刻，扬天成望着拍卖台上，那柄金色的长剑，眉头微微一皱。
<br/>　　“天成师兄，你怎么了？”绿衣女子见此，不由得疑惑问道。
<br/>　　“这柄剑好像在哪里见过？”扬天成疑惑的说道。
<br/>　　绿衣女子闻言，转头打量了一番拍卖台上的那柄金色长剑，轻笑道：“这就是普通的尊级中品宝剑，很常见啊！”
<br/>　　绿衣女子并未在意，这柄剑模样普通，很大众化，她都见过很多次。
<br/>　　“或许吧！”扬天成微微点了点头，也没有多想，这种宝剑的确比较常见。
<br/>　　第一件拍卖品，没有持续多久，便以一百二十快中品灵晶的价格成交了。
<br/>　　包厢中，苏莫面露微笑，一件尊级中品宝器，便卖出了一百多块中品灵晶的高价，那尊级上品呢？
<br/>　　还有那三支被称为伪圣器的庚金破虚箭，岂不是能拍出惊人的价格？
<br/>　　第一件拍卖品结束之后，第二件拍卖品很快上台，竞拍继续进行。
<br/>　　不过，第二件拍卖品，并不是苏莫所有。
<br/>　　包厢中，有侍女送上了灵茶，苏莫一边品尝灵茶，一边观看着拍卖会，显得颇为悠哉。
<br/>　　拍卖台上，各种宝物如同流水一般，不断的被人拍取。
<br/>　　苏莫的宝物，亦是不时的出现，只不过，尊级下品的宝器，价格极低，没有一件能拍出十块中品灵晶以上。
<br/>　　但是，虽然价格不高，但甚至数量较多。
<br/>　　随着时间的不断流逝，苏莫的宝物，一件有一件的被拍卖。
<br/>　　约莫三个时辰之后，就被拍卖了一大半以上。
<br/>　　期间，苏莫没有参加竞拍，一般的宝物他根本看不上眼，即便是尊级上品的宝器，他也不需要。
<br/>　　吕罡倒是成功拍得了一双尊级上品拳套，花费了多达两千二百中品灵晶，几乎耗尽了他身上的财富。
<br/>　　“下面，将进行下一件竞拍！”
<br/>　　龚忠朗声高喊，随即后台再次走出了一名侍女，手中拖着长达一丈的托盘。
<br/>　　托盘上，同样盖着一张鲜红色的丝绸，遮盖了宝物的面貌。
<br/>　　唰！
<br/>　　龚忠雷厉风行，没有多少废话，直接便一把掀起了丝绸，一柄金色的大弓，便出现在了众人的眼中。
<br/>　　“尊级上品宝器，金裂弓，品质绝佳，底价五百块中品灵晶，每次出价不得低于一百中品灵晶，现在开始竞拍！”龚忠朗声高喝。
<br/>　　这柄金色大弓，正是苏莫送来的竞拍品，弓身之上刻有名字，名为金裂弓。
<br/>　　“六百中品灵晶！”
<br/>　　“七百五十块中品灵晶！”
<br/>　　“八百五十块中品灵晶！”
<br/>　　“……”
<br/>　　霎时之间，出价声不绝于耳，现场显得有些火爆了起来。
<br/>　　从拍卖会开始，到目前为止，出现的尊级上品宝器，不超过二十件，每一件都是抢手货。
<br/>(本章完)
