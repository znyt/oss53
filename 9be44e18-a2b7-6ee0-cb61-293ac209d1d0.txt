        因为，季思妍在输了官司后，来澳洲见了孩子最后一面，当晚在酒店里就割腕自杀了……
<br/>
<br/>        但是，她没死成功……
<br/>
<br/>        傅昼景及时赶到酒店，救了她，将她送去了医院，但至今昏迷不醒。
<br/>
<br/>        童筱颖带着季小景，也就是冷尚轩在家中，坐立不安。
<br/>
<br/>        小尚轩依旧唤她“妈咪”，却不知道他的这个“妈咪”，差点逼死了他的亲生妈妈。
<br/>
<br/>        这些天，傅昼景一直守在医院里，陪伴在季思妍的身边，因为不敢通知季思妍的父母，所以他在医院里照顾季思妍都是自己亲力亲为。
<br/>
<br/>        童筱颖忽然觉得，自己更像电视剧里的恶毒女配。
<br/>
<br/>        在男主角耳边煽风点火，逼死纯情女主角，夺走女主角的孩子。
<br/>
<br/>        其实，她好像什么事情都没有做。
<br/>
<br/>        要非得说她做了什么事情，那么唯一的一件事情，就是在法庭上，当面向法官允诺，她还爱着她的丈夫，愿意给丈夫生个孩子，去挽救这个私生子的生命，并将私生子视为己出。
<br/>
<br/>        法官综合各方面有利于孩子的健康成长考虑之后，才将小尚轩判给了傅昼景。
<br/>
<br/>        官司胜出的那晚，童筱颖先哄小尚轩睡着后，自己才去洗干净了、释然了、脱光了，一脸娇羞地躺在床上等待着他傅昼景的“宠幸”。
<br/>
<br/>        只为了给他傅昼景生个孩子，去延续小尚轩的生命。
<br/>
<br/>        但是，那天晚上，季思妍一通电话，就把傅昼景给叫走了。
<br/>
<br/>        童筱颖心知肚明，季思妍不是真的想死。
<br/>
<br/>        季思妍若想死，何必又在割腕自杀前掐准了时间给傅昼景打那通视频电话？
<br/>
<br/>        “筱颍，我们离婚吧！”
<br/>
<br/>        苍白无力的语调，伴随着有些疲乏的声音，从傅昼景的口中说出。
<br/>
<br/>        傅昼景刚从医院里回来，鞋都未脱，直接来到了厨房门口。
<br/>
<br/>        童筱颖正站在厨房里的洗涤盆前择菜，手下的动作微微一滞。
<br/>
<br/>        她没回头，收了收心思，继续着手中的动作。
<br/>
<br/>        傅昼景不再吭声，伫立在那儿，默默地等待着她的答复。
<br/>
<br/>        童筱颖淡定地将所有的蔬菜都洗得一干二净，因为这些菜都是小尚轩要吃的，所以她才会洗得格外勤，洗得格外仔细。
<br/>
<br/>        明明不是她亲生的孩子，她却对这个孩子百般呵护。
<br/>
<br/>        知道这是为什么吗？
<br/>
<br/>        其中的原因，很简单。
<br/>
<br/>        爱屋及乌。
<br/>
<br/>        她爱小卫，愿意为小卫不求回报的付出，包容小卫的一切，所以连带着也会爱护小卫的这个孩子。
<br/>
<br/>        可惜，小卫不懂。
<br/>
<br/>        童筱颖慢条斯理地关掉了水龙头，转过身来，对着站在厨房门口的傅昼景，面无表情地点了点头，淡淡地应了声：“好。”
<br/>
<br/>        一个星期不见，一个电话不打，彼此的沉默，将彼此的这段婚姻，推向了尽头。
<br/>
<br/>        傅昼景未料到童筱颖会答应得如此爽快，心里又莫名其妙地升起了一丝不甘。
<br/>
<br/>        “你应该主动打个电话的。”傅昼景深沉地说道，语气里满是埋怨。
<br/>
<br/>        童筱颖抬眸看着傅昼景，心痛地反问：“你要我打电话问什么？”
<br/>
<br/>        “筱颍，你以前不是这样的。”傅昼景满腔酸涩地说。
<br/>
<br/>        “那我以前，应该哪样？”童筱颖敛了敛那痛苦的滋味，心平气和地问。
<br/>
<br/>        以前的筱颍，温柔体贴、善解人意、小鸟依人、楚楚动人……
<br/>
<br/>        他至今都还记得，他因为负债，不想连累她，第一次跟她提出“离婚”的时候，她神情恍惚，害怕得语无伦次。
<br/>
<br/>        甚至，她含着眼泪，楚楚可怜，却依旧坚定地对他说：“我不离婚！”
<br/>
<br/>        当时的筱颍，多么令他心疼，多么令他感动。
<br/>
<br/>        而如今……
<br/>
<br/>        “算了！”傅昼景深沉地叹息。
<br/>
<br/>        对于离婚这件事情，童筱颖只有一个要求，那就是回云城西镇的民政局去领离婚证。
<br/>
<br/>        他们两是从哪里开始的，童筱颖便想从哪里结束。
<br/>
<br/>        季思妍出院后，经过傅昼景的许可，带走了小尚轩。

<br/>        小尚轩和童筱颖分开前，紧紧地抱住了童筱颖的大腿，因为不愿意跟季思妍走而哇哇大哭起来。

<br/>        就连季思妍也难以置信，自己的亲生儿子，竟然会对一个才养了他不到两个月的女人如此依依不舍。

<br/>        其实，对于孩子而言，陪伴是最长情的告白。

<br/>        童筱颖不知道季思妍平常是怎么带小尚轩的，但是，她能感觉到，小尚轩不仅仅缺乏安全感，而且还缺乏陪伴与关爱。
<br/>
<br/>        就在小尚轩赖在童筱颖身边，不愿意离开的时候，季思妍突然恼火地瞪红了眼，踱步上前，伸手扯着小尚轩的衣领，开始动手打小尚轩。
