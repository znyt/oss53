        &lt;h3&gt;第三卷第五十六章  财叔的忧虑&lt;/h3&gt;
<br/>
<br/>        财叔已经为陈子昂带了八年的孩子。
<br/>
<br/>        他本来是落家的人，跟着落家家主已经很多年了。落家家主将小小姐接回来之后，他就负责小小姐的一切，小小姐的安全都是由他来负责。
<br/>
<br/>        小小姐的吃喝用度是春姨负责，原本小小姐走哪里他们就跟到哪里，可是小小姐带回了陈子昂，将陈子昂安顿在落家后，他们也就在落家安了家。
<br/>
<br/>        小小姐心地善良，带回来一个孕妇，他们都知道小小姐在撒谎，可是谁也不愿意去戳穿她，为什么不让她一直善良下去呢？
<br/>
<br/>        落家不在乎一个人的吃喝。
<br/>
<br/>        当天，他就将事情汇报给落家家主了，老板让他调查陈子昂的一切。
<br/>
<br/>        这一调查，老板就收留了陈子昂。
<br/>
<br/>        而且他和春姨成了陈子昂的佣人。
<br/>
<br/>        孩子生下来之后他们是欢喜，看着俩个能萌化老夫心的小宝宝，他和春姨爱不释手，从此他们就是团团和嘟嘟的爷爷奶奶。
<br/>
<br/>        陈子昂忙于事业，他们是看着这个孩子是如何努力的，越发的对着俩个孩子疼爱有加了。
<br/>
<br/>        他负责接送孩子上学，春姨负责孩子的吃喝拉撒。
<br/>
<br/>        俩个孩子也很懂事，从来都不让他们分心。
<br/>
<br/>        上幼儿园，上小学都是他来接送，他来为孩子办理一切，包括孩子的户口都是他来办理的，陈子昂还是太年轻，只知道赚钱，不知道给孩子办理户口，不知道孩子的户口是有多么的重要。
<br/>
<br/>        孩子生在落家，长在落家，嘟嘟又姓了落姓，孩子户口当然就落在了落家，落家无形之中多了俩个孩子。
<br/>
<br/>        生在大户人家的孩子都身不由己，就像是团团和嘟嘟的父亲一样，他们的婚姻都是有父母亲做主，哪里能轮到自己做主啊，他家小小姐也如此，当初老板硬是用陈子昂母子三人的秘密威胁小姐，小姐才嫁了人，可是嫁过去之后不久，小小姐也不见了踪迹，这几年落家故意隐藏小姐的行踪，陈子昂也从落家搬出来了。
<br/>
<br/>        看着陈子昂一个人带着孩子，虽然说自己买了房，置办了产业，可是一个女人带着孩子还怎么做事情呢？
<br/>
<br/>        况且这些年他们已经习惯了俩个孩子的吵吵闹闹，已经不舍和他们分开了，于是和春姨一商量，俩个人就跟着陈子昂搬到了陈子昂的新家，给陈子昂当起了保姆。
<br/>
<br/>        而老板也是支持的，当初他将他的想法给老板说出来的时候，老板是这样说的：“子昂在我们家已经住了这么久了，嘟嘟已经姓了落，也算是我们家里的人，她搬出去我本来是不同意的，但是人家也有自己的生活，你去帮着照顾一下也是好的。将俩个孩子一定要照顾好。”
<br/>
<br/>        俩个孩子虽然没有父亲，但是过得很幸福，很开心。
<br/>
<br/>        团团永远一副故作高深老气横秋的模样，嘟嘟永远都是调皮捣蛋的一个，团团的老气横秋总是会被妹妹给了破了功。
<br/>
<br/>        带着俩个孩子，他心里也是幸福的很呢。
<br/>
<br/>        刚刚看见俩个孩子和张家的人打招呼，他就急速的叫走了孩子。
<br/>
<br/>        财叔是落家的人，自然是什么人都知道的，那颗藤结什么瓜，他都一清二楚。
<br/>
<br/>        他也知道张函偷偷的见过了团团和嘟嘟，只是张函毕竟是孩子的父亲，不会对孩子形成威胁，也不会和陈子昂抢孩子，但是张家的人就不一定了。
<br/>
<br/>        看来是必须要给孩子们叮咛一番了。
<br/>
<br/>        “团团，嘟嘟，爷爷交代你们的事情你们都忘记了吗？”财叔严厉的说。
<br/>
<br/>        “爷爷，我刚刚是给那个奶奶捡东西，她的东西丢了。”团团有些委屈的辩解。
<br/>
<br/>        “是啊，爷爷，我也是看到了，那个奶奶的东西掉地上了哥哥捡起来还给她的呢。”嘟嘟罕见的跟哥哥站在一起。
<br/>
<br/>        “嗯，是这样啊，那还了东西就赶紧离开啊，为什么还要站在那里不走呢？”
<br/>
<br/>        “我们刚说了两句话，您就喊我们啦，我们没有和陌生人说话的。”
<br/>
<br/>        团团和嘟嘟已经过了说黄八的年纪了，如今是小学二年级的学生了，说起话来头头是道。
<br/>
<br/>        “那就好，你们要记得，坏人脸上是不会写字的，尤其是看着漂亮的人，有些小孩子就是那样被人拐跑了的。”
<br/>
<br/>        “爷爷，您放心，只有我们拐别人，别人是拐不走我们的啦。”团团稚嫩的脸上满是自信。财叔却是不能将他的担忧说出来，只好对自己说，以后还是要跟紧两个孩子，尽量不要和张家人接触。
<br/>
<br/>        其实，他们就是偶遇，谁也没有想到他们会碰见，而且弥青云也不认识俩个孩子。
<br/>
<br/>        世界说大不大，说小不小，总之这个世界上有些事情是躲不掉的，尤其是血缘关系。
<br/>
<br/>        财叔带着俩个孩子回去，将在路上碰见弥青云的事情给春姨说了。
<br/>
<br/>        “张函结婚这么多年，都没有孩子，一定是有问题的。我是怕张家以后会和子昂争孩子。”财叔有些担心，俩个孩子是他们带大的。
<br/>
<br/>        “张函不是都见过俩个孩子吗？这都好几年了，也没有见过他想要孩子的抚养权啊。”
<br/>
<br/>        “张函是爱着咱家子昂的，可是他的家人就不一定了，张函他们结婚七年都没有孩子，张家如果知道子昂生了一对双胞胎，估计是会想办法要一个回去的。”财叔担忧的说。
<br/>
<br/>        “那可不行，咱不能让俩个孩子分开，要不给老爷说一下？”
<br/>
<br/>        “给老爷说了也不顶用啊，如果张家强行要孩子的话，张函那边不能生育的话，法律上也许会偏向他们的。”
<br/>
<br/>        “那怎么办啊？不过嘟嘟不是已经姓落了吗？”春姨说。
<br/>
<br/>        “也是啊，走一步算一步吧，也许他们都不会知道呢。我呢最近找个时间见一见张函。”
<br/>
<br/>        “你见他干什么啊，事情没有挑明还好说，挑明了就不好说了。”
<br/>
<br/>        “我要告诉他，不要想着从咱家子昂手中将孩子抢过去，他们不配！”
<br/>
<br/>        “你看你，都多大年龄了，还如此的毛毛躁躁的，人家这不是还没有抢吗？好了好了，歇着去吧，车到山前必有路，别杞人忧天了。”春姨笑着说。
<br/>
<br/>        “你说真到了那个时候怎么办？”
<br/>
<br/>        “你傻呀，咱家子昂现在是越来越强大了，生意也做的很好，如果他们真的知道了，咱俩带着孩子远走高飞不就行了？”
<br/>
<br/>        “这也成？”
<br/>
<br/>        “怎么不成了，到时候带着孩子出国也行啊。”
<br/>
<br/>        “可是咱家都在国内，子昂也在国内呢，就我们俩个人带着孩子出国？”
<br/>
<br/>        “唉，我说老财头，你今天有完没完啊？张家现在不是还不知道孩子的事情吗？”春姨有点不耐烦了。
<br/>
<br/>        “我这是不是担心吗？咱家子昂势单力薄的。”
<br/>
<br/>        “没事的，别想那么多，对了子昂回来了你不准胡说啊，不要给她增添负担，让她好好的干一番事业，让张家那些势利眼后悔死。你去超市买些菜回来，子昂晚上要回家吃饭呢。”
<br/>
<br/>        “好吧。”财叔泱泱的去买菜了。
<br/>
<br/>        陈子昂出差回来了，晚上要回家吃饭。
