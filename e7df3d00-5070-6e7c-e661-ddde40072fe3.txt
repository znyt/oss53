　　墓道很宽，可以两三个人并排，于是妙心让叶少阳掌灯，跟在自己身边，朝墓穴下方走去。
<br/>　　结果刚走没几步，大伙就被墓道两旁的东西所吸引了：
<br/>　　在墓道的两旁，每隔一段路程，就有一块岩壁被掏空，里面矗立着一座座的雕像，大小跟真人差不多，每一尊都不一样，而且形象都很诡异：有的是人头蛇身，有的是三头六臂，有的干脆就是一只长角的猛兽……
<br/>　　所有的形象，不光是模样凶神恶煞，最特别的是，这些雕塑的眼珠子都是宝石镶嵌，在火光照耀下熠熠生辉，好像一双隐藏在黑暗中的眼睛，一直盯着人看，不知道为什么，叶少阳总觉得这些雕像的眼神很邪恶。
<br/>　　雕像也有眼神？
<br/>　　叶少阳心中有些不安起来。
<br/>　　“这些是什么神，佛门中的吗？”卢晓清好奇问道，转头去看梧桐，梧桐是峨眉山俗家弟子，对佛像自然是了如指掌，她仔细看了一会眼前的这些雕像，摇头说道：“雕塑的形态是像佛门的，不过这些神祇，不是佛门中的。看上去像是苯教的……苯教是一种萨满教，供奉着很多凶神，这些看上去有点像。”
<br/>　　叶少阳听见“苯教”两个字，立刻就想到了自己当初在西域沙海之中探索过的那座古墓，本来以为下面葬的是苯教的某位大祭司之类的，哪里想到最后挖出来的是白起……
<br/>　　叶少阳收回心思，盯着这些神像，不解地说道：“这些神像的眼睛，为什么用发光的珠子代替？”
<br/>　　妙心说道：“这好像是苯教的一种传统，认为用宝石为神像开眼，神像就可以通灵，能够保佑供奉者。不过据说这只是苯教中一个分支的作法，这个分支住在西域的石头城，城里很容易找到这种晶莹剔透的石头，在黑暗的地方被光线一照，就像是眼睛一样了。别的地方没有这种石头，所以教徒们没有这种习惯，都是因地制宜。”
<br/>　　叶少阳听她一口气说了这么做，不由大感意外，很想问她为什么会知道这么多，又怕被人嘲讽，只好将这问题咽了回去，问妙心：“你不是说这里是南宋皇族子孙的墓吗？为什么又跟苯教扯上关系了？”
<br/>　　“这个我也不清楚，只有走下去看看了。”
<br/>　　一行人顺着墓道继续往下走，前方出现了一个转弯，墓道向右边弯过去，继续往下。叶少阳打着三色莲花灯额走在最前头，刚要迈步下去，身边妙心突然拉了他一把。叶少阳转头看去，却发现她在盯着墙上的雕像发呆。
<br/>　　“怎么了？”有人问道。
<br/>　　妙心手指着雕像的脸，说道：“这雕像笑得好吓人。”
<br/>　　大伙这才注意到雕像的脸，的确，这具四臂八手的雕像，嘴角是向上翘起来的，微微低头，眼神十分阴沉。就这么看过去，有一种雕像上的人在看着他们冷笑的感觉。
<br/>　　这笑容，让在场的人都感到很不舒服。
<br/>　　“像这种诡笑的雕像，我还是第一次见到……”妙心喃喃说道，饶是她见多识广，也从来没有见过眼前这种形态的雕像：雕像大多数都是没表情的，还有威严的、慈祥的，有凶神恶煞的，只是从来就没见过会笑的。
<br/>　　大伙也是面面相觑。
<br/>　　叶少阳想起之前好像见过类似的雕像，想要走回去看一看，于是提着三色莲花灯往回走去，对那些雕像逐个照过去，刚看到第一个，他就愣住了，接着想到什么，又赶忙走向另一座雕像，又愣了一会，走到第三个雕像前面，看了半晌之后，这才站住了，怔怔地望着面前的雕塑，失声说道：“怎么会这样啊。”
<br/>　　“怎么了，神神叨叨的。”大伙感到好奇，一窝蜂朝他走了过去，然后循着他的目光，朝面前那尊雕像望去，一时间大伙也没注意到什么，忙问叶少阳怎么了。
<br/>　　叶少阳把三色莲花灯提到雕像的面前，吸着气说道：“看看！”
<br/>　　大伙借着火光，仔细观察起来，梧桐第一个叫出来：“这神像在笑！”
<br/>　　没错，眼前这尊神像，跟之前看到那尊四臂八手的神像一样，嘴角都带着一抹似有若无的微笑，再配上那炯炯有神的两束目光，直看得人心里发虚。
<br/>　　“怎么回事？”陈晓宇叫起来，“之前我好像没看到这雕塑在笑啊，好像是一脸严肃的，你们说呢？”
<br/>　　大伙都记不清楚了，互相看着，不敢妄下定论，只有叶少阳心如明镜，也不说什么，直接拉他们往来路走，每遇到一尊雕像，手中的莲花灯就对着脸庞照过去。
<br/>　　当参观到第三尊神像的时候，在场所有人都明白了叶少阳让他们看什么：这些神像，无论之前是表情严肃的还是凶神恶煞的，现在只有一个表情：都在笑。
<br/>　　所有的神像，都在看着他们笑。这笑容带着一些讥诮和嘲讽，仿佛在笑话他们看不透背后的真相。
<br/>　　这不可能再是巧合了……
<br/>　　大伙面面相觑，说不出话来。
<br/>　　“怎么会这样，这些雕像，难道是活的不成！”卢晓清失声道。
<br/>　　“我就不信这个邪！”陈晓宇吹了声口哨，命令他那鬼面猴子爬上一尊雕塑的脸上，把那两只镶嵌在眼睛上的发光石头抠下来。
<br/>　　妙心觉得不妥，想要阻止的时候已经晚了：两颗宝石被抠了下来，那两颗好像瞎子一样的眼窝里，连五秒钟都没到，就喷涌出了红色的液体，顺着脸颊流下来，宛如两行血泪。
<br/>　　大伙一下子就怔住了，急忙后退，仔细看去，这“血泪”顺着雕像一直流下来，在地上堆积起来。
<br/>　　大伙凑近了看，这才发现红色的并不是液体，而是跟流沙一样细小的颗粒，从雕像的眼中源源不断地流出来，很快就在地上堆起了高高的一堆。
<br/>　　妙心呵斥大家退后，自己找符纸从地上挑了一点，凑近鼻子边上闻了闻，道：“毒砂，还好，这是防盗墓贼的手段。”
<br/>　　墓中用毒砂一类的东西来对付盗墓贼，实在是太常见了。
