        童筱颖和季思妍聊得情投意合，季思妍也会把自己身边所发生的有趣的事情说给童筱颖听。
<br/>
<br/>        两人说着说着，季思妍突然提议道：“筱颖，不如你给我当‘妹妹’吧！我有个妹妹，年龄和你一般大，但是我和她不亲。难得我两这么有缘分，而且又聊得来。所以，我想你当我的妹妹。”
<br/>
<br/>        “当你的妹妹？”童筱颖一脸懵然，她们这才认识好像还不到几个小时，就直接当起了“姐妹”？
<br/>
<br/>        “对啊！我们义结金兰，结为异姓姐妹！”季思妍一把拉过童筱颖的手，红唇微扬，弯弯的眼睛里满是憧憬。
<br/>
<br/>        童筱颖愣了一下，不知道该怎么接季思妍的话，她总觉得这事儿哪里怪怪的，却又说不出来的感觉。
<br/>
<br/>        她和季思妍并非第一次见面，而第一次见面的时候，季思妍对她可不是现在这么亲热的态度。
<br/>
<br/>        “你不作声，我就当你答应了噢！”季思妍自顾自地接着说道，“以后，你就叫我‘姐姐’，我直接叫你‘妹妹’，好不好？”
<br/>
<br/>        “……”童筱颖有点儿语塞，表情也有些尴尬。
<br/>
<br/>        季思妍仍旧可以滔滔不绝地找话题聊下去。
<br/>
<br/>        童筱颖开始变得很被动起来。
<br/>
<br/>        总而言之，她两越是聊得热火朝天，露台那边，躺在凉椅上的傅昼景，听得心里越是不爽。
<br/>
<br/>        现在，他有些反感季思妍了。
<br/>
<br/>        季思妍明明知道筱颖是他的合法妻子，她还若无其事地和筱颖套近乎，很明显是居心叵测。
<br/>
<br/>        傅昼景看了看手机上的时间，觉得差不多了，便从露台的凉椅上起身，走进屋内，毫不留情地下逐客令：“季秘书该回自己房间去休息了！”
<br/>
<br/>        “啊！瞧瞧我这不懂事的！确实该回去了！妹妹，你和傅总也早些休息啊！晚安！”季思妍说完后抿嘴一笑，从沙发上起身的时候，偷偷地瞄了傅昼景一眼。
<br/>
<br/>        傅昼景并未看她，而是将目光落在了童筱颖身上。
<br/>
<br/>        “你也早点儿休息！”童筱颖跟着起身，将季思妍送到了房门口。
<br/>
<br/>        “拜拜，晚安！”季思妍说“拜拜”的时候，是看着童筱颖说的，而说“晚安”时，她的目光却落在了童筱颖的身后。
<br/>
<br/>        正巧，傅昼景朝这边看来，与季思妍四目相对。
<br/>
<br/>        季思妍不经意间地嘴角上扬，向傅昼景暗送秋波后，转身离去。
<br/>
<br/>        傅昼景却皱起了眉头。
<br/>
<br/>        童筱颖送走了季思妍，转身见傅昼景愣在客厅里，薄唇紧抿，眉头紧锁，一脸心事重重的样子，便朝他走了过去。
<br/>
<br/>        “昼景，你怎么呢？是不是有什么心事？”
<br/>
<br/>        童筱颖走到傅昼景的跟前，微微仰头看着傅昼景。
<br/>
<br/>        她的声音，声线温柔婉转，确实是个唱歌的好料子。
<br/>
<br/>        傅昼景抬起手来，捧住童筱颖的脸颊，深沉地回答道：“我是有心事，但是我不知道要怎么跟你说，比较好！”
<br/>
<br/>        “如果昼景觉得为难，可以不用告诉我！我只希望，你能开心。”童筱颖微微一笑，温柔体贴地说。
<br/>
<br/>        傅昼景看着童筱颖的眼睛，在她这双清澈的眼睛里，他看到了肮脏的自己。
<br/>
<br/>        他都觉得自己脏，那筱颖一定会更加觉得他脏吧！
<br/>
<br/>        “昼景？”童筱颖见傅昼景看着自己发呆，轻轻地唤了声。
<br/>
<br/>        傅昼景猛然回神，顿了顿，才接着说：“筱颖，不要与季思妍深交，她是我大哥的女人！”
<br/>
<br/>        “大哥的女人？！”童筱颖诧异。
<br/>
<br/>        大哥傅夜沉不是有苏漫雪了吗？怎么这个季思妍也成了大哥的女人呢？
<br/>
<br/>        傅昼景点了点头，见童筱颖一脸疑惑与吃惊，叹了口气，惆怅了起来：“筱颖，我的身世背景，以及在傅家是个什么地位，想必，你已经从旁人那儿听了不少。没错，他们说得都对！我确实是傅家的私生子，不受爷爷喜欢与重视，更不受大妈，也就是大哥生母的待见！现在，我凭借自己的本事，慢慢的有实力了。所以，大哥开始忌惮我了！把季思妍派到我身边来监视我！”
<br/>
<br/>        “你们是兄弟啊！为什么要这样？”童筱颖听得有些匪夷所思。
<br/>
<br/>        难道是她没入过豪门，不懂豪门里的勾心斗角，才会太孤陋寡闻的吗？
<br/>
<br/>        虽然以前听说过有些家庭里两兄弟为了争夺父亲的遗产而闹上法庭的新闻，但现实生活里，她还从未遇到过。
<br/>
<br/>        不过，在童筱颖的潜意识里，总觉得大哥傅夜沉不像是那种会使手段的男人。
<br/>
<br/>        可是，昼景也不会骗她啊！
<br/>
<br/>        这件事情，既然昼景开口跟她说了，她自然得把事情弄明白才行。
