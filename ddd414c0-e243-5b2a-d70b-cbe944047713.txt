        “你是说，包养雅姗的那个男人，可能就是这个连少？”季思妍一惊一乍起来。
<br/>
<br/>        同父异母的妹妹季雅姗被男人包养这事儿，是她估摸着揣测的。
<br/>
<br/>        因为，季雅姗的母亲曾经病重继续医药费，季雅姗来家跪在地上问父亲要钱，结果被她母亲毒打了一顿给轰出了家门。
<br/>
<br/>        后来，不知道季雅姗是从哪儿凑到的钱，不仅季雅姗的母亲的病痊愈了，而且，季雅姗还住起了别墅，穿着限量版的衣服，背的是时尚名牌的包包。
<br/>
<br/>        季思妍那次见季雅姗，就知道季雅姗一定是被男人给包养了。
<br/>
<br/>        在她的生活中，已经有了季雅姗和季雅姗她母亲这活生生的例子。
<br/>
<br/>        季思妍自然是不想自己将来，以及自己的将来的孩子，步入季雅姗她们母女两的后尘。
<br/>
<br/>        所以，她必须得“除掉”童筱颖。
<br/>
<br/>        季思妍觉得，自己做得最明智的一点是，不是让自己的身份被暴露，成为外人的笑柄。而是让她童筱颖“出轨”被抛弃，让傅昼景对童筱颖心灰意冷，与她童筱颖离婚。
<br/>
<br/>        “这种事情，我怎么知道？”季运恒纳闷道。
<br/>
<br/>        季思妍回过神来，一五一十地把自己的想法，全告诉了季运恒：“我就是想要童筱颖出轨，让阿景跟她主动离婚。所以，哥哥，你得帮我！”
<br/>
<br/>        “好吧！你说，我要怎么帮？”季运恒双手一摊，反问。
<br/>
<br/>        季思妍咧嘴一笑，朝季运恒勾了勾手指，示意他凑过来侧耳倾听。
<br/>
<br/>        在傅昼景出差的这段时间，童筱颖把驾驶证给考到了手，只是她已经联系不上季雅姗了。
<br/>
<br/>        她有去季雅姗住的别墅里找过她，但是别墅内已经是人去楼空。
<br/>
<br/>        童筱颖记得，季雅姗跟自己提过，她要去找十爷。
<br/>
<br/>        希望她找到十爷后，和十爷有个好结局吧！
<br/>
<br/>        最近对于她来说，真的是多事之秋……
<br/>
<br/>        暗恋她的男生因她而死，她好不容易才深交的好朋友又离她而去，自己老公又因为忙于工作经常不回家……
<br/>
<br/>        而她，又迫于大哥傅夜沉的淫威，在傅昼景未回来之前，不得不继续住在大哥傅夜沉的私宅里。
<br/>
<br/>        现在苏漫雪不住私宅了，她童筱颖每天的日常就是跟大哥傅夜沉一起坐车上下班。
<br/>
<br/>        或许是因为在公司里有大哥罩着的缘故，童筱颖上不上班现在已经可以随心所欲了。
<br/>
<br/>        这天，家里一大清早就停电了，傅夜沉和连华生坐在餐桌前吃早餐的时候，童筱颖仍旧未下来，于是傅夜沉让刘婶上去看看。
<br/>
<br/>        刘婶问候完童筱颖后，下楼来回禀道：“大小姐说她今天想睡懒觉，不想去上班。”
<br/>
<br/>        因为傅夜沉不喜欢刘婶唤童筱颖“二少奶奶”，刘婶这才不得不唤童筱颖为“大小姐”。
<br/>
<br/>        傅大少爷对待二少奶奶的态度，还真的就像对待自己的恋人一样宠溺着。
<br/>
<br/>        就连大少奶奶苏漫雪，都没有这么好的待遇。
<br/>
<br/>        “那就随她睡吧！到中午的时候，记得叫她起床吃午餐。”傅夜沉放下手里的刀叉，抽了一张纸巾擦拭着嘴唇。
<br/>
<br/>        连华生见童筱颖今日不跟他傅夜沉同车，于是想搭个顺风车出门：“今天我不想开车出去，就坐你傅大少爷的车好了！”
<br/>
<br/>        “怎么，今天家里停电，不方便你做事呢？”傅夜沉意味深长道。
<br/>
<br/>        连华生点了点头：“没办法，今天得交报告！家里既然停电了，我就只能去外面把资料打印出来，拿去给那个人看。”
<br/>
<br/>        “嗯，在外面小心点。”傅夜沉嘱咐道。
<br/>
<br/>        连华生微微颔首，像是在服从命令。
<br/>
<br/>        “临海大学马智杰的那个案子的尸检报告，你看过后，觉得有没有问题？会不会跟我们查的事情有关联？”傅夜沉忽然想起这件事情，若有所思地问。
<br/>
<br/>        连华生瘪了瘪嘴，摇头道：“我暂时没发现有什么问题，不过，我会一直盯着。”
<br/>
<br/>        “好，辛苦你了！”虽然他们两之间不言谢，但是傅夜沉还是客套地说了一句。
<br/>
<br/>        连华生看着傅夜沉，只笑不语。
<br/>
<br/>        待傅夜沉和连华生都出门后，童筱颖一觉睡到家里来电了，她才伸着懒腰从床上爬起来。
<br/>
<br/>        她洗漱完后，换好了衣服，打开房门，从卧室里走了出来。
<br/>
<br/>        当她走到楼道口，楼上“咔嗞咔嗞”的声音，吸引了她的注意力。
<br/>
<br/>        好像是打印机正在自动打印文件……
<br/>
<br/>        童筱颖循着声音，爬上了楼。
<br/>
<br/>        傅夜沉的私宅里，二楼只有一个房间是属于她童筱颖的，其余全是傅夜沉的房间，而三楼是连华生的住所和实验室。
