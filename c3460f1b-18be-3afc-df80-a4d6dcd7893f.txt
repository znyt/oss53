　　“好，现在没有那些妖兽守护，现在就开启洞府，进入其中！”
<br/>　　司空轩扫视了周围一圈，当机立断，立刻对着那颜古龙帝洞府的大门前，吟唱起来。
<br/>　　极为古老的语音，蕴含有着无尽奥秘，就连楚风眠都无法参透这语言中的奥秘。
<br/>　　“吼！”
<br/>　　一声龙吟，突然响彻起来，在这司空轩吟唱的语言之中，猛然一条真龙凝聚出现。
<br/>　　在这一声龙吟之下，那颜古龙帝洞府前的禁制，全然破碎开来，其中的大门，缓缓打开。
<br/>　　在这大门的背后，深邃到极点。
<br/>　　楚风眠感觉到，在这大门打开的一刻，他身躯中的血脉，都在沸腾起来。
<br/>　　仿佛是其中有些什么东西，吸引着楚风眠，令楚风眠的血脉都为之沸腾。
<br/>　　“这颜古龙帝的洞府中，到底有着什么东西，居然是让我的血脉如何渴望！”
<br/>　　楚风眠心神一动，也是震惊无比。
<br/>　　他身上的战龙血脉，如今一些诸天奇物，他根本看不上，只有像是那龙丹，天元神丹这种奇物，才能够让他们稍稍渴望。
<br/>　　可是现在，在这大门打开的一刻，楚风眠都能感觉到，他身躯中血脉的渴求。
<br/>　　在这颜古龙帝的洞府中，肯定是有着什么东西，对楚风眠的血脉有着大用处，才会让现在楚风眠的血脉，如此渴望。
<br/>　　“他们要进去了，动手！”
<br/>　　楚风眠身形唰的一动，直接隐藏在身形，逼近了过去。
<br/>　　那司空轩，打开了这洞府的大门，脸上也是露出了几分喜色。
<br/>　　“我们快进去！”
<br/>　　说着，司空轩便是身形一动，想要直接踏入其中。
<br/>　　但是几乎同时，司空轩刚刚脚下一动的一刻，一道巨大无比的身影，猛然现身。
<br/>　　楚风眠的身形，悄然之间已经是无声无息的接近了他们四人，在这洞府开启的一刻，楚风眠突然现身。
<br/>　　远古战龙精血，被全然凝聚。
<br/>　　“轰！”
<br/>　　巨大无比的战龙之爪，突然从天而将，足足有着一尊真龙，突然现身，一爪轰击过去。
<br/>　　这一爪的威力，没有任何的留手，简直是楚风眠的全力。
<br/>　　让一尊生死境武者前来，都要被直接轰杀。
<br/>　　“怎么回事？”
<br/>　　“哪里来的攻击！”
<br/>　　“龙族武技？”
<br/>　　司空轩，云公子二人率先反应过来，面对这战龙之爪，一身灵力也是猛然爆发了出来。
<br/>　　二人合力之下，凝聚灵力，突然在周围化为了一尊巨大无比的阵法，将他们四人，全然笼罩其中。
<br/>　　剩下的麟木，罗辰二人，一时震惊，也是极快反应过来，一股股灵力，注入到阵法之中，四人合力，催动阵法来将他们全然包裹在了其中。
<br/>　　“先抵挡住他，我们赶快进入到洞府之中！”
<br/>　　司空轩一边抵挡着，一边大声开口道。
<br/>　　可是下一刻，一声嘲笑声音，突然响起。
<br/>　　“小小阵法，也想要抵挡住我？给我碎！”
<br/>　　突然之间，楚风眠的身形恢复过来，化为了他本来的样子，身上的狼狈样子，一扫而空，脸上露出了几分阴冷笑容。
<br/>　　楚风眠凝聚力量，大手一挥，战龙之爪再次落下。
<br/>　　“轰隆！”
<br/>　　巨大的轰鸣声之下，那四人合力，凝聚灵力的阵法，在战龙之爪的轰击下，猛然开裂，化为无数裂痕。
<br/>　　下一刻，更是直接全然炸裂，化为了一团团的灵力，全然破碎开来。
<br/>　　“什么？楚风眠，是你？”
<br/>　　司空轩退后几步，嘴角都震出些鲜血，他看向楚风眠，一眼满是不可置信。
<br/>　　“你，你没死，你是怎么从妖兽手中逃出来的，不对，你根本就没有受伤！”
<br/>　　司空轩看到楚风眠的突然出现，简直如同见了鬼一般。
<br/>　　本来楚风眠去引走妖兽，气息消失，在他看来，肯定是楚风眠已经死在了妖兽的手中了。
<br/>　　可现在楚风眠的突然出现，一爪之下，居然是将他们四人联合布置而出的阵法都给打破。
<br/>　　这哪里像是有着一点伤势的样子。
<br/>　　司空轩算计了半天，将楚风眠这样一个小人物玩弄在鼓掌之中，可想不到，就是这个在他看来的小人物，居然是完全打破了他的算计。
<br/>　　“死？你真以为你们的那点小小算计，可以瞒得过我的双眼？”
<br/>　　楚风眠看向司空轩，双眼如同就是在看向一个死人。
<br/>　　“我不过是故意引走妖兽，就是为了让你开启这洞府的禁制罢了，现在洞府已经开启，你们也已经没有任何意义，已经没有在活下去的必要了。”
