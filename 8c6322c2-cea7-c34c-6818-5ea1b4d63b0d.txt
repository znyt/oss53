　　第613章
<br/>　　程大雷感觉全身骨骼都僵硬起来，本以为雨天抱进怀里一只小白兔，可没想到这小白兔是美女蛇，会突然扑过来咬断自己的脖子。
<br/>　　现在周桃就躲在自己身后，如果她真的是那位鬼脸婆婆，在这个距离杀掉自己是绝对没有问题的。
<br/>　　程大雷一点一点的挪着身体，也不想东方童看出异样。他哈哈干笑一声：“那鬼脸婆婆和长风关也该是井水不犯河水，这次为何和她过不去？”
<br/>　　“唉……”
<br/>　　东方童长叹一声，抬头和刘凯的目光撞上，刘凯道：“要与他说么？”
<br/>　　东方童鬼鬼祟祟，程大雷怎么看也不像好人，大家都是彼此怀疑。
<br/>　　“都到这时候，还有什么不好说的。”东方童道：“实不相瞒，那贼婆娘将大人的军印到了，我们这次抓他，也就是为将军印讨回来。”
<br/>　　程大雷想起刚才周桃的模样，道：“看她身上，不像带东西的样子，军印毕竟也不是什么小物件。”
<br/>　　曾经在青牛山，程大雷一人挂三家印，对于军印这种东西还是蛮熟悉的。帝国军中有见印如见人的说法，在军中它代表了权力。陆长迢丢了，往大了说是失职掉脑袋的事情，往小了说，也是一件蛮丢人的事情。
<br/>　　“谁知道那贼婆娘将军印藏在何地，我们抓到她后，也没从她身边找到，准备带回去慢慢审问。谁想那女人如此狡猾，竟让她在眼皮底下逃了。”说起这件事，东方童就有些气闷，抓到周桃可不是一件容易事，可以说费了九牛二虎之力，结果却是功败垂成。
<br/>　　程大雷也闹不清双方说得谁真谁假，不过，这件事和他好像没什么关系。心中还颇有些幸灾乐祸，他只是出于好奇问道：“想阁下的手段应该不弱，对付一个女人法子还是蛮多的，何至于问不出什么。”
<br/>　　“阁下有所不知，鬼脸婆婆的样子阁下也是看到了，她还是有几分姿色的。”东方童。
<br/>　　程大雷点点头，他倒是不否认这点。
<br/>　　“如这样的妖妇，最擅长勾引男人，她裙下之臣有不少，每个都愿意为她玩命。其中有一人，更是大大的有名，堪称京州第一高手。”
<br/>　　程大雷摸了摸鼻子：“谁？”
<br/>　　“扶风一剑柳追风。”
<br/>　　程大雷对老一辈江湖人还是蛮熟悉的，如果年轻些的，他就不认识了。那柳追风住在京州广陵城，大概四十余岁，也算一代宗师，家里豢养的徒弟门人有上千人。
<br/>　　传说他在柳树下舞剑，狂风吹落叶，每一片柳叶都被他斩成两半，如此才有扶风一剑的外号。如这种人，往往勾结城里的达官贵人，真如果柳追风出面护着她，一个边关守将怕还真不被柳追风看在眼里。
<br/>　　东方童嘴里骂骂咧咧，行伍中人大多粗俗，只说周桃是一双藕臂万人枕，两半粉唇任人尝，那柳追风怪不得姓柳呢，活该他头上绿油油……
<br/>　　程大雷摸摸鼻子，东方童可不知道周桃就在程大雷身后躲着，这些话会一字不差的传入她耳中。至于程大雷，他听得还是蛮开心的。
<br/>　　“大哥，这里有个山洞，我们先在这里躲躲，明天再赶路。”
<br/>　　“好，为今之计，只有先在此地避避雨。”
<br/>　　正这时候，外面响起说话声，两个中年汉子走进山洞。当他们走进来时，东方童一行人腾地站了起来，拔出了随身的兵器。
<br/>　　说曹操曹操便到，这两个汉子其中一人正是柳追风，另外一人是他请来的帮手，在京州绿林道也有些名气，名叫孟一刀，有刀法无双的之称。
<br/>　　这柳追风对周桃是一往情深，知道她被东方童抓走后，立刻什么事情都顾不得了，将老婆孩子抛下，立刻来追东方童。追来追去，便走到这荒山野岭，赶上这场雨，说巧不巧，正好走进这山洞避雨。
<br/>　　两拨人竟然在此地碰上，当真是仇人见仇人，必定眼发昏，若要报仇很，拔剑斩仇人。
<br/>　　程大雷和小白狼无辜的坐在石头上，心道：这下热闹了。
<br/>　　柳追风颇有种踏破铁鞋无匿处，得来全不费工夫的感觉。他咬着牙道：“呔，你们把小桃抓到哪里去了，快快将她交出来。”
<br/>　　“柳追风，你敢包庇那贼婆娘，不怕我长风关的披甲士么？”
<br/>　　“呵，你让陆长迢来，看我怕不怕他。”
<br/>　　啧……
<br/>　　程大雷长叹一口气，看得出来东方童明显有些害怕。这柳长风以剑法扬名，本事自然不弱，这逼仄的山洞又适合他发挥。如果动起手来，定然是东方童一行人吃亏。
<br/>　　可程大雷是在这里避雨，也不能让他们搞得此地乌烟瘴气，他摆摆手道：“唉，我说两位……”
<br/>　　双方的目光同时落在他身上，如刀子一般扎人。
<br/>　　程大雷恍然未觉，道：“出门在外，大家遇见了就是朋友，什么仇呀狠的，不如大家放下兵器，好好喝杯酒。”
<br/>　　“住口。”
<br/>　　双方几乎同时向他吼道。
<br/>　　柳追风盯着程大雷冷哼道：“你算个什么东西，这里也有你插口的份儿，莫非，他是你们请来的帮手。”
<br/>　　“我……”
<br/>　　程大雷正准备说什么，忽感觉背后一道杀意袭来。周桃用束发的木钗抵住他的脖子，贴在他耳边冷冷道：“动一动，杀了你。”
<br/>　　柳追风一见她，立刻惊喜道：“小桃，你没事吧。”
<br/>　　周桃扬起头，用另外一只手将湿漉漉的头发撩在脑后，如此风情，当真惊心动魄。只不过程大雷现在一动不敢动，算是无福欣赏。
<br/>　　小白狼现在也是投鼠忌器，不敢出手。
<br/>　　“你可是小看了这位，无论在什么地方，他都有开口说话的资格，他说话的时候，你们也最好认真听着。”周桃吐出猩红的嘴唇舔了舔舌头：“这位便是琴川之主，大名鼎鼎的程大雷程当家。”
<br/>　　声音落地，如闷雷炸响，山洞里除了火堆的噼啪声，再无别的声音。东方童一开始就猜到程大雷有些来历，但没想到程大雷的来历这么不简单。
<br/>　　“你瞧瞧，误会了不是。”程大雷笑着道：“姑娘，咱们是自己人呐。”
<br/>(本章完)
