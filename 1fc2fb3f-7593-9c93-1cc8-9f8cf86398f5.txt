　　这一切在圣龙秘境之中，已经是引起了轩然大波。
<br/>　　这四宗之一的六绝宗，就这样毁了？
<br/>　　哪怕是当年楚风眠毁灭执法堂的时候，都没有如此令人震惊过。
<br/>　　四宗的实力，可是远超八堂。
<br/>　　六绝宗虽然身为四宗的末尾，可整体实力，比起八堂加起来还要强横，现在无声无息之间，就连六绝峰都被人毁了。
<br/>　　“圣龙秘境，要变天了。”
<br/>　　许多圣龙秘境弟子，都是喃喃自语道。
<br/>　　另一边，楚风眠的身形，在离开了六绝宗之后，便是直指圣院宗而来。
<br/>　　六绝宗，圣院宗这两大宗门，是曾经圣龙宗的两大仇敌。
<br/>　　灭了六绝宗之后，下一个轮到的，便是圣院宗了。
<br/>　　楚风眠身形闪烁，刹那间来到了圣院峰前。
<br/>　　如今圣院峰之上的护山大阵，早已经是全部开启，化为了一道道的屏障，笼罩在整个圣院峰。
<br/>　　楚风眠在灭了六绝宗之后，无数人都明白了，这一次楚风眠出手，闹出这么大的动静，绝不会仅仅只是为了一个六绝宗。
<br/>　　楚风眠站在圣院峰的天空之上，看着下方那些严阵以待的圣院宗弟子，嘴角露出了冰冷笑容开口道。
<br/>　　“跟六绝宗一样，圣院宗今日也没有在继续存在下去的必要了，所有圣院宗弟子，不想死的，出去跪下，如果不臣服，便是死。”
<br/>　　楚风眠冰冷的声音，传递到了下方圣院宗所有弟子的耳中。
<br/>　　这句话，已经不是楚风眠第一次说了。
<br/>　　之前楚风眠开口的时候，还有人怀疑过楚风眠所说的话。
<br/>　　可现在眼看着楚风眠已经是将六绝宗给彻底毁灭，就连六绝峰之中的灵泉都被楚风眠硬生生抽取出来了之后，再也没有任何人，怀疑楚风眠的话。
<br/>　　虽然圣院宗的底蕴，要远超六绝峰，隐隐约约之间圣院宗已经是圣龙秘境的第一大宗了，不然的话，圣院宗也没有敢跟圣龙宗去争夺的资本。
<br/>　　可楚风眠已经是灭了六绝宗，那也未必不能灭了圣院宗。
<br/>　　听到楚风眠的话，无数圣院宗弟子的心中都已经没有了多少的底气了，有些人都在打这退堂鼓。
<br/>　　“楚风眠，你不用在哪里嚣张，你胆敢攻打六绝宗的事，我已经传递到了圣地，等到圣地之中的强者到来，就是你的灭亡之日！你已经没有几个时辰可活了，还敢来嚣张？”
<br/>　　就在这众人犹豫之际，一道声音突然响起。
<br/>　　从圣院峰的深处，一尊身影，缓缓的走了出来，这是一尊老者。
<br/>　　这一尊老者，身穿龙袍，头戴宝冠，一步踏出的一刻，雄厚的天地之力，都降临了。
<br/>　　无数圣院宗弟子，听到这老者开口的一刻，仿佛心中的不安都化解了。
<br/>　　“无不源？无家的老祖宗，圣院宗的太上长老。”
<br/>　　有些围观的圣龙秘境弟子，已经是认出了这老者的身份，圣院宗的太上长老，无家的老祖宗，无不源。
<br/>　　那无道，便是无家的弟子，无家在整个圣龙秘境之中，都算的上是排的上号的大家族了。
<br/>　　随着无不源的现身，从他的背后，又是几道身影，跟随现身，都是一些中年人
<br/>　　这几位中年人身上的气息，也无比强横，无一例外，都是九劫古帝。
<br/>　　而那无不源的实力，更是达到了一种无比接近半步天人境的境界，跟曾经的无道相比，都毫不逊色。
<br/>　　几位九劫古帝的到来，让圣院宗弟子的心也是安定了不少。
<br/>　　“没想到还有着几个漏网之鱼。”
<br/>　　楚风眠目光扫过这无家的几人，眼神之中露出几分惊讶，本来楚风眠以为圣龙秘境中的九劫古帝，都已经被楚风眠杀完了。
<br/>　　现在看来，倒是有着几条漏网之鱼。
<br/>　　不过几个九劫古帝，在楚风眠的面前，也跟没有一样，楚风眠要灭了圣院宗，仅仅凭借这几个九劫古帝，也想要拦住楚风眠，简直是在做梦。
<br/>　　“小子，我劝你现在快点想想该如何应对圣地到来的强者吧，以你的罪孽，就算是死都死便宜你的了。”
<br/>　　那无不源，看向楚风眠，露出几分阴狠笑容道。
<br/>　　“圣地到来的强者？”
<br/>　　楚风眠听到无不源的话，大笑起来道。
<br/>　　“怎么，你以为圣地的强者到来，就能够救下圣院宗了？今日谁来，圣院宗都得灭亡。”
<br/>　　“你们无家的人，现在跪在地上自尽，我可以给你们留一个全尸，不然等到圣院宗被破之日，你们就连全尸都留不到。”
<br/>　　楚风眠的话一出，其他围观的无数圣龙秘境弟子，都是震惊无比。
<br/>　　圣龙秘境的真正强者，多数都是在圣地之中，驻守到圣龙秘境之中的，只有小猫两三只。
<br/>　　他们已经是见识了楚风眠的嚣张了，可听到这句话，还是吓到了不少人，圣地中的强者，楚风眠都不在乎？
<br/>　　在圣地中，可是就连半步天人境界的强者，都有存在，现在楚风眠都不在乎？
<br/>　　这种嚣张，从楚风眠的口中说出来，却是没有人觉得楚风眠是在虚张声势。
<br/>　　楚风眠已经是将他说过的话，一一兑现了。
<br/>　　“竖子狂妄！”
<br/>　　无不源脸上愤怒无比，随即大手一挥道。
<br/>　　“将护山大阵的力量全部催动！”
<br/>　　轰隆隆！
<br/>　　圣院峰之上护山大阵的力量，已经是背催动到了极限，一道道古老的纹路在圣院峰之上流淌。
<br/>　　许许多多的纹路，组成了一道道的禁制阵法。
<br/>　　一个宗门的护山大阵，是最容易看出其中底蕴的地方了。
<br/>　　单单是看起来，这圣院宗的护山大阵，就要超越六绝宗的护山大阵无数倍。
<br/>　　两者虽然同为四宗之一，但是也有着极大的差距。
<br/>　　这护山大阵全然催动起来，其中蕴含的种种力量，让九劫古帝都不敢去攻打，反噬的力量，哪怕是九劫古帝都承受不住。
<br/>　　护山大阵的力量被全然催动，那无不源的嘴角露出几分冷笑看向楚风眠。
<br/>　　他也知道楚风眠的实力，绝不是寻常的九劫古帝，所以他也没有打算来跟楚风眠硬拼。
<br/>　　而是在选择拖延。
<br/>　　只要是拖延到圣地之中强者到来，便是楚风眠的末日。
<br/>　　“小子，我看你如何可以毁了圣院宗！”
<br/>　　一名无家的九劫古帝，不屑的看向了楚风眠一眼，挑衅的说道。
