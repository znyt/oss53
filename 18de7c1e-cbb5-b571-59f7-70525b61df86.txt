　　“480万两，顾家主出价480万两，还有人出更高价吗？”
<br/>　　拍卖师欣喜地问道。
<br/>　　一层宾客，纷纷翻了个白眼。
<br/>　　二层，王家、赵家、白家，一片灰丧；顾家、钟家，则一副小人得志的模样。
<br/>　　“唉，可惜了。”白家主叹了口气，“如此重宝，看来是和我白家无缘了。”
<br/>　　下一秒，他脸上的难看之意，完全褪去，恢复了正常。
<br/>　　看得出来，白家主是个有雄心壮志，但又很乐观的人。
<br/>　　萧逸见状，笑着问道，“怎么，白家主很想要这星芒石？”
<br/>　　“这是当然的。”白家主笑道，“你可知星芒石代表着什么，上品灵器，又有和意义？”
<br/>　　“整个北山郡，只有两把极品灵器。一把在裂天剑派，一把在郡都猎妖殿。”
<br/>　　“破玄九重武者使之，足可斩杀地元境武者。”
<br/>　　“除此之外，拥有上品灵器的，也只有寥寥几个大势力；全是北山郡霸主级别的实力。”
<br/>　　“如北山慕容家，暗影楼。”
<br/>　　“像我等普通的大势力，只有中品灵器罢了。”
<br/>　　“你可想而知，若我白家拥有上品灵器，家族势力，必将大增。”
<br/>　　说着，白家主脸上满是希冀之色。
<br/>　　“我只是破玄四重；若手中有上品灵器，实力足可达到媲美破玄七重的地步。”
<br/>　　灵器作为整个世界最珍贵的东西，就是因为它能大大增幅武者的实力。
<br/>　　当然，要想完全发挥上品灵器的威力，起码要达到破玄境。
<br/>　　而一个破玄七重和一个破玄四重，差别还是非常大的。
<br/>　　白家若有破玄七重武者坐镇，其家族势力，必然会上升到另一个高度。
<br/>　　“白家主，你猜顾家和钟家联手，能拿得出多少钱？”萧逸问道。
<br/>　　白家主一愣，虽然不知道萧逸为何这样问，但还是说道，“据我估计，最多不过500万两。”
<br/>　　萧逸点点头，道，“那便好。”
<br/>　　“易霄小兄弟的意思是？”白家主面露疑惑之色。
<br/>　　下一秒，萧逸从乾坤袋中拿出一物，挥手一弹，将其弹到下方展览台上。
<br/>　　拍卖师先是一惊，以为有人偷袭，星芒石，可万万不能出意外。
<br/>　　当看到不是偷袭时，才松了口气。
<br/>　　拍卖师将此物拿过，看了一眼，而后双眼一亮，道，“哦，三品巅峰丹药之一，生灵丹，还是完美级别的。”
<br/>　　作为拍卖师，自然本身也有一定的鉴定能力。
<br/>　　“紫炎，不知你将这丹药弹来是何意？”拍卖师问道。
<br/>　　二层，萧逸说道，“不知此物你们可收？可否代替银两。”
<br/>　　“当然。”拍卖师连忙说道，“你有多少我们要多少，不过，必须是完美级别的，普通级别不要。一粒，可抵三千两。”
<br/>　　一层宾客中，听到拍卖师的话，也是微微一惊，“生灵丹，还是完美级别的？这可不只三千两阿，不过若是用来代替银两，倒是差不多。”
<br/>　　在普通人的生活中，银两是交易资本。
<br/>　　可在武者的生活中，丹药，却可以充当硬通货。
<br/>　　炎龙大陆，炼药师乃是最尊贵，也是最稀有的职业。
<br/>　　对比亿万武者，炼药师的数量，只不过九牛一毛。
<br/>　　庞大的需求量下，造成丹药经常供不应求。
<br/>　　也就是说，不会有卖不出的丹药；只有卖不完的丹药。
<br/>　　这，便让丹药成为了硬通货之一。
<br/>　　不过，这也是有区别的。
<br/>　　像雪魄丹，这种只有冰属性武魂武者才会服用的特殊丹药，则不能成为硬通货。
<br/>　　而像先天丹、洞玄丹、生灵丹、巨力丹、隐气丹等等，任何人都有用的丹药，则能成为硬通货。
<br/>　　当然，丹药的品阶必须在三品之上。
<br/>　　同时，丹药的质量不能差；不能粗制滥造，起码要在良好以上。
<br/>　　丹药的质量，分劣质、合格、良好和完美。
<br/>　　其效果没有区别；区别只在起效时间。
<br/>　　像生灵丹，这是疗伤圣药。
<br/>　　一个先天武者，若受重伤；良好的生灵丹，能在一个时辰内恢复伤势；完美级别的，则能在半个时辰内就恢复伤势。
<br/>　　对于在外历练的武者来说，一旦受伤，自然是越快恢复伤势越好的。
<br/>　　故完美级别的生灵丹，完全可以作为硬通货。
<br/>　　这时，萧逸大手一挥，弹出5个玉瓶，道，“每瓶有20粒生灵丹，总共100粒，皆是完美级别，你可以检查一下。”
<br/>　　“哦？”拍卖师连忙检查起来。
<br/>　　一层的宾客中，则已经满座哗然，“100粒完美级别的丹药？紫炎易霄，不愧是能将四品巅峰丹药轻松炼制成功的炼药师。”
<br/>　　“在炼药一道，整个北山郡，怕是只有那寥寥几位五品炼药师能胜过他了吧。”
<br/>　　炼制完美丹药，一是靠本事，二是靠运气。
<br/>　　完美丹药，对炼药师的能力要求非常高；一百个同品级的炼药师中，难出一个有这样的本事。
<br/>　　一般，炼药师的品级，高出丹药品级两品以上，就有很大的几率经常炼制出完美丹药了。
<br/>　　也就是说，五品炼药师，才能较为容易地炼制出完美的三品丹药。
<br/>　　除此之外，就只能靠运气了。
<br/>　　就算水准再差的炼药师，炼制几百颗丹药，终归会有那么几颗是完美的。
<br/>　　而萧逸一下子就拿出100粒，证明他靠的是本事，而非运气。
<br/>　　仅仅十数秒时间，拍卖师便鉴定完毕了。
<br/>　　“100粒，皆是完美级别，我们商行以30万两向你收购，可行？”拍卖师问道。
<br/>　　“好。”萧逸点点头，而后再次弹出五个玉瓶，道，“这是三品巨力丹，同样是完美级别，100粒，出个价吧。”
<br/>　　“完美级别的巨力丹？”拍卖师一惊。
<br/>　　萧逸道，“你可以检查一下效果；先天九重武者服之，可在瞬间发挥出半步洞玄实力，维持半个时辰。”
<br/>　　“什么？”拍卖师立刻检查起来。
<br/>　　若真如易霄所说，那么这巨力丹，其价格可是比之前的生灵丹更高。
<br/>　　半晌，拍卖师脸上尽是惊骇之色，道，“果然都是完美级别的。”
<br/>　　“紫炎易霄，名不虚传。这些巨力丹，我们商行以每粒5000两收购；100粒，便是50万两。”
<br/>　　“可以。”萧逸点点头，而后看向白家主，道，“不知白家主这时可有信心得到星芒石了？”
<br/>　　萧逸没有掩饰自己的话语。
<br/>　　其实，在场所有人，也都知道萧逸此刻拿出丹药的用意，肯定是想帮白家主。
<br/>　　“易霄小兄弟，你这....”白家主一时犹豫起来。
<br/>　　萧逸摆摆手，道，“白家主不必客气，有什么话，等拍卖会结束再说吧。”
<br/>　　“谢了。”白家主拱了拱手，而后高声道，“星芒石，我出价530万两。”
<br/>　　“你...”顾家家主还有钟家家主，其实从萧逸拿出丹药那一刻，脸色就已经变得无比难看。
<br/>　　此刻，更是变得冰冷至极。
<br/>　　“紫炎，别以为只有你有丹药。”顾家家主冷声道，“星芒石，我顾家势在必得。”
<br/>　　“丹药、功法、武技、哪怕是灵器，我大不了都卖给拍卖会。”
<br/>　　“我看你小小一个猎妖师，如何比得上我整个顾家的财富。”
<br/>　　他们能拿得出的钱，确实只到500万两。
<br/>　　但，那只是真金白银。
<br/>　　若是加上本身的底蕴，自然是远远不止的。
<br/>　　他们要继续竞价，只能变卖东西。
<br/>　　“卖东西？”萧逸淡淡一笑，道，“顾家主，易某提醒你一句。”
<br/>　　“功法、武技，或者下品灵器，现在直接变卖，价值可是跌了一半都不止阿。”
<br/>　　完美丹药，能直接作为硬通货，代替银两；该是多少价值就多少价值。
<br/>　　因为那些丹药谁都会买，甚至一层的宾客中，已经有不少人打算拍卖会结束后，就直接在商行买一些回去。
<br/>　　可功法武技灵器，可不是谁都会买，一来本身单价就贵；二来，如果属性不相符合，也是无用。
<br/>　　必须找到适合的人。
<br/>　　所以，现在直接变卖，价格绝对会大跌。
<br/>　　萧逸继续道，“顾家主能得到星芒石，就算将顾家所有财产变卖，那也是值得的。”
<br/>　　“可是，若无法得到。现在变卖给商行，日后难道还想用同样的价格买回来？到时候，顾家亏个倾家荡产，可别怪易某没提醒过你们。”
<br/>　　“毕竟....”
<br/>　　这时，萧逸顿了顿，狂傲道，“说到完美丹药，易某多的是。随便几千粒，还是能拿出来的。”
<br/>　　“就是不知道顾家主有多少家财能与我耗下去。”
<br/>　　萧逸说罢，淡淡地坐着，再也不发一言。
<br/>　　下方，一众宾客不自觉地咽了咽口水。
<br/>　　“随便几千粒完美丹药？”
<br/>　　“都说炼药师出了名的有钱，今日一看，还真是闻名不如见面。”
<br/>　　上方，白家主也同时说道，“顾家主，你要变卖宝贝，我白某人也会卖。”
<br/>　　“这竞拍，便继续下去吧。我与易霄小兄弟，等着你出价。”
<br/>(本章完)
