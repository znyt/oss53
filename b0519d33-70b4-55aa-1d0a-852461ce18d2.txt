　　黑水靺鞨，始于商周，古称肃慎，战国称之为挹娄，直到隋时那改称其为靺鞨。
<br/>　　如果这样还不算清楚的话，靺鞨部还有另外一个人们比较熟悉的名字，那就是女真，女真不满万，满万不可敌的女真，也是后世夺了大明朝江山的那个女真。
<br/>　　不过在大唐时其，黑水靺鞨还没有后世的那个实力，此时的他们生活还比较原始，过着游牧与游猎的生活。一部分人生活在东北的深山老林里面，靠着打猎，捕鱼为生，有另一部分人才生活在草原的附近，靠着放牧为生。
<br/>　　整个族群主要分为七大部族，而这七个部族又以粟末部马首是瞻。
<br/>　　曾经高句丽还在的时候，靺鞨人与其打了数百年的战争，后来时间长了便互相妥协慢慢的变成了和平共处。
<br/>　　但现在，高句丽被干掉了，这片黑土地有了新的主人——大唐。
<br/>　　大唐与高句丽不同，她不会因为敌人的强大便妥协，尤其是李二在位的时候，更是如此。
<br/>　　所以和平再一次被打破，黑水靺鞨不甘心自己的领地被大唐无限制的侵占，于是有粟末部头领大祚荣便生出了攻唐的念头。
<br/>　　贞观十一年的时候，他联合了薛延陀和西突厥，打算三方联合，给大唐一个教训。
<br/>　　可约定的时间到来之后，靺鞨七部之间又产生了一些分歧，导致他们并没有按照约定与薛延陀同时发起攻击，让李绩在边境那里空等了好几个月。
<br/>　　……
<br/>　　如今已经是贞观十五年，大祚荣已经完成了七大部族的统一，将族中所有反对的声音全部统一起来，变成一个共同的声音——夺回原本应该属于他们自己的东西，辽东！
<br/>　　所以在一个十分祥和的早晨，边境线上响起了低沉的号角声，一队队黑水部的骑兵从密林中走了出来。
<br/>　　十万靺鞨大军陈兵边境，要与大唐决一死战，夺取辽东。
<br/>　　……
<br/>　　辽东大都督府，大都督李绩正在与一个年轻的将军对弈，冷不防有亲卫推门走了进来，沉声说道：“报，边境急报，黑水靺鞨大军压境！”
<br/>　　“大军压境……”李绩放下手中棋子，眼中闪过一道冷芒：“他们终于还是忍不住了么？”
<br/>　　“大都督，末将愿带兵迎敌。”坐在李绩对面的年轻人此时也停下了手里的动作，起身正色请命。
<br/>　　“常之稍安勿躁，且先看看靺鞨部有什么动作再说。”李绩对那青年摆了摆手，而后转头对进来的亲卫问道：“仔细说说，靺鞨部军力如何？何人领兵？”
<br/>　　“回大都督，靺鞨部出兵十万，尽数为骑兵，由粟末部族长大祚荣统领。”
<br/>　　“十万，呵呵……，好大的手笔！常之，看来这次靺鞨人可是真的急了。”听了亲卫的回报之后，李绩露出一个期待已久的表情，似乎他一直就在等着这一天一样。
<br/>　　“大都督，末将以为来者不善，善者不来，那黑水靺鞨与高句丽争战数百年未曾被灭必有其过人之处，所以末将以为还是不要轻敌才好。”那被李绩称为常之的年轻人说话的声音有些怪，听上去不似大唐口音。
<br/>　　“靺鞨骑兵的确在战场上有过人之处，但那是以前而不是现在，常之考虑的过多了。”李绩再次摆了摆手，想了想又说道：“不过你这样也好，否则本都督还真不放心把兵交给你。”
<br/>　　“大都督……”那青年似乎没想到李绩真的会让他领军出战，闻言立时有些激动的不知说什么才好，双眼紧紧盯着李绩，生怕他会反悔一般。
<br/>　　“出战可以，给你一万骑军，不过，此战只可智取，不可力敌，你可有把握？”李绩好像根本没有把靺鞨的十万骑兵放在眼中，之所以让那青年出战，似乎就是想考校一下他的能力。
<br/>　　而那青年不也含糊，站直身体敬了一个大唐军礼：“末将可保证！”
<br/>　　“嗯，那你便去吧，人你可以自己挑，需要什么装备也都带上，明白一早出发，记住，不可力敌！”
<br/>　　“是！”那青年朗声应道。
<br/>　　……
<br/>　　黑齿常之，百济人，初时声名不显，百济被灭之后，李承乾曾命人将其找到，然后再入军中，后来李绩到了辽东，接手了辽东的一切军务，黑齿常之也就自然而然的成了李绩的部下。
<br/>　　而正所谓是金子在哪里都会发光，李绩在接手辽东防务不久后，很快就发现了异于常人的黑齿常之。
<br/>　　首先是亲和力，这黑齿常之非常善于安抚士卒，稳定军心，那此投降过来的百济或者新罗俘虏只要是投入到他的帐下，几乎很快就会稳定下来。而且他人处，那些俘虏往往总会惶惶不安一段时间，有些时候甚至还会出现械斗之类的事情。
<br/>　　其次就是黑齿常之的脑子非常好使，数次围剿叛军的过程中，都会主动出击，然后给敌人留下明显的漏洞，等敌人反击的时候他在迅速将漏洞补上，而直到这时敌人才会发现那其实只是一个圈套。
<br/>　　基于这两点，黑龄常之很快得到了李绩的赏识，时常将其招到自己的都督府中与其聊一些治军方面的事情，看上去颇有些收徒的意思。
<br/>　　……
<br/>　　书山府，其地理位置也就是现代的长春。
<br/>　　为了在辽东地区进行统一教育，经李二同意，孔颖达等一些老夫子在这里设立了所谓的书山府，一些来自于大唐的文人、士子大多居住于此。
<br/>　　从某些方面来说这书山府便是辽东一代的文化圣地，很多原来高句丽和新罗、百济的学生都聚集在这里，学习来自大唐的文化知识。
<br/>　　可是，就在某一个清晨，书山府的平静被打破了，十万铁骑将书山府团团围住，大有屠城之势。
<br/>　　大祚荣穿着一身裘皮制成的长袍坐在马上，右肩上停着一只硕大的鹞鹰，左手甩着马鞭，目光紧盯着不远处的那座城池。
<br/>　　在数百年前，那座城池还不是这个样子，它的名字当时叫‘茶啊冲’，也是靺鞨部落的第二王都。
<br/>　　可是现在，它已经成了大唐的战利品，虽然这城池并不是大唐在他们手中夺走的，但大祚荣还是想要把它夺回来。
