　　第148章
<br/>　　李如趴在地上，保持标准五体投地的姿势。如果讲不通道理，聪明无用武之地，她还有最后一道杀着，那就是她的身体本身。她明白自己的身体对男人有怎样对吸引力，再加上自己是苏樱继母的身份，对男人更有另样的诱惑。
<br/>　　在需要的时候，她毫不介意使出这道杀着。但面对程大雷与面对公羊哀不同，面对公羊哀，自己是赏景者；面对程大雷，是让自己成为美景本身。
<br/>　　她不相信，程大雷有抵挡这种诱惑的能力。
<br/>　　“你能否起来说话？”程大雷的声音明显有些不自然。
<br/>　　李如站起身，以仰视的姿态看着程大雷，借此烘托程大雷的高大。男人都喜欢高大，难道不是这样么。
<br/>　　“机关算尽，最后不过落得竹篮打水一场空而已。若程将能施舍一二分怜悯，赏奴家一条活路。奴家所求，无非是安稳二字而已，日后为奴为脾，结草衔环，都要念程将恩情。”
<br/>　　“你……”程大雷语气越发不自然，呼吸急促了几分。
<br/>　　没有人能抵挡这道杀着，程大雷更不能。她贴近程大雷，呼吸想要呵在程大雷脸上。
<br/>　　“程将，若想奴家做什么，在此地便可以。奴本就是伺候人的命，奴……很会伺候人的。”
<br/>　　“你能不能……”
<br/>　　“你想我做什么？”李如迫得更紧，心底却有些轻蔑，终究只是个山贼而已。
<br/>　　“你踩着我袍子了。”程大雷一脸肉疼道：“我今天刚发下来的官服，还没过水呢。”
<br/>　　“……”李如。
<br/>　　难道他一脸悲催，就是心疼自己的袍子，却看不见眼前站着的人要比袍子珍贵千百倍。是该说他是快木头，还是说自己高估了自己的魅力。
<br/>　　咚！
<br/>　　此刻变故突生。
<br/>　　李如看到，程大雷身后的柜子里窜出个人，手持一把腰刀，照着程大雷背心扎来。
<br/>　　“狗贼，去死吧！”
<br/>　　是公羊哀。
<br/>　　不知他何时躲在柜子内，该是看到自己引着程大雷过来时，提前潜入屋子内，酝酿最后的杀着。
<br/>　　可即便杀了程大雷也扭转不了大势，现在李如真心期盼，这一刀伤不着程大雷。
<br/>　　公羊哀不该喊出声来，因他的声音，程大雷提前已有准备，他转身来不及拔刀，只好连刀带鞘硬接了公羊哀一刀。
<br/>　　来势好猛的刀，公羊哀的刀法不成套路，却是简单直接，只从要害入手。这是与野兽在旷野中厮杀锤炼出来的刀法，如此更见凶狠。
<br/>　　程大雷没有碰到过这样的敌手，对方狠得像条狗，杀起来完全是你砍我一刀，我也要啃你一块肉的厮杀方式。
<br/>　　砰！
<br/>　　程大雷一脚踹开屋门，扩大战斗空间，同时也是为引外面人进来。自己是领导者，逞什么匹夫之勇。
<br/>　　而公羊哀此刻已抓住李如的手腕，将其护在身后。
<br/>　　“如姐姐，莫怕，我带你杀出去！”
<br/>　　“好汉子！”程大雷赞了一声：“天大地大，你又能逃得了那里去，来，与我单打独斗，胜过我手中刀，我放你们离去。”
<br/>　　公羊哀目光闪烁，已经有几分动心。
<br/>　　“莫信他，他在拖延时间！”李如道。
<br/>　　“狗贼！”公羊哀恼怒得大吼一声，挥刀向程大雷砍来。
<br/>　　诓得了公羊哀却是诓不过李如。
<br/>　　但即便被识破也无关系，因为程大雷的人已经到了。斜刺里一柄铁枪接住公羊哀的刀，两人战成一团。
<br/>　　这才是棋逢对手，将遇良才，十合之内竟不分胜败。围观的蛤蟆寨众人都有些佩服公羊哀的武艺，李如手下竟有这样的高手。
<br/>　　因此的缘故，大家没有选择一拥而上。
<br/>　　“秦大哥，你且退下，让我会会他。”是赵子龙的声音，他没有用枪，而是同样选择用一把腰刀。
<br/>　　公羊哀借换阵的光景呼哧呼哧喘气，如同被困的兽。他回头紧紧将李如护在身后，道：“如姐姐，莫怕，我护你出去！”
<br/>　　“嗯。”
<br/>　　“来呀，杀呀！”公羊哀面对众人狂吼。
<br/>　　突然一块石头从背后狠狠砸在公羊哀脑袋上，顿时血流如注，公羊哀转过头来，见李如手忙脚乱的把石头抛掉。
<br/>　　“如姐……你……”公羊哀一句话没有说完整，噗通跌倒在地上。
<br/>　　李如吓得花容失色，可她又在极短的时间镇定下来，抬起头道：“此人蓄意行凶，家有恶奴是我平日管教不严，一切听程将军处置。”
<br/>　　低头扫过昏死的公羊哀时，李如眼底闪过一丝黯然：傻弟弟啊，逃又能逃到那里去呢，至于我，并不用逃的。
<br/>　　所有人都被这一幕震慑住，脑海里闪现过一个词：最毒妇人心呐。
<br/>　　程大雷轻轻拍掌：“好，好！来人，将这人送进大牢，专业的事由专业的人处理，至于苏夫人么……这是你们的家务事，听苏小姐安排。”
<br/>　　苏家是有很多家产的，如城外的田地，城内的屋舍产业，码头上来往的货船……有些生意除了苏四海本身之外，他的叔伯兄弟也有份参与。而原先属于苏四海那份，自然完全有苏樱继承，其他人不会也不敢有意见。
<br/>　　苏樱还按照程大雷的话，说些：日后大家一起努力，让苏家越来越兴旺之类的。
<br/>　　以暴力威慑，以利益连接，所谓统治术，无非就是如此，无聊又实用。
<br/>　　至于如何处置李如，程大雷没有给任何意见。苏樱发挥了她从未有过的冷漠，将李如及其女儿赶出苏府，给了几亩薄田度日。习惯掌握的李如，日后大概只能掌握锄头了。
<br/>　　“大当家，斩草不除根，小心养虎为患呐，这样处置她，是不是太仁慈了。”徐神机过来道。
<br/>　　“一介女流，算什么老虎，顶多一个母老虎。”程大雷撇撇嘴，随之缓缓道：“我掌中的剑是要一会天下英雄的，这些乱七八糟的家务事，没多大意思。”
<br/>　　徐神机愣了愣：“大当家，你手里是刀？”
<br/>　　“……”程大雷忽然大吼：“我说的是象征，什么是象征你懂么！”
<br/>　　嘟，你已完成任务『帮苏樱争夺家产』，奖励抽奖次数1，是否现在抽取？
