　　第522章
<br/>　　关鱼的情况十分糟糕，手下人甚至来不及将他抬走。如今人里三层外三层围着，刘悲蹲在他面前。
<br/>　　“好啦，散一散，散一散。”程大雷摆摆手，挤过人群钻了进来：“怎样，严不严重？”
<br/>　　刘悲一颗心牵在关鱼身上，心情紧张，倒是没有听见程大雷的话。
<br/>　　关鱼倒在地上，不仅胳膊肿了，脸也肿了半边。此刻面前若有个地方，他恨不得立刻扎进去，当然，假若他有这个力气的话。
<br/>　　他抬起左手，无力的挥了挥：“没事，我……没事。”
<br/>　　仍然强撑着嘴硬，但却是有气无力。程大雷俯下身，紧盯着关鱼此刻模样。情况不是很好，他一张脸由红转黑，看上去有些诡异。
<br/>　　程大雷命人将关鱼抬到医馆，蛤蟆城的大夫但凡能动弹的，都叫到一起，看谁能解决这件事。
<br/>　　屋子里挤满了人，程大雷甚至被挤出了门外。他在院子里来回踱步，心里牵着关鱼的状况。
<br/>　　来回不少大夫进出，手中托着各种各样的草药。为了治好关鱼，蛤蟆城不惜血本，但候在门口的他，一直没等到好消息。
<br/>　　蛤蟆城现在不缺武将，秦关张赵都是万人敌的存在，也不缺少理政的高手，一座小小的蛤蟆城，有一个和珅就玩转了。可治病救人，妙手回春的大夫，程大雷却一直没遇到什么高手。
<br/>　　如今，也就是刘悲在前面顶着，他手中一帮大夫，治个小病小伤还是没问题的。可论医术，确实连刘悲也比不上。
<br/>　　这次关鱼的事，也是刘悲料应全局，程大雷等了好久，刘悲才从屋里出来。
<br/>　　“怎样了，有没有好些？”程大雷迫不及待问。
<br/>　　刘悲叹了口气，他虽然经常叹气，但今天这口气，叹的特别长。
<br/>　　“不是很好。”刘悲摇摇头：“二弟是中毒了。”
<br/>　　“中毒？”程大雷皱紧眉头。
<br/>　　“应该就是二弟碰到的那只蛤蟆。”刘悲道：“他怎地如此不小心。”
<br/>　　程大雷挥挥手，手下捧着一个盘子来，上面摆着半只蛤蟆。如今已经焦糊了，烤得外焦里嫩。
<br/>　　刘悲和程大雷凑过去脑袋，几乎同时抽了抽鼻子。
<br/>　　“哎呀，好香！”
<br/>　　程大雷将刘悲脑袋推开，道：“好啦，看看得了，关鱼只是碰了碰，就废了半条命，你要真有胆量把它吃下去，估计也就看不见明天太阳了。”
<br/>　　“二弟现在就是被这只蛤蟆害的。”他咬牙切齿，拍着手掌：“只是不知道这只蛤蟆是从哪里来的。”
<br/>　　程大雷也同样摇摇头：“大概是从哪里不小心跑过来的吧。”
<br/>　　任程大雷怎么想，却也想不到这蛤蟆是白衣老尼带进来的。也是程大雷造化不小，这蛤蟆先在陶罐里闷了一路，又在火海里烧烤，烤得外焦里嫩，香气诱人。
<br/>　　如此，毒性去了大半，可就算这样，也害得关鱼半死不活。如果它没经历这些事，以这蛤蟆的毒性，不说祸害整座城，祸害大半个城主府是没问题的。
<br/>　　当然，这些程大雷一概不知，现在他只知道，关鱼中毒了。
<br/>　　“你刚才看过，能不能帮他解毒？”
<br/>　　“刚才，我已给他放血逼毒，熬了草药，看情况，暂时没有性命之忧。但现在还不能下地，必须尽快想办法给他解毒，不然的话，二弟这条命就保不住了。”
<br/>　　刘悲的脸色不是很好看，当然，他的脸色一直不好看。可今天显得格外苍白，坐在那里一动不动，还不停往外渗冷汗。
<br/>　　程大雷也很心忧，不说关鱼对山寨的作用，毕竟这么多年过来，程大雷也不能眼睁睁看着关鱼去死。
<br/>　　“难道真没有法子了么？”程大雷认真问。
<br/>　　“唉，我才疏学浅，当年随师父学艺时也不如何勤奋，总觉得人生一世，草木一秋，最后都不过是个死，有些事也不用太在意。到今日方才悔不当初……可如今说什么都晚了。罢罢罢，瓦罐难免井口破，将军总是阵上亡，谁都会死，为什么不能是我二弟。我现在就给二弟准备后事，让他好好走。”
<br/>　　“你现在别扯这些没用的。”程大雷撇撇嘴，道：“我就问你一句话，究竟有没有法子。天上地下，只要能救了关二爷，其他你不用管，事情由我去做。”
<br/>　　“这个……也不能说没有法子。”刘悲想了想道：“我学艺不精，也不好给我师父扬名。若我师父在此地，当能救回二弟姓名。”
<br/>　　“你师父……他心中何处？”
<br/>　　“师父他一大半年纪，也不知还在不在人世，我只知道他家在徐州，徐州有个华家村，师父就住在那里。”
<br/>　　“华家村……”
<br/>　　程大雷心中一动，莫非关二爷命中有此一劫，就是该找个姓华的为他刮刮骨。
<br/>　　“关二爷的命，你还能保多久？”程大雷问。
<br/>　　“以二弟现在的情况，至多六个月，六个月后，怕是大罗金仙也救不回二弟的命。”
<br/>　　嘟，获得任务，前往东海寻找华氏一族，期限六个月。
<br/>　　正这个时候，脑海里获得一条系统提示。程大雷从石凳上坐起，道：“莫怕，如此我就走一趟东海，把那华氏一族找回来，正好，也补充一下我们山寨的医务人员。”
<br/>　　看到程大雷如此信心满满，刘悲却有些不明白，当然，程大雷经常会说些稀奇古怪的话。
<br/>　　“大当家，西北距东海万里之遥，况那华氏一族已经很久没有消息。大当家有信心找到他们么？”
<br/>　　“怕什么，事情总是人做的么。再说，不是还有半年时间。这半年，你就守着关鱼，能用的药都用上，不惜代价保住关二爷的命。”
<br/>　　刘悲感激涕零，向程大雷深施一礼：“大当家做的事，我们兄弟都记得，我先替二弟谢过大当家的大恩大德。”
<br/>　　“好啦，好啦。”程大雷将他扶起：“都是自家兄弟，不必说这些。”
<br/>　　事不宜迟，程大雷一刻也不能耽搁。但蛤蟆城一摊子事情，程大雷也不能说走就走，最关键的一件事是：
<br/>　　徐神机就不好处理。
<br/>(本章完)
