　　“大师兄留下的东西？”
<br/>　　楚风眠眼神之中露出了几分激动之色，一下子站了起来。
<br/>　　对现在的楚风眠来说，只有曾经剑道门留下的东西，才足以让他这般激动了。
<br/>　　万年之前的事，楚风眠只是知道一个大概。
<br/>　　那个时候楚风眠没有一点点的灵力，所以需要当初的细节，楚风眠都不知道。
<br/>　　如果真有大师兄剑凌留下的东西，很有可能跟当年剑道门有关。
<br/>　　剑凌建立起北邙学院，也是希望有朝一日，北邙学院能出来一位强者，毁灭七大宗门，为剑道之主报仇。
<br/>　　楚风眠灵识笼罩而出，整个北邙学院，都被楚风眠尽收眼底。
<br/>　　现在北邙学院，还是无比平静，在这宫殿之内发生的事，他们都不知道。
<br/>　　北邙学院之中的一座座宝库，藏书阁等等，都被楚风眠尽收眼底，但是这些东西，楚风眠都不在乎。
<br/>　　在一直探查到北邙学院地下的时候，楚风眠突然发现了就在地底，万米之下，居然是有着一座宫殿。
<br/>　　在这一座宫殿内，上面的禁制，居然是足以抵挡楚风眠的灵识，让楚风眠都无法探察其中。
<br/>　　“应该就是这里！”
<br/>　　楚风眠脸色一喜。
<br/>　　在整个北邙学院之中，也就只有这一座宫殿，是楚风眠的灵识无法探察而出的。
<br/>　　大师兄剑凌，几乎是剑道之主麾下的最强者了，也是首徒，他的实力，楚风眠都现在都无法看得出，只怕是比起现在的楚风眠，还要更强。
<br/>　　只有他留下的禁制，在万年之后，还足以阻拦楚风眠的探察。
<br/>　　“你们下去吧。”
<br/>　　楚风眠看了一眼邓灵跟武皇二人开口道。
<br/>　　“是。”
<br/>　　邓灵跟武皇二人起身，身形一闪，离开了宫殿之中。
<br/>　　“我需要离开一天。”
<br/>　　知道了剑凌留下来的宫殿，楚风眠也有些焦急了。
<br/>　　“去吧。”
<br/>　　寒月漓跟洛汐笑了笑，他们也很少见楚风眠如此焦急的样子。
<br/>　　上一次楚风眠如此焦急，还是在六道魔门之中，被逼入绝境的时候了。
<br/>　　楚风眠点了点头，身形一闪，一下子来到了万米地下。
<br/>　　万米地下，对凡人也许是无法触及的存在，但是对楚风眠来说，却是一念到来。
<br/>　　达到古帝境界，就足以肉身横渡星空了，更何况区区地底。
<br/>　　楚风眠面前，一座气势磅礴的宫殿映入眼帘，在这宫殿之上，有着一道极为熟悉的印记。
<br/>　　一把看起来贯穿天地的灵剑，刻画在这宫殿的大门之上。
<br/>　　就是这一把灵剑，死死的守护住了宫殿的大门，万年一来，都无人进入。
<br/>　　“极道剑印！”
<br/>　　这印记楚风眠在熟悉不过了，正是只属于剑道门的印记，极道剑印。
<br/>　　如果说来到这里之前，楚风眠还有些怀疑那邓灵的话，现在见到这极道剑印的一刻，楚风眠心中便是在也没有任何的怀疑了。
<br/>　　能够布下极道剑印的，在这北邙学院之中，也就只有楚风眠的大师兄剑凌了。
<br/>　　这极道剑印，外人根本无法打破，纵然是以楚风眠的实力，强行打破，都会遭受灭顶之灾。
<br/>　　但是楚风眠也曾经是剑道之主的弟子，对着极道剑印如何化解，一清二楚。
<br/>　　“剑！”
<br/>　　在楚风眠的身上，一道剑意迸发而出，这一道剑意飞了出去，跟这大门之上的极道剑印，顿时融为一体。
<br/>　　轰隆！
<br/>　　大门轰然打开。
<br/>　　一步踏入其中。
<br/>　　映入楚风眠双瞳之中的，却是一把灵剑，这一把灵剑看起来足有七尺之长，上面的剑意，贯穿天地！
<br/>　　“凌天神剑！”
<br/>　　楚风眠一眼就认的出来，这一把灵剑，就是曾经剑凌的配剑，凌天神剑。
<br/>　　“发生了什么事？为何要呼唤我苏醒？”
<br/>　　轰隆！
<br/>　　就在这大门打开的一刻，从这凌天神剑之中，一股剑芒迸发而出，就在空中，化为了一道身影。
<br/>　　这一道身影，看起来无比沧桑，可纵然是过了万年，楚风眠也足以一眼看的出，这人影的模样。
<br/>　　“大师兄！”
<br/>　　楚风眠喊出的一刻，声音都有些颤抖。
<br/>　　在这宫殿之中留下的，居然是剑凌的一股灵力所化的化身，就像是在九玄仙尊洞府之中，那九玄仙尊一样的存在。
<br/>　　“大师兄？你是？剑白师弟？”
<br/>　　那人影听到楚风眠的称呼，都要突然在空中一震，随即看向楚风眠，语气之中都是不可置信的说道。
<br/>　　“你，你怎么还活着？”
<br/>　　楚风眠掉落天绝峰下，已经死去的消息，在他后来逃出之后，也曾经听说过。
<br/>　　可现在，楚风眠居然是出现在了他的面前？
<br/>　　虽然是样子改变，但是这剑凌，依然是一眼足以认得出这就是他曾经的师弟，楚剑白。
<br/>　　“这事，说来话长了……”
<br/>　　楚风眠感叹了一声，放松了下来。
<br/>　　曾经剑道之主，虽然是楚风眠的师尊，但是事务繁忙，楚风眠学习剑道的时候，许多都是跟这剑凌所学习的，可是说他就是相当与楚风眠兄长一般的存在。
<br/>　　在他的面前，楚风眠不由自主的就会放松下来，好像是回到了万年之前，那无忧无虑的时光之中。
<br/>　　楚风眠诉说着他重生以来的故事。
<br/>　　剑凌默默的听，没有打断。
<br/>　　一直是等到楚风眠说完，剑凌才感叹了一声道。
<br/>　　“世事无常，转世重生，这等虚无缥缈的事情，居然是真的，没想到这一世，你已经足以修炼剑道了，你的资质，要超越师尊，未来不可限量啊。”
<br/>　　前世的时候，剑道之主就无数的夸奖过楚风眠的剑道领悟。
<br/>　　一个废人，没有灵脉的废人，可以拜入剑道之主门下，所凭借的，就是这一身剑道领悟。
<br/>　　万年之前，楚风眠绝对是名声最大的一名废物了，不少人都对当时的楚风眠嗤之以鼻，剑道再强，无法修炼，也终究是一个废物。
<br/>　　可这一世，却彻底改变了，这改变，没有任何一人，可以想到。
<br/>　　“转世重生，真的是虚无缥缈的事？”
<br/>　　楚风眠看着剑凌，突然开口道。
<br/>　　“我曾听闻，只要是踏入天人境，羽化登仙，就可以转世重生，永生不死？”
