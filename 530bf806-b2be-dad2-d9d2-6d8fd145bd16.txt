　　妖族大量的强者自爆，也激起了巫族的凶狠，紧接着无数巫族强者也自爆起来。
<br/>　　“轰！”
<br/>　　“轰隆隆！”“隆隆！”……
<br/>　　整个洪荒顿时地动山摇，大陆破碎，虚空坍塌，山河颠覆，星辰爆炸。无数三族强者死亡，火光冲天，声如雷震，惊天动地。
<br/>　　“混蛋，就算要死，也给我滚远点。”
<br/>　　玄天看到这一幕，顿时脸色大变，连忙祭出武塔，快速变大，将无数人族收拢过来，收入塔中，如今情况危急，能拯救多少就算多少，他可不想让人族就此落幕啊。
<br/>　　妖族妖师鲲鹏见此，知道妖族大势已去，不可挽回，他连忙幻化出本体，卷起地上的河图洛书两件先天灵宝，便向着北冥方向飞去。
<br/>　　“这……”
<br/>　　三清、女娲、接引、准提等圣人愣住了，，先是东皇太一被人族玄天打残镇压了，紧接着，帝俊与祖巫也分出了胜负，还未等他们反应过来，就看到妖帝帝俊疯狂的自爆了，紧接着，无数妖族更是尾随帝俊一起自爆，之后好像起了连锁反应，巫族也跟着自爆起来，拉着对方走向毁灭。
<br/>　　紧接着，就天崩地裂，星辰破碎，洪水泛滥，一桩接着一桩的事情，让人应接不暇。
<br/>　　“嘶！”
<br/>　　三清等圣人以及无数关注此战的大神通者们，纷纷倒吸一口气，目之所及，只有斑斑血迹，断裂的刀剑，堆积如山的尸骨残骸，巫妖两族修士更是十不存一。
<br/>　　冷风呜咽，稀稀落落的剩余人族等三族修士，在压抑的呻吟，哭泣，洪水泛滥，一片凄凉。
<br/>　　娲皇宫中，女娲圣人突然一声大叫：
<br/>　　“哥哥！”
<br/>　　紧接着，她瞪了元始一眼，随即，她的身影便消失在娲皇宫中。元始面色不变，一步踏出，同样消失在娲皇宫中。
<br/>　　洪荒之中，女娲的身影迅速出现，她挥手将身边的巫族化为飞灰，向着那爆炸中心一抓，顿时，抓出一道魂魄，正是伏羲妖圣的魂魄。
<br/>　　“哥哥，怎么会这样？”
<br/>　　女娲眼中闪过痛苦之色，看着手中的残魂，眼中茫然无比。
<br/>　　元始的身影此刻也出现在洪荒之中，他看着破败不堪的大地，滚滚泛滥的洪水，火山爆发般的大山，皱了皱眉头。
<br/>　　此刻，太上、接引等圣人也出现在洪荒之中，看着残破不堪的洪荒大地，太上开口问道：
<br/>　　“师兄，这该如何是好？”
<br/>　　“嗯，我等出手，将洪荒之中的****平息，将破碎的大陆拼接起来，至于那些流入到时空乱流或者星空深处的大陆碎片，便随他们发展吧。”
<br/>　　“大兄此言甚是。”
<br/>　　“善哉。”接引等也赞同道。
<br/>　　“轰！”
<br/>　　接着，元始等圣人出手，平息****。元始祭出混沌钟，化为一道镇压诸天的大钟，钟声一响，瞬间便镇压那暴乱的地水火风，那风暴不断的空间乱流，也随之被镇压下来，紧接着，他用大法力将之平息。
<br/>　　太上祭出太极图，天上顿时出现一道遮天蔽日的阴阳太极图，太极图覆盖天穹，阴阳流转，相互转化，将周围一切****转化平息。
<br/>　　接引祭出功德金莲，功德之光闪烁，照耀下来，普度众生，将此次战场上所陨落留下的魂魄度化，送进幽冥界。
<br/>　　准提祭出菩提法相，木之力运转，造化万物生机。
<br/>　　通天祭出诛仙四剑，勾连地脉，疏通河道，平乱火山，改造山河，重塑日月。
<br/>　　……
<br/>　　整整三天，诸圣才将破败的洪荒重新改造了一遍，使得大地重新焕发光彩，星空再次变得璀璨，生机再次降临人间。
<br/>　　不过，大陆终究是被打破过，洪荒本源有所损伤，使得洪荒之中的灵气有所倒退，洪荒中的灵气从开天初的混沌灵气，变化为上古时的先天灵气，如今经过此次大战，灵气已经开始向后天灵气转化了。
<br/>　　洪荒还有许多碎片，流入到了时空乱流之中，消失不见，在那些碎片之中，还生存这许多生物，有人族，有妖族，有巫族，有魔族，也有其他种族。随着时间的流逝，它们都会衍化成一个个小千世界，或者中千世界，它们的文明进化，修炼发展也会有所不同。
<br/>　　一切，都在向着一个未知的领域在发展。
<br/>　　此次大战的影响，还远不止如此，巫妖两族近乎灭族，族人所剩无几，退下了历史舞台，人族的崛起，已经势不可挡了。
<br/>　　此次大战，虽说巫妖两族近乎灭族，但是依旧剩下一些族人，特别是妖族，仍有十几万，巫族少些，但也有十万左右，最重要的是，巫族还剩下后羿、玄冥两位祖巫，以及刑天、蚩尤等顶尖大巫。
<br/>　　当初帝俊自爆，玄冥身为女子，在知道躲不了后，第一时间被帝江等三大祖巫保护起来，因此，虽然被重创得奄奄一息，但也得以保全了性命。至于后羿，那更是因为他在大战之前就受了重创，没有参加大战，所以保全了性命。
<br/>　　妖族的妖神，也剩下白泽、鬼车等几位，至于其他的，大多都已经陨落了。
<br/>　　元始身为众圣之首，众人的大师兄，他当即淡淡开口道：“诸位说说，如今该如何处理这些余下的巫、妖二族？”
<br/>　　接引满脸愁苦，开口道：“巫妖大战，生灵涂炭，吾怜其修行不易，不如交由我将之点化，以去其戾气，造福苍生。”
<br/>　　女娲在旁，听了此话，心中暗骂一声，好不要脸，嘴上却淡淡道：
<br/>　　“此行怕是不妥，本宫为妖族圣人，这些妖族乃本宫之子民，不如由我收押，我有山河社稷图，正好用来关押这些妖族，令其默默思过。”
<br/>　　准提道：“你既然为有圣人，又怎么责罚他们？我看妖族与我西方教有缘，不如让我度化，修行积善，日后才能证得大道。”
<br/>　　通天的性子直来直去，向来看不起准提，听了他的话后，便指着剩下的巫族，说道：
<br/>　　“道友既然有如此慈悲，那为何不将这些巫族收了呢？巫族好战嗜杀，留着也是个祸害，不如就由道友将之度化吧？”
<br/>　　西方教修的是今生来世，巫族没有元神，一旦死亡，便会真灵自灭，不入轮回，怎么能修西方神通呢？
<br/>　　准提微微一笑，也不在意通天的为难，笑道：“我西方只度有缘之人，可惜巫族与我无缘，天数如此，奈何奈何。”
<br/>　　通天嘿嘿一笑，道：“准提道友倒是好神通，尔等度化众生，怎么连你们西方自己地盘上之人也不曾度完啊？”
<br/>　　准提与接引脸色阴沉，他们知道通天说的是西方太华山上的劫魔殿，自从此次大战，劫魔殿那深厚的底蕴便显现在众人的眼前，让洪荒众多大神通者为之震惊。
<br/>　　“诸位圣人可是担忧这剩下的巫、妖二族，不如交给本祖来处理好了，全部杀了，如何？”
