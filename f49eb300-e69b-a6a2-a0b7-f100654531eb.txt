　　叶少阳冷笑：“大道理我听得多了，但是他们至少有信仰，我曾经也有，今天却崩塌了。”
<br/>　　道风不做声，拉着他继续往前飞，飞过闹事，来到山区，叶少阳也不知道道风要去哪，索性就被他拖着往前飞。
<br/>　　飞过闹市，飞到野外山上，一直往前飞，突然，一声惨叫从下方传来。叶少阳低头一看，在一片树丛里，一个东西在地上趴着，跟鳄鱼似的，手里撕碎了一个人，满地都是鲜血，对面还有一个人吓的瘫坐在地上，尖叫声就是这个人发出的。
<br/>　　叶少阳再看了一下，才发现下面是沼泽地，那僵尸应该是水尸，在水草中爬行着，朝那个吓瘫了的人游过去。
<br/>　　“停下！有水尸！”
<br/>　　叶少阳指着下方。
<br/>　　道风低头看了一眼，道：“为什么你要去救人？”
<br/>　　叶少阳一下子就懵了。
<br/>　　为什么？
<br/>　　除魔卫道，不是法师该干的事吗？
<br/>　　但他陡然就想到了自己的身份，除魔，自己可能就是魔……卫道，卫的是什么道？欺骗自己、利用自己的那个道？自己有什么资格，又有什么义务去这么做？
<br/>　　天下再乱，邪物伤人，跟自己有什么关系？
<br/>　　但是……看着那水尸一点点靠近岸上那人，叶少阳内心在做着剧烈的斗争。
<br/>　　道风可以放慢了飞行速度，等着他。
<br/>　　“走吧！”
<br/>　　道风什么也没说，带着他离开。
<br/>　　水尸爬上了岸，朝坐在地上的人身上扑过去，在即将咬住他喉咙的那一刻，突然一只铁钩从后面甩过来，正好勾住他的脖子，用力一拉，水尸侧翻在地上，没等起身，锁链从后面个又连续绕了几圈，水尸力大无穷，但尽管用力扭动还是无法挣脱。
<br/>　　一道人影飞来，直接钻进他身体，水尸浑身剧烈抖动，几秒钟后，肉身崩裂，碎了一地。
<br/>　　叶少阳站在岸边，朝获救那人走过去，突然，那人化作了一道烟，叶少阳猛然一怔，回身望去，见水尸那毁掉的肉身也化作了一轻烟，猛然明白过来，抬头看着从天而降的道风，质问他：“是你捣的鬼吧。”
<br/>　　“小小幻术，你也不想想，这世上哪来这么多邪物，随便就能让你遇到一个。”
<br/>　　“你！”
<br/>　　叶少阳冲过去，一把抓住道风的衣领。
<br/>　　“事情是真是假，对你都是一样的，少阳，你已经做出选择了。”
<br/>　　叶少阳松开他，失神地摇了摇头，“我总不能干看着有人死在邪物手上吧？”
<br/>　　“这就是你的选择啊，是你的道。你是不是天选之人，别人是不是骗了你，跟你去做什么，有什么关系吗？”叶少阳走到树下坐着，默默地思索起来，道风走到他身边，跟他并排坐下。
<br/>　　“你一直都知道我的真实身份吗？”
<br/>　　“不，我跟你一样，也被骗过，起先，我离开人间，就是为了寻找我自己，后来我知道了‘真相’，我以为你是天选之子，但我觉得你不可能是无极鬼王的对手，于是谋求斩尸证道，成立风之谷，想要靠自己的力量铲平太阴山……就在不久之前，我知道了真相，原来，我也不过是个棋子。”
<br/>　　道风一只手搭在叶少阳肩膀上，揉着他的脑袋，“当棋子的感觉，很不爽，所以我才要你变得更强大，跳出棋盘，不再被任何人掌控。”
<br/>　　“所以你做到了。”
<br/>　　“是了，因此，大帝才对我变了态度，一路追杀，我而今又证道不久，实力有一段真空期，不得不躲在山河社稷图中静养修炼……”
<br/>　　说到这，他深邃的目光望向远处，缓缓说道：“我血洗悬空观，失宠于人间法术界，梨山一战，与阐教和佛门结仇，两次大闹轩辕山，得罪了七大长老和盘古僧，更不要说太阴山和尸族……如今又彻底开罪酆都大帝，天下之大，几乎没有我容身之处，但是……少阳，我所做这一切，都是为了今天能跳出棋盘……这结果，是值得的。”
<br/>　　叶少阳定定地看着他，默然道：“你在我心中，一直都是无坚不摧的强者。只是，你能做到的，我未必能做到。”
<br/>　　“你已经做到了，现在你只是要迈过自己内心这道坎。”
<br/>　　“我……尽力。”
<br/>　　道风听到他这么说，心情也是陡然放松下来。
<br/>　　叶少阳却抓住了他的手，轻轻靠过来，喃喃道：“我很想师父。”
<br/>　　“以前，有师父在，什么事都能帮我顶着，我可以耍赖，可以躲在山上不下去，什么事都能搞得定，这一晃，什么事都得我自己来了，哦，多亏还有你。师兄，余生，请多多指教。”
<br/>　　道风心中一动，用力揉了一下他的头发，放松下来，说道：“这条路总归不是你一个人走，有我陪着你呢。而且看上去辛苦，其实也乐趣无穷，毕竟，命运不再被任何人掌控。”
<br/>　　这一句话，倒是让叶少阳对未来充满了期待，笑了笑道：“这条路走到终点是哪里？”
<br/>　　“不知道，但是，这样不是更有趣吗？”
<br/>　　终点，便是命运，既然已经跳出了命运，又怎么会知道终点在哪儿，随心所欲，照自己心思去做就好了。
<br/>　　叶少阳并没有彻底悟道，却已经迈过了命运的这道坎。
<br/>　　“要是我不是天选之子，那冷玉是不是转世鬼童，还是像你们之前说的——”
<br/>　　“先回去，还有很多事你不还不清楚，我们回去接着聊。”
<br/>　　道风不由分说，拖着他往来处飞行。
<br/>　　小九也纵身飞行，远远地在后面跟着，之前发生的一幕幕她都看在眼里，对话也都听到了。少阳这么快就迈过这道坎，她也感到无比的欣慰，而现在事情还没说完，她不想现身打乱节奏。
<br/>　　“那你先告诉我，你实力恢复了没有？”在空中飞行时叶少阳问道。
<br/>　　“只恢复到了六成，但毕竟证道了，上限也比之前高了数倍。”
<br/>　　把叶少阳拖回家里，魂魄塞进他的肉身，道风立刻打开虚空裂缝，又将他带到了青丘山。
<br/>　　林三生三人都还在，看到叶少阳和道风回来，一个个瞪大眼睛看着他们。
