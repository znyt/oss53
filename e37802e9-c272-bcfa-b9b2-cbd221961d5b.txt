　　云破天和另外两个人，都是无极宗的人，叶尘早晚都会和无极宗撕破脸皮，所以他自然不会留手！
<br/>　　云破天显然也察觉到了叶尘的杀意，顿时脸色大变，刚巧此时此刻，他退到其中一名无极宗弟子的身旁，于是想都不想，一把抓住那个同伴的胳膊，果断将其朝叶尘丢了出去。
<br/>　　“彭！”
<br/>　　那人根本没有想到，云破天竟然会拿他做挡箭牌，还完全没有反应过来，就已经被叶尘一拳生生打爆，化成了一团血雾！
<br/>　　“啊！！”
<br/>　　另外一名无极宗弟子，看到同门师兄弟的惨状，顿时惊呼了一声，吓得脸色煞白，同时又对云破天的残忍感到极度恐惧，当即取出传送玉牌，直接将其捏成了粉碎。
<br/>　　“哗啦！”
<br/>　　在传送玉牌破碎的那一刹那，那名无极宗弟子，便被一股强大的空间规则之力所包裹，转眼便凭空消失了，显然已经被送回了初始星。
<br/>　　而另外一边，云破天用自己的同伴挡住了叶尘的攻击之后，他几乎是连滚带爬，瞬间冲到后方，那具冰甲猛犸象的尸体，然后朝腹部的位置用力一掏，一颗金灿灿的妖丹，便被云破天取了出来！
<br/>　　不得不说，云破天这个人，不但心狠手辣，而且反应极快，在明知道远远不是叶尘对手的情况下，依然没有捏碎传送玉简，而是选择最后一搏！
<br/>　　“咔嚓！！”
<br/>　　就在云破天一手捏碎妖丹，要将其炼化的同时，叶尘此刻也已经追了上来，
<br/>　　“死吧！！”
<br/>　　叶尘脸色冰冷，直接一掌拍出，将所能动用的真元全部调动了起来，狠狠一掌砸落在了云破天的胸膛之上。
<br/>　　“噗！！”
<br/>　　云破天顿时再次狂喷了一大口鲜血，整个人犹如断了线的风筝般，直接倒飞了出去，连同胸口的位置，也深深的凹陷了下去。
<br/>　　“嗯？”
<br/>　　叶尘本以为，以云破天如今只有筑基境五重的修为，他这一拳砸落下去，云破天肯定必死无疑，没想到竟然被对方给挡住了！
<br/>　　不过很快，叶尘便发现了其中缘由，
<br/>　　“是护身仙器吗？倒是我大意了！”
<br/>　　原来，这云破天身上所穿的衣服，乃是一件极为稀有仙器级别的护体法衣，和那些需要真元加持的法宝不同，这护体法衣，在这个地方，纵然无法完全施展，却也足以抵挡叶尘那一掌，四、五成的力道。
<br/>　　再加上，云破天本身也是化神境的修为，而且也是法武双修，肉身修为并不比叶尘差多少，所以竟然勉强挡住了叶尘这一掌，居然没死！
<br/>　　而等叶尘反应过来之后，再次追至跟前，那云破天却已经将那枚妖丹炼化，继而周身出现了一股强悍的规则之力，将其包裹在了其中，令叶尘已经无法靠近。
<br/>　　“小子！今日之仇，本公子记下来！来日本公子会加倍讨还！你给我等着……”
<br/>　　云破天那赤红的双眼，死死盯着叶尘，威胁的话语还没有说完，整个人就已经被那股强悍的空间规则之力，传送去了下一重星域。
<br/>　　说时慢，其实从叶尘出现，到无极宗的四人，死的死，逃的逃，也不过一转眼的功夫。
<br/>　　诺大的雪林之中，便只剩下叶尘和洛冰玄两人了。
<br/>　　“还是让他给跑了！”
<br/>　　叶尘一闪身，回到洛冰玄跟前，微微叹了一口气道
<br/>　　虽然被云破天成功逃脱，让叶尘多少有些郁闷，不过其实倒也不算太出乎意外。
<br/>　　毕竟是一代天骄人物，无论是修为，还是心智，都远超常人，而且各种保命手段层出不穷，让人防不胜防，想要彻底将其击杀，肯定没有那么容易。
<br/>　　洛冰玄微微颔首，俏脸上震惊之色，依旧没有散去，不由得深深望了一眼叶尘，
<br/>　　她是真的没有想到，叶尘居然有如此实力！
<br/>　　轻而易举便斩杀了两名无极宗的天才，吓跑了一个，而且连云破天，虽然侥幸逃脱，也被他打成了重伤！
<br/>　　恐怕即便是那龙辰，也做不到如此程度吧？
<br/>　　难道此人的天赋，竟然还在龙辰之上？
<br/>　　“你怎么样？还能坚持吗？我来帮你处理一下伤口吧？”
<br/>　　叶尘望了一眼洛冰玄背部，早已经被鲜血全部浸透的白色裙衫，脸上顿时露出一抹担忧之色。
<br/>　　说起来，两人前世也算是关系不错的朋友，虽然这女人冷是冷了点，对他一向不见辞色，但对曦月却如亲妹妹一般。
<br/>　　眼看他受伤如此之重，叶尘自然不能置之不理。
<br/>　　说话间，叶尘便要伸手去扶住，此刻已经摇摇欲坠的洛冰玄，然后为他疗伤。
<br/>　　却不料，洛冰玄不知道是那根神经没有搭对，忽然一连倒退数步，冷喝一声道：
<br/>　　“你！不准碰我！”
<br/>　　叶尘顿时一愕，刚刚伸出的手，顿时僵硬在空中，不由得好笑道：
<br/>　　“我说大姐，我若不碰你，怎么给你疗伤？你可要想清楚了，在这个地方，连储物法宝都无法使用，没有疗伤丹药，你的伤口又如此严重，如果不及时处理的话，可是会要你命的！”
<br/>　　洛冰玄听到叶尘这话，顿时脸色微微一变，显然也十分清楚，叶尘所说的都是实情，她如今的伤势，若是继续耽搁下去，别说和其他天骄一争高下，甚至很有可能会丧命在这里！
<br/>　　只不过，她一面对叶尘，心中就会有一种奇怪的感觉，从而会严重影响到她的心境，这让她不敢去冒险，不敢和这个男人产生任何感情纠葛。
<br/>　　一番纠结之后，
<br/>　　最终，洛冰玄一咬牙，继续一脸冰冷，道：
<br/>　　“我怎么知道，你是不是想害我？总而言之，你不准靠近我！我的伤势，也用不着你操心！”
<br/>　　说完这话，洛冰玄脚下猛然一顿，直接纵身跃起，便要迅速离开这个地方。
<br/>　　可惜，她如今已经深受重伤，哪里还经得起折腾，才刚刚奔出数千米，顿时感觉气血不继，脑袋不由得一阵眩晕，发出一道娇呼之后，竟然直接从空中重重坠落了下去。
<br/>　　更为糟糕的是，不知道是不是天意，好巧不巧的，此时下方，赫然正是一片妖兽群！
