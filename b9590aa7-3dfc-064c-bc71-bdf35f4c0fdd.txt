        &lt;h3&gt;第三卷第一百二十五章  威胁&lt;/h3&gt;
<br/>
<br/>        江俞轩接手西城的第一个项目就是他们跟着的洛城的那个项目。
<br/>
<br/>        原本他们对这个项目看得很好，只是后来经过那个项目一拖再拖，一个项目都拖了将近两年，后来他们也不抱什么希望了，陈子霖却接到了傅海生的电话。
<br/>
<br/>        他们是生意人，只要有生意当然会去做的，况且前期还做了一番工作。
<br/>
<br/>        江俞轩也不会客气，安排人去买了标书，这个项目是在省网上挂出来的，招标也是在某代理公司。
<br/>
<br/>        标书买回来之后，江俞轩就安排呼延寰宇带人做标书，至于现场他们之前已经让人驻扎在哪里一段时间了，情况还是比较熟悉的，而且提的要求也和当初一样，变动不是太大。
<br/>
<br/>        项目还是可以做的。
<br/>
<br/>        再要开标的一个礼拜，江俞轩接到了一个电话；
<br/>
<br/>        “江总，你好！”一个女声嗲着声音。
<br/>
<br/>        江俞轩感觉很熟悉，但是一时又想不起来是谁。
<br/>
<br/>        “请问，你是？”江俞轩很想挂了电话。
<br/>
<br/>        “江总，真是贵人多忘事，我是夏颜。”
<br/>
<br/>        “哦，原来是夏总啊。”江俞轩微一愣，夏颜找他？
<br/>
<br/>        “是啊，江总不知道最近这两天有没有时间？”夏颜问
<br/>
<br/>        “夏总有什么事情吗？”
<br/>
<br/>        “江总报名参加了洛城的项目？”
<br/>
<br/>        “是！”
<br/>
<br/>        “如果江总有兴趣的话，我们见一面如何？”夏颜说。
<br/>
<br/>        “好啊，夏总定时间。”
<br/>
<br/>        “那就明天下午江总来我办公室一趟吧。”夏颜在电话那头笑着说。
<br/>
<br/>        “好的。”江俞轩也想知道夏颜葫芦里卖什么药。
<br/>
<br/>        陈子霖曾经说过，夏颜会对这个项目会进行阻碍，不但会对他们的这个项目上会有阻碍，有可能对西城分公司也会有阻碍，但是因为西城他们所做的类似的项目这几年基本都已经完工，后期都是服务性质的，夏颜也就没有什么能力来阻止。
<br/>
<br/>        只是他们在很多别的项目上确实份额比之前少了很多，这些到底是不是夏颜的功劳他们也说不清楚。这几年出现了很多的代理公司，当然有利也有弊。
<br/>
<br/>        江俞轩身兼数职，其实他原本可以不理夏颜的，但是他很是好奇夏颜想干什么，她有项目样的目的。
<br/>
<br/>        江俞轩第二天到了夏颜的办公室。
<br/>
<br/>        夏颜是某信息部的老总，虽然是老总，但是因为单位房子紧缺，她的办公室也是一间房，大概十几平米，办公室很简陋，一个单人折叠沙发，一个铁皮文件柜，一张办公桌，一台电脑一台打印机，办公桌上堆放着乱七八糟的的文件。
<br/>
<br/>        一台饮水机，饮水机旁边是一个衣服架，衣架上挂着貂皮大衣，一个限量版的背包。
<br/>
<br/>        夏颜似乎是特意在等江俞轩。
<br/>
<br/>        她穿着低v长款塑腰连衣裙，将成熟的身材很好的显露出来，而那山茶花图案，黄十八k金，镶嵌钻石项链随着主人的移动而晃动着，**的手腕上那款十八k白金镶钻，镀银金-珍珠贝母的腕表显示着主人的身份。
<br/>
<br/>        “江总，好久不见。”夏颜笑着招呼江俞轩。
<br/>
<br/>        “好久不见。”江俞轩也是很客气的回应着。
<br/>
<br/>        “请坐，办公室很简陋，让江大少来我办公室真是委屈了江大少。”
<br/>
<br/>        “再简陋的办公室也掩饰不了夏总贵气的身份啊。”江俞轩感叹着，陈子昂那货好像很少有珠宝首饰哦。
<br/>
<br/>        “哪里，我们只是拿着一份死工资的人，哪里有什么贵气的身份呢。”
<br/>
<br/>        “夏总客气了，你这年纪轻轻，身份地位可不是一般人能比得上的。”江俞轩说。
<br/>
<br/>        “江总真是客气，我们这身份怎么能和江总比呢，江总才是人中龙凤，让人仰慕不已呢。”
<br/>
<br/>        夏颜的话让江俞轩浑身起了一层鸡皮疙瘩。
<br/>
<br/>        “麻蛋，我可是为我家楠楠守身如玉三十载呢，你千万不要仰慕我。”江俞轩心里暗暗的骂了一句。
<br/>
<br/>        “呵呵呵，夏总也是国色天香，不知道令多少人仰慕呢。”江俞轩呵呵笑着说。
<br/>
<br/>        “江总，真是会说笑，我有点不明白，不知道能不能向江总请教。”夏颜笑颜逐开。
<br/>
<br/>        “在夏总如此美丽的人面前，知无不言。”江俞轩也笑着说。
<br/>
<br/>        “江总，江总的身份怎么会屈居在陈子昂的手下？”夏颜盯着江俞轩一瞬不瞬的看着。
<br/>
<br/>        “唉，签了卖身契了呗？”江俞轩故作无奈的说。
<br/>
<br/>        “你们签了用工合同？”夏颜问。
<br/>
<br/>        “是啊，当初不懂，一签就是好多年，依着我江大少的身份又不好毁约，只好干下去了。”江俞轩叹口气。
<br/>
<br/>        “原来还以为你们是一对，后来才知道你和张美女快要结婚了。”夏颜笑着说。
<br/>
<br/>        “你怎么知道？”江俞轩故作吃惊的问。
<br/>
<br/>        “现在的社会想了解一个人还不容易啊，尤其是像你这样的名人，只要有心当然是会知道了，怎不可能像有些傻子一样。”夏颜意有所指。
<br/>
<br/>        “夏总说得是，那么夏总不知道今天叫我来是有什么事情？”江俞轩问道，时间宝贵，他可没有时间来和夏颜聊天。
<br/>
<br/>        “江总知不知道，陈子昂和张家的瓜葛？”夏颜问。
<br/>
<br/>        “夏总是怎么知道的?”
<br/>
<br/>        “陈子昂的二嫂冯睿告诉我的，这几年冯睿一直为陆家和张家提供着陈子昂的信息。如果陈子昂稍有差错，陆家和张家都不会放过他们家的，这几年冯睿一直为陈子昂说着好话，现在冯睿和陈子建离婚了。”夏颜微笑着说。
<br/>
<br/>        “呀，那现在谁要是和陈子昂有联系岂不是倒霉了？”江俞轩大惊。
<br/>
<br/>        “是啊，要是张大美女误会你和陈子昂了怎么办？而且冯睿还说你们都有孩子了。”夏颜看着江俞轩的眼神充满了同情。
<br/>
<br/>        “你相信这种鬼话吗？”江俞轩问道
<br/>
<br/>        “我不相信，但是别人相信啊，比喻说冯睿。”夏颜说
<br/>
<br/>        “那你说我现在该怎么办？”江俞轩着急的问。
<br/>
<br/>        有人送文件来让夏颜签字，夏颜摆摆手：“明天来，没有看到我这里有事情吗？”
<br/>
<br/>        “江总，和陈子昂划清界限啊，你们江家那么有钱的，你干嘛还要在陈子昂的公司呆着呢？”夏颜说。
<br/>
<br/>        “那我该怎么和她划清界限呢？望夏总给点拨一下。”江俞轩用手擦了一下脸。
<br/>
<br/>        “陈子昂的公司最近想参加洛城的项目是吧？”夏颜问
<br/>
<br/>        “是的，已经报名了。”
<br/>
<br/>        “那我再给江总透露一下信息，洛城了两个项目我们都拿下来了，他们过去也拿不到。”夏颜满脸骄傲。
<br/>
<br/>        “夏总方便透露一下你们的公司名称吗？”江俞轩问道。
<br/>
<br/>        “公司名称就不透露了，那个公司的负责人是我的高中同学，也是陈子昂的高中同学。我已经告诉江总这么多了，江总自己衡量吧，对了，冯睿这几天会去北京的。”夏颜抬起手腕，看了一眼手表。
<br/>
<br/>        “谢谢夏总告诉我这一切，我会慎重考虑的，另外我会阻止陈子昂的公司去参加招标，毕竟我现在还在他们公司任职，我说了还是算话的，或者说让他们走一个形式，你看这样行不行？”江俞轩考虑了一下，对夏颜说。
<br/>
<br/>        “江总不但帅气还很聪明，改天有时间了我请你吃饭，一定要来参加啊。”夏颜高兴的说。
<br/>
<br/>        “吃饭就不比了吧。”
<br/>
<br/>        “我叫上马总相陪，江总觉得如何？”夏颜看着江俞轩那张帅气的脸，眼神一刻也不行离开。
<br/>
<br/>        “那，好吧，我感谢夏总，今天就不打扰夏总了。”江俞轩站起来准备走。
<br/>
<br/>        “好，江总，不送，我们下次见。”夏颜伸出手。
<br/>
<br/>        “不好意思，你看，我这手上都是汗，不敢脏了夏总的手，你看，呸！”江俞轩并没有和夏颜握手，并且“呸”的吐了一口口水在地上，拿脚摩擦了几下。
