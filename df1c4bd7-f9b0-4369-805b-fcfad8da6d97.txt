　　一名大臣忍无可忍，起身道：“国师不要以为我们不知道你的目的，你想留天女在天运国，不过是看上了她的美貌！”
<br/>　　国师训斥道：“荒唐！”
<br/>　　眼看吵架已经升级，首位上的国王淡淡开口：“今天的会议就到此结束。”
<br/>　　一干大臣无话可说，只能低头行礼：“恭送国王陛下。”
<br/>　　国师嘴角轻浮一勾：“国王陛下请留步，今夜月亮很圆，不如……”
<br/>　　国王头也不回，冷淡道：“国师还是先处理好国师府上的百名美丽女子吧，很多父母都告到了王宫，你情我愿国家管不着，但强抢的话，还希望国师自重。”
<br/>　　一番话说的国师面色阴沉，大臣们也一个个冷哼离开。
<br/>　　更有一人摆着臭脸：“连国王都敢肖想！”
<br/>　　国师瞬间目光阴森的看着此人：“国王也到了适婚年纪，试问整个天运国除了我谁能配得上？不嫁我难道嫁给你吗？”
<br/>　　“嫁？！”那人吹胡子瞪眼：“国王是至高无上的！要嫁的不是国王，国王是娶！”
<br/>　　“哈哈哈！”国师大笑出声，言语更是轻佻：“若不是王室无后，怎么可能轮得到一个公主当国王，不过是天运国无人罢了，公主当上国王的这些年，我可是尽心尽力，她嫁给我后，我自然会帮她打理这整个天运国。”
<br/>　　“你放肆！”那名大臣起的面色涨红。
<br/>　　但面对国师却怎么也不敢动手，却正如国师所说，他已经在天运国无敌了。
<br/>　　紧急会议不欢而散，国师更是自大无比的离开，引起了天运国的百名大臣不满却毫无办法。
<br/>　　年轻的国王回到房间，第一时间就反扣上了门，而后重重的叹了口气，坐在床边缓缓的将厚重又端庄的衣服褪去。
<br/>　　楚涵瞪眼一看，这是国王的衣服没错！
<br/>　　于是他一下子就从床底下钻了出来。
<br/>　　于是，大眼瞪小眼。
<br/>　　还没来得及换上睡衣的国王就站在楚涵面前，与白天的场景一样，那叫一个香·艳。
<br/>　　国王张嘴想大叫，楚涵眼疾手快，一把就将国王擒住，一只手死死捂住国王的嘴。
<br/>　　年轻的国王面色惨白，她低头看了眼，然后就差点没晕过去。
<br/>　　她竟然赤·裸·着，被一个陌生男子死死搂在怀里！
<br/>　　她从小到大，还没被人这么欺负过！
<br/>　　楚涵也很高尴尬，说好的国王呢，怎么又是这个女人？
<br/>　　“你不许叫，我把衣服递给你。”楚涵别开眼，道。
<br/>　　国王连连点头，但就在楚涵刚松手的一瞬间，一声尖叫就猛然脱口而出。
<br/>　　“啊——”
<br/>　　楚涵那叫一个气，整个将国王拎起来就仍在床·上，手一伸就把被子一拉，将两人裹在了被子里。
<br/>　　嘭！
<br/>　　一群人破门而入，惊慌失措的侍女及侍卫将房间占得满满当当。
<br/>　　“国王陛下！”
<br/>　　他们跪在地上，满眼担忧。
<br/>　　这时候在被子里的国王，完全被楚涵禁锢的不能动，这种场景已经不是能用言语解决的，更何况她还没穿衣服，如果告诉别人，被子被掀开……
<br/>　　一想到这里，国王就整个快疯，努力让自己冷静下来。
<br/>　　“我只是做噩梦了，你们出去吧。”她道。
<br/>　　一群人狐疑的抬头看了眼，但很快就全体退了出去。
<br/>　　屋中不在有第三人后，楚涵这才将手松了松。
<br/>　　国王却在得到自由后直接翻身，用利落的身手将楚涵整个人摁在床·上，一柄匕首更是从枕头下摸出，想要一口气将楚涵杀死！
<br/>　　但楚涵就这么直接伸手，挡住了国王的手腕。
<br/>　　那匕首就蹲在空中，再也无法靠近一分！
<br/>　　国王脸都憋红了，羞辱感涌上心头，紧接着她闭上眼，忽然就将匕首反向一刺。
<br/>　　她要自尽！
<br/>　　楚涵内心翻了个白眼，手一翻将国王的整个手腕握住，让其进退不得。
<br/>　　杀楚涵不行，自杀也不行。
<br/>　　国王满眼的绝望，她整个人都无法动弹，要命的是这种姿势，被楚涵看的一览无遗。
<br/>　　没多久，大颗大颗的泪珠就这么滚落下来。
<br/>　　她真的不知道还能怎么办！
<br/>　　楚涵叹口气别开眼：“我有妻子了，四个，都比你漂亮，早就跟你说我松开你，你穿衣服，还不听。”
<br/>　　国王更特么委屈了，这回不是默默哭，竟然一下子哭出了声。
<br/>　　她好歹是国王啊！
<br/>　　有这么嫌弃吗？
<br/>　　这回楚涵是彻底不懂了，皱眉道：“让你穿衣服你不听，现在我松开了你又不走，你特么到底要干嘛？”
<br/>　　国王还在哭，楚涵无法，强行起身一拉被子，将其整个罩在里面。
<br/>　　“现在可以正常说话了吗？”他站在床边，道。
<br/>　　国王露出一个脑袋，红着眼睛点头。
<br/>　　楚涵叹口气，道：“我问你答。”
<br/>　　国王再次点头。
<br/>　　“你是国王？”楚涵真是没想到。
<br/>　　首先女人当国王太少见，而且也太年轻了，顶多二十岁。
<br/>　　这就是一小姑娘好吗！
<br/>　　国王这回开口说话了，声音带着抽泣：“我都当了天运国一百年的国王了，你竟然不认识我？”
<br/>　　咔！
<br/>　　楚涵差点扭到脖子：“一百年？”
<br/>　　什么鬼！
<br/>　　这女人几岁了？
<br/>　　“你几岁了？”楚涵震惊问道。
<br/>　　国王回答：“一百零五岁。”
<br/>　　楚涵在原地震惊了几秒，随后坦然接受：“行吧，管你几岁，我接着问。天运国的天女，是不是叫白允儿？”
<br/>　　这回换成国王震惊瞪大眼睛：“你怎么知道？”
<br/>　　天女有名字，这是天运国只有国王知道的秘密，其他知道的人，早已被国王暗地里处理掉。
<br/>　　包括国师在内，都不知道天女的名字。
<br/>　　楚涵露出果然如此的神情，最后嘴角一勾，道：“白允儿就是我的妻子之一，另一个星球的人。”
<br/>　　“啊？”国王呆住，目瞪口呆。
<br/>　　楚涵继续问：“白允儿什么时候来到天运国的？”
<br/>　　国王从震惊中回神：“二十年前吧，我被人追杀，她救了我。”
<br/>　　“二十年前？！”楚涵诧异的直接吼出来。
<br/>　　“国王陛下屋里有人！”这时候的屋外，一阵呼喊声响起。
<br/>　　楚涵一急，又钻进了国王的被子。
<br/>　　一群人又冲了进来，一个个面带煞气。
<br/>　　“国王陛下您还好吗？”
<br/>　　“我们听到了声音！”
<br/>　　国王面色一红，简直不知道如何形容现在的心情。
