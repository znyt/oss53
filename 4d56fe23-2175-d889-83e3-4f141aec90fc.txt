　　这一刻，洪荒失声。众多大能，诸多大神通者们，无论如何也不曾想到，仓颉竟有如此决心，如此毅力，如此勇气，以羸弱之躯，直面上苍，以文证道，证得文圣果位。
<br/>　　这种意志，这种气魄，堪称无敌！
<br/>　　天地间，诸圣心头震撼，面色十分凝重，数道宛若实质般的目光，落在仓颉身上，升起感叹之意。
<br/>　　“仓颉此子，真是好大的气魄！”通天教主敬佩起无上意志，赞叹出声。
<br/>　　“仓颉若是不死，洪荒之中，又要多出一种修炼体系！”太上道主目光毒辣，一下子想到了很多，人族日后，不可限量。
<br/>　　“仓颉开创文道，当为人文之祖，与武祖玄天并列！”女娲娘娘赞叹道。
<br/>　　“人族气运大增，不愧是天地主角！东方还真是人杰地灵！”似乎是想到了什么，接引道人暗叹一声，脸上愁苦更胜了。
<br/>　　“人族，真是令人羡慕，武祖玄天，再加上文圣仓颉，天皇伏羲，真是……”准提圣人无奈，脸上充满苦涩，心中更是为西方教的未来担忧不已。
<br/>　　“文祖仓颉，可于祖地开创文圣宫，与武帝宫并列！”
<br/>　　此时，祖地之中，传来武祖玄天的声音。显然，他认可了仓颉的地位。众多人族高手纷纷露出喜悦的心情，人族又添一名证道高手，气运大增。
<br/>　　“多谢人祖！”仓颉点头谢道。
<br/>　　“一文一武，一张一弛，文道，武道，不错！”连一向不苟言笑的元始圣主也开口赞叹道。
<br/>　　一众大神通者们也纷纷开口赞叹，除了祝贺仓颉证道之外，心中难免有些苦涩，连一个人族后辈都证道了，他们这些远古神话时代的大神们，还没有成功，难免对他们会有些打击。
<br/>　　一番祝贺自后，各自也就散了，这时，有熊氏公孙轩辕来到部落中，求见仓颉。
<br/>　　仓颉微微思索，便知道了轩辕来此的目的了，是想要邀请他对付蚩尤。可是，在他成圣的那一刻起，他便知道了圣人不得直接干涉人族事务，不然会影响人皇的证道，继而影响人族气运。圣人想要出手，只能通过其弟子等人出手。
<br/>　　于是，他开口对旁边之人说道：“你去告诉轩辕，就说我要去祖地开辟文圣宫，人皇之事，圣人不得出手干预，让他回去吧。”
<br/>　　“是，文祖！”
<br/>　　仓颉微微一叹，便不再多管，他在证道那一刻，通过天机得之后面的事情，不需要他操心。
<br/>　　外面，轩辕听了来人的汇报之后，无奈，只得失望而归。人族之事，还是得靠自己。两族大战，人族一统尽在眼前，他不得不操心，这关乎到他证道人皇之位。
<br/>　　昆仑山中，元始召来白鹤童子，对其说道：“去敲响金钟，召来你各位师兄弟，吾有要事欲告之！”
<br/>　　“是，老爷！”白鹤童子躬身退下。
<br/>　　不久，元始的各大弟子便来到玉虚宫大殿。
<br/>　　“拜见老师！”
<br/>　　众弟子看着上首愈发威严的元始，躬身拜道。
<br/>　　“嗯，今日为师召尔等前来，便是要通知各位，人皇争夺最终决战即将开启，尔等若是有心，即可前去帮助尔等师侄姬轩辕，助其得证人皇果位。”元始扫视了一眼众人，见众人的修为都没有落下，其中广成子与孔宣更是突破到了准圣，令他欣慰不已。他淡淡开口道。
<br/>　　“记住，我阐教不惧任何人，谁要欺负上来，那便揍回去！”
<br/>　　“是，老师！”众人听此，目光闪烁，不知道在想些什么，但表面上至少都对元始毕恭毕敬。
<br/>　　“嗯。”元始轻轻一声，便消失不见了。
<br/>　　一众弟子你看看我，我看看你，至少表面上和蔼可亲。众人对着广成子开口道：
<br/>　　“大师兄，我等愿去助一臂之力，若是有帮得上的忙，只管说。”
<br/>　　“多谢众位师弟！”
<br/>　　广成子自然是自无不可，虽然他们去了会分去不少人皇功德，但没办法啊，谁让蚩尤这个堪比祖巫的魔神太强大了，他手下的九黎魔神兵更是令人闻风丧胆。
<br/>　　天地间看重人皇功德的可不止阐教一家，天庭，西方教等一样对此虎视眈眈。天庭派下九天玄女等来相助轩辕，其中意味，谁都看得出来，西方更是想要派出弥勒，欲谋取一份功德，只不过是遭到了东方仙界的抵触，当初三教分封人皇之师时，便已经说清楚了，一致对外。
<br/>　　此次大战，汇聚了洪荒无数人的目光，其中，不只有人族，其余百族，甚至龙族，都有复出的迹象，如今人族内乱，谁也不希望人族一统，不然哪还有上古百族的地位。
<br/>　　远古龙、凤、麒麟三族大战，洪荒百族更是参与其中，最终落得个同归于尽的下场，各族底蕴耗尽，高手尽失，如今过去这么多年，才得以恢复元气，现在更是想要借助人族内战，夺取天地霸主之位。
<br/>　　这些由不得人族不重视，谁也不想在大战之时，背后跑出一群敌人前来偷袭。
<br/>　　……
<br/>　　人族，有熊氏部落。广成子带领阐教众人面见人皇轩辕。
<br/>　　“轩辕见过老师，见过诸位师叔。”
<br/>　　公孙轩辕看着眼前十几位形容各异，气度不凡的道人，恭敬道。这些人他当年在昆仑山便已经见过了，倒也不是很好奇。而且他也知道他的众位师伯们的能力，此次有求于人，因此对他们很是尊敬。
<br/>　　“嗯，轩辕不必多礼！”
<br/>　　广成子笑道。他对于轩辕这个徒弟很是满意，修为提升快速，如今已经是大罗金仙了，为人杀伐果断，而且对师门也很是尊敬。
<br/>　　“人皇不必多礼。”
<br/>　　孔宣等众阐教弟子也不敢无故受人皇如此大礼，人皇证道后地位等同于圣人，他们可不敢让人皇行礼，连忙把身体向旁侧一闪，躲避开来。
<br/>　　“轩辕。”广成子看向公孙轩辕，问道：“我不在这些日子，蚩尤那边如何？”
<br/>　　“哎——”公孙轩辕长叹一声，说道：“蚩尤那厮真是厉害，几次大战，我有熊部落损失不少。”
<br/>　　想到大战中损失不少人，公孙轩辕心中不由一暗，面上露出惋惜的神态。
<br/>　　“轩辕不必担忧，如今我阐教众仙已经齐到，自然不惧他魔神蚩尤。”广成子笑道。
<br/>　　“嗯。”公孙轩辕见此，心头也是送了一口气，说道：“一个月后我等决战于逐鹿，到时需要就众位师叔的帮助了。”
<br/>　　“我等自会助一笔之力，助人皇除了那蚩尤。”众人颔首道。
<br/>　　众仙齐聚，只待最终决战，人皇之争，亦是人族一统之日。
<br/>　　……
