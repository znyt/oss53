
范二哥表情僵硬，又有点凝重的看着王惊蛰的背影离开，书画初时没什么感觉，待身边男人半天都没有声音，她才扭过头就看见范二哥的额头上，都渗出了一点汗水。<br/>　　<br/>　　“你怎么了？”书画轻声问道。<br/>　　<br/>　　范二哥舔了舔嘴唇，思索了片刻，问道：“过几天你有时间么？”<br/>　　<br/>　　“没什么事，你也知道，我以后唯一要做的就是和你结婚，然后相夫教子就行了”书画叹了口气说道。<br/>　　<br/>　　“啪”范二哥笑着拍了拍她的手说道：“像我这种注定要成功的男人，后面是必须得稳定站着个可靠的女人，我仔细的找了找发现也就你能胜任了，所以你得耐得住寂寞受了，你的牺牲换回来的可能就是你男人的一飞冲天了”<br/>　　<br/>　　“喷嘴！”书画嫣然一笑，然后诧异的问道：“你刚才问我过几天要干什么？”<br/>　　<br/>　　“收拾打扮下，后过几天找那个低调的王惊蛰……嗯，吃个饭吧”范二哥幽幽的说道。<br/>　　<br/>　　书画瞬间就愣了愣，有些不解的问道：“这是从何说起的呢？”<br/>　　<br/>　　范二哥深吸了口气，缓缓的说道：“他刚才一句话，可能是把我从徘徊不定中给拽了出来，我得要找他解解惑啊”<br/>　　<br/>　　范二哥自打参加工作以来，履历都是很平坦的，最开始在街道办，没过两年往上调到了直属机关，再过几年就进了部委然后职位上也同样跟着变动，这是他们这种人在三十五岁以前正常的走向，但一过了三十五岁之后就不能在部委机关里面了，俗话说叫做外放，直接去某个区县任职，然后一步一个脚印的走上来，大概在五十岁左右的时候回调京城，呆几年再往出放的时候，脑袋上可能就该顶着封疆大吏的头衔了。<br/>　　<br/>　　范二哥这时就到了该往外放的时候，半个月前家中特意就此事慎重的考虑过，当时是范二哥的父亲，大伯还有他家老爷子一同参谋的，从众多候选地方中挑出了两个来，却始终没有定下到底该去哪个，这两处地方都是范家关系直属地区，可以护着他顺风顺水一直平步青云往上走的。<br/>　　<br/>　　有人可能会问了，二选一而已随便去哪不行，那不都是自己经营的地方么？<br/>　　<br/>　　其实这么说来是挺简单的，可往深说了的话确实不太好选，就比如去了这个地方能出不少的政绩，那履历就是嘎嘎光彩的，可万一去了另一个地方政绩没有，出了事故的话，那可能剩下的就是暗淡的前程了，所以这个外放对于范二哥来说那是慎重又慎重的，到过完年三月份他就必须得做出抉择，到底该要去哪里了。<br/>　　<br/>　　这两个地方选对了，他以后是后备梯队中的一个，选错了，可能早早的就出局了。<br/>　　<br/>　　但他万万没想到，今天被常文书偶然约出来的一次，居然被人给点出了他要去的两个地方，并且还把结果给他指了出来。<br/>　　<br/>　　就是王惊蛰在别人看来说的没头没脑的那句话，阳春三月别下扬州去往油菜花开满山的地方，一个扬州一个上饶，两个地方可难坏了范二哥。<br/>　　<br/>　　常文书和王惊蛰上到车里，他就问道：“你云里雾里跟二哥说的那句话，我虽然不明白是怎么回事，肯定也是大有深意了吧？”<br/>　　<br/>　　王惊蛰说道：“过几天他要是找你约我，你就说我出京城办事去了，得一阵才能回来”<br/>　　<br/>　　常文书皱眉说道：“这是怎么的呢，他主动送上门来，你还不要？”<br/>　　<br/>　　王惊蛰拍了下他的肩膀，说道：“这勾搭男人跟勾引女人都是一个道理，你若去泡妞，一泡就到手了，那肯定觉得没什么意思了，但你要约个三番五次的才把对方调出来的话，你绝对会上心的，并且到手了还得当个宝一样哄着，过程都一样但结果可不同”<br/>　　<br/>　　常文书顿时“哎呀”了一句，说道：“我是没看出来，你这也是个老中医啊？那以后说什么都得在这方面切磋切磋了，互相取取经！”<br/>　　<br/>　　王惊蛰矜持的说道：“一直都在花丛里走，始终未品尝过一败，在女人这方面来讲我对于自己的评价也就一个词能形容了……寂寞”<br/>　　<br/>　　“那是得要探讨一下了，你给我来点精髓，上上课”<br/>　　<br/>　　“看你领悟的能力吧……”<br/>　　<br/>　　王惊蛰说他过几天不在京城，一半是托词，另一半是他真有事要忙，主要就是想洞察一下东四环的那块地皮被胡胡他们给操持的怎么样了。<br/>　　<br/>　　两天过后，王惊蛰独自一人来到了那块荒地外，挑了个视线好的地方，注目观望，如今这块地里最近些天一直都挺忙碌的，自从地皮入手后玄门命字脉中人就已经开始布局了。<br/>　　<br/>　　那三十几栋别墅的门前，全都被竖起了两个一米半高的石狮子，左雄右雌算是一阴一阳，这是从里往外看的摆放状态，公狮脚下踩球母狮脚下放幼狮，这种风水神兽是用来压五行的，可以起到镇宅的效果。<br/>　　<br/>　　同时别墅内部也有人在测量，然后在加以改动，就是将原来是卧室的地方改成书房或者客厅之类的，厨房换成卫生间等等，虽然到最后感觉肯定有点怪怪的，但这种事开发商有一百种借口去解释，到最后肯定会让买房的人深信不疑。<br/>　　<br/>　　除此以外，地上的杂草已经全都给清理干净了，挖出了一个个的树坑，俗话讲冬天栽树夏天活，这是打算在入冬的时候就开始栽种了，但种的肯定不是一般的景观树，应该是鱼尾葵或者铁树，这两种树木用来挡煞，那都是绝佳的效果。<br/>　　<br/>　　王惊蛰转了一圈，心里就估算的**不离十了，差不多再有一个星期左右，这里就能完事了。
