　　第498章
<br/>　　程大雷窝在地牢里绝不出去，打定将牢底坐穿的信念，外人一概不见，每天的饭菜都是有高飞豹或张肥等人送进来。
<br/>　　程大雷如此谨慎，或者说胆小还是很少见的事情，足见他对聂隐娘的重视。程大雷如此重视，山寨诸人也不敢放松警惕，他们都明白这次碰见了棘手的敌人。
<br/>　　地牢外布下重兵警戒，里三层外三层的人保护程大雷安全。以地牢为中心，展开拉网式搜索，一寸地一寸地的搜过，整个蛤蟆城翻过一遍后，继续向外搜。那段时间的蛤蟆城风声鹤唳，草木皆兵，方圆十里之内经常有巡逻队出没，现在想要进出蛤蟆城要比以前更加麻烦。
<br/>　　以整个山寨的力量对付一个人，还是以前从未遇见过的事情。
<br/>　　日子一日又一日过去，屈指算来，时间已过去一个多月。
<br/>　　这一日刘悲来到地牢，向程大雷汇报最近发生的事情，看到程大雷的样子时，他表情有些无语。
<br/>　　只见程大雷穿着照月蛤蟆甲，手中无时无刻不握着匹夫剑，刘悲一点都不怀疑，程大雷睡觉时都这幅打扮。
<br/>　　“长安传来消息，李行哉已平安抵达落凤城，白元飞也在回来的路上。”
<br/>　　程大雷重视情报工作，如今帝国是二帝十八王的局面，程大雷命令阿喜培养人手，安插眼线在各诸侯王的封地中。如今这件工作才刚刚开始，还未完成，不过长安城这样的信息流通地，阿喜已布下了眼线。
<br/>　　“平安到了就好。”程大雷道：“路上遇到什么事没有？”
<br/>　　“据白元飞传来的消息，路上碰到几次暗杀，根据他的判断，当是将军府派出来的。最后李行哉假死脱身，平安到了封地。”
<br/>　　程大雷松了口气：“我便知道，李行哉还是很激灵的，不过，东海与凉州相距太远，以后我们能帮上的忙，怕就不多了。”
<br/>　　“另外一件事，现在各诸侯王都发了招贤令，招募江湖上的高手，吸收到自己手下。如今这件事，在江湖上闹得很大。”
<br/>　　“预料之中，江湖人也靠此谋个出路。”
<br/>　　“大当家，咱们要不要采取些行动，比如也建立一座招贤馆，招募一下江湖上的高手？”
<br/>　　程大雷白了他一眼，心道：我需要高手么，我纠结的是手上高手太多。他摆摆手：“下一件事？”
<br/>　　“喔，还有最后一件事，凉州宋伯康派人来了，想要面见大当家，说是有要事相商？”
<br/>　　“不见。”程大雷干脆道：“我现在什么情况你也不是不知道，能和他们见面嘛。对了，他们这次是有什么事？”
<br/>　　“由和珅与他们派过来的人谈过，但没能把他们的话套出来，喜字队正在调查，相信很快就有消息的。”
<br/>　　程大雷点点头，对于手下人的办事能力，他还是很放心的。
<br/>　　刘悲汇报完毕，目光看着程大雷，一副欲言又止的表情。
<br/>　　“怎么了？”程大雷问。
<br/>　　“大当家，您打算窝到什么时候，您是有一个月没换衣服了吧，都快馊了。”刘悲捂住鼻子。
<br/>　　“你以为我愿意这个样子么，但天大地大活着最大，我要跑出去，被人一剑杀了怎么办。”程大雷无奈道。
<br/>　　“这都过去这么长时间了，依我看，那刺客知道事不可为，怕已经走了。”
<br/>　　程大雷撇撇嘴：“说得容易，但这种事怎么能冒险。如这种杀手的性情，你是无法了解的。他们脑子都有些不清楚，不达目的，是绝不会罢休的。”
<br/>　　“可这样一直躲着，总也不是个办法，大当家一直关在这里，怕脑袋也快有些不清楚了。”刘悲道。
<br/>　　程大雷白了他一眼，道：“还有事没有，没有我要闭关悟剑了。”
<br/>　　刘悲开口，欲言又止，吞吞吐吐半天道：“有件事，我不知当讲不当讲，只是一件私事，其实也不太重要，不过对大当家而言，可能有些重要，但大当家现在的情况……”
<br/>　　“说！”
<br/>　　刘悲表情一僵：“万物教的圣女要走了。”
<br/>　　“什么情况，你们刚成亲就闹矛盾了，她要去哪儿，回娘家？”程大雷眼睛一亮，八卦之魂熊熊燃烧。
<br/>　　“不是原先的圣女，是现在的圣女，明玉公主她，她要走了。”
<br/>　　程大雷一怔：“这……是什么情况？”
<br/>　　“公主殿下的意思，是想去天下走走，至于究竟原因为何，我们也无法知道，但据属下观察，公主殿下的去意已决。”
<br/>　　“看看，当了圣女之后脑袋也有些不清楚了。”程大雷道：“把她叫回来，我当面和她谈谈，令她回心转意。”
<br/>　　“怕不太好办。”刘悲语气有些迟疑：“公主殿下今日就要走。”
<br/>　　程大雷愣了愣，腾地一下站起：“现在走到哪儿了？”
<br/>　　“该是刚出城。”
<br/>　　等刘悲说完，还没回过神来，眼前已不见了程大雷的踪影。程大雷大步流星，向地牢外走去，走到门口，他双臂推开牢门，一阵清风扑面。
<br/>　　程大雷顿了顿，下意识道：“原来已经夏天了。”
<br/>　　李莫愁守在牢外，她算是程大雷的第一道防御。今天算是她和程大雷第一次正式见面，看到程大雷时明显愣了愣。
<br/>　　窝在地牢内这么久，纵然衣食无忧，但毕竟还是有些颓废。李莫愁上三眼下三眼的打量程大雷，最后微微撇撇嘴：“原来这么丑呐。”
<br/>　　“走啦。”
<br/>　　程大雷身子忽然跃起，跨过城门，脚踩屋顶向城门方向而去。
<br/>　　这一手倒是令李莫愁眯起眼睛，却是没想到程大雷这蛤蟆城之主，竟有如此出色的轻功。
<br/>　　她脚尖点地，同样跃起，随在程大雷身后。
<br/>　　李婉儿的确刚出城不久，万物教的信徒一直送到离城五里外。
<br/>　　“不必送了，你们回去吧。”李婉儿向身后挥手。
<br/>　　“圣女殿下，让我们随你一起去吧。”万物教的信徒都舍不得与李婉儿分离。
<br/>　　“这是我自己的事情，诸位留在此地就好。”
<br/>　　李婉儿身着粗布麻衣，未施粉黛，手中一根竹杖，身边立着一头毛驴。
<br/>　　“喂，等一下。”
<br/>　　程大雷此刻大步流星赶到。
<br/>(本章完)
