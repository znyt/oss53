　　但是这天元宗，既然是能够拥有一方小世界，那绝不可能是默默无闻的势力才对。
<br/>　　毕竟如今的九域之中，除了七大宗门之外，根本没有几个势力有着自己单独的一方小世界。
<br/>　　北邙学院，也就是现在剑道门的这一方小世界，也是楚风眠的大师兄剑凌，独自一人摄取的。
<br/>　　不然以北邙学院曾经的势力，根本没有占据一方小世界的可能。
<br/>　　“天元宗，天元，天元，莫非是天元商会？”
<br/>　　楚风眠脑海之中不断的向着一个个曾经九域霸主级别势力的名字。
<br/>　　突然一个名字让楚风眠想了起来。
<br/>　　天元商会。
<br/>　　曾经十万年前，九域最大的商会，也几乎是无数时代之中，最为繁华庞大的一个商会。
<br/>　　现在九域之中虽然也有着无数像是，通天阁那般的商会，但是跟天元商会相比，却是相差甚远。
<br/>　　现在的通天阁，虽然势力遍布整个九域，但是却始终在七大宗门之下，而像是万器阁那样的势力，本身就是属于天剑宗的一部分。
<br/>　　可天元商会，在当时却是绝对的霸主，比起现在的七大宗门势力还要强大的多。
<br/>　　据说天元商会的崛起，是因为天元商会的一种独特圣丹，天元圣丹。
<br/>　　这天元圣丹，对帝尊境的武者，都有着极大的提升，一枚天元圣丹，足以造就出一位古帝来。
<br/>　　比起那始祖圣丹，甚至是还要神奇。
<br/>　　不过天元圣丹，也是昙花一现，唯有当初的天元商会拥有。
<br/>　　随着后来天元商会的没落，这天元圣丹也是就在九域之中彻底的消失不见了。
<br/>　　而至于九域之外，更是没有过任何的记载，所以知道的人并不多。
<br/>　　“如果真的是天元商会的遗迹，才可就发财了！”
<br/>　　楚风眠眼神之中露出了几分兴奋之色。
<br/>　　天元商会，也许不是历史之上最为强大的势力，但是绝对是最为富足的势力之一。
<br/>　　当时天元商会，作为九域的霸主，不知道是收敛了多少的财富。
<br/>　　这些财富，哪怕是千分之一，万分之一流传下来，放在现在都是一个天文数字。
<br/>　　而且如果可以得到几枚天元圣丹的话，楚风眠突破把握，将会大很多。
<br/>　　甚至可能得到天元圣丹的丹方的话，那楚风眠可才是真正的发财了。
<br/>　　不过天元圣丹的丹方，早就已经消失不见了，无人知道是哪里。
<br/>　　在这遗迹之中，得到的几率也微乎其微，楚风眠能够得到几枚天元圣丹，都算是无比幸运的了。
<br/>　　想到这里，楚风眠不由的加快了遁光。
<br/>　　炎撼世跟楚风眠一路飞行，一直是来到了这小世界接近中心的区域。
<br/>　　突然之间，被一道光芒拦住了。
<br/>　　一道七彩霞光，笼罩住的面前的一片区域，从七彩霞光之外，根本看不出这里面到底是什么样子。
<br/>　　楚风眠看得出这七彩霞光，应该是一道禁制，就跟那九玄仙尊洞府入口处的禁制极为相似。
<br/>　　应该是只有得到资格的人，才可以进入其中。
<br/>　　“手捏这块晶石，就可以进入其中。”
<br/>　　炎撼世在七彩霞光前停了下来，手心一动，一枚七彩晶石，落入到了他的手心之中，随即递给了楚风眠。
<br/>　　楚风眠接过这七彩晶石，观察了一下。
<br/>　　这七彩晶石跟上面七彩霞光的纹路，几乎是一模一样，看来应该是同属于一方势力。
<br/>　　这七彩晶石，应该就是得到这天元商会宝藏的钥匙。
<br/>　　“我回去照顾诸位师兄师弟。”
<br/>　　炎撼世将这七彩晶石交给了楚风眠之后，便是准备转头离开。
<br/>　　“等一下。”
<br/>　　突然间，楚风眠想到了什么，叫住了炎撼世。
<br/>　　“楚兄还有什么事？”
<br/>　　炎撼世听到楚风眠的话，还是停了下来，回头有些奇怪的询问道。
<br/>　　“告诉五灵宗的圣子，要小心沧海都，齐林山，五行天这三方势力的人。”
<br/>　　楚风眠目光冰冷的开口道。
<br/>　　“我从他们的口中听说，这三方势力，如今要联合对付圣龙秘境，碰到这三方势力的人，一定要小心。”
<br/>　　“这三方势力的人，很有可能，就会在九域动手，围攻圣龙秘境弟子。”
<br/>　　这个消息，楚风眠也是偶然运气不错得到的，只怕是圣龙秘境的其他人，都不知道。
<br/>　　五灵宗显然既然是跟楚风眠站在了一边，也是自己人，楚风眠可不想要五灵宗产生太大的损失。
<br/>　　楚风眠将这个消息告诉炎撼世，也是要让五灵宗做好防备。
<br/>　　“什么？楚兄你说的可是真的？”
<br/>　　在听完楚风眠的话后，炎撼世瞪大了眼睛，有些不可置信的说道。
<br/>　　在说完这句话，炎撼世才发现了话说的有些不对，急忙补充道。
<br/>　　“我不是在怀疑楚兄，只是这件事……”
<br/>　　“我知道这件事不好相信，但是我的确是从几名齐林山弟子的口中听到的，应该不假。”
<br/>　　楚风眠看到炎撼世的反应，也没有觉得有什么奇怪的。
<br/>　　大罗天星域的四大势力，数万年间，都没有起过什么大的冲突，一直以来都是相安无事。
<br/>　　在大罗天星域之内，保持了一个平衡。
<br/>　　现在楚风眠居然是说其他三大宗门，要来对付圣龙秘境，这件事的确有些难以相信。
<br/>　　不过楚风眠说了就说了，这五灵宗圣子们信不信他，就不是楚风眠可以控制的了。
<br/>　　“这件事，我会去告诉木麟师兄的。”
<br/>　　炎撼世看到楚风眠的反应，也明白，楚风眠绝不会是那种无的放矢之人，随即凝重的点了点头。
<br/>　　如果齐林山，沧海都，五行天这三方势力，真的是要一起对付圣龙秘境的话。
<br/>　　那对圣龙秘境来说，绝对是一个灾难。
<br/>　　这四方势力，如今明面上的关系，还是合作的关系。
<br/>　　如果在合作的时候，其他三方势力反水的话，那圣龙秘境的处境，已经不是危险可以形容的了。
<br/>　　而是一场灾难。
<br/>　　越想到这里，炎撼世额头上的汗珠就不断的滚落下去。
<br/>　　“多谢楚兄提醒。”
<br/>　　炎撼世对楚风眠道了个谢，便是急忙化为了一道遁光离开了。
<br/>　　显然是要早早准备，将这个消息去告诉五灵宗的其他圣子了。
