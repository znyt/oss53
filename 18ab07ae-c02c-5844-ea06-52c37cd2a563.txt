　　冰漓长老也想不到，楚风眠居然是如此之狠。
<br/>　　在这司空景胜面前，居然是将司空弘的剑都给夺走，还让司空弘直接跪倒在地上。
<br/>　　这种羞辱，如今丢的，可不止是司空弘一个人的脸面了，而是直接羞辱的是整个司空家的脸面。
<br/>　　遇到这种羞辱，司空家要是能忍，他们也不配成为武道家族了。
<br/>　　“不行！今天的事，不可能就这么轻易的化解！”
<br/>　　司空景胜霸道无比，如同一只咆哮的狮子，疯狂的吼道。
<br/>　　“这小子，必须要交给我们司空家处理！不给个交代，今日的事休想善终！”
<br/>　　“交代，给什么交代，这个废物跟我约战，被我击败，夺了他的剑，也不过是免得他再去招惹是非，免得在惹到不该惹的人，我留下一条性命，已经算是对你们司空家的大恩大德了，还想要楚某给什么交代！”
<br/>　　楚风眠傲然之间开口道。
<br/>　　“一个老家伙，倚老卖老，不过就是一个小小的司空家，在敢来招惹我，从此之后，司空家也就没有存在的必要了！”
<br/>　　“再不滚，我让你一起跪在这里，跟这废物作伴！”
<br/>　　“狂妄！”
<br/>　　司空景胜听到楚风眠的话，已经是完全无法忍受了。
<br/>　　只见他的一声灵力，全然暴起，生死境巅峰的实力，已经是全部展现了出来，无尽剑意，围绕在司空景胜的身边。
<br/>　　“小子，你刚刚磕头认错，老夫还可以念在你是一个小辈的份上，不跟你计较，不过如今既然你如此狂妄，那今日便是你的死期，就算是北邙学院的人，今天也保不住你！”
<br/>　　司空景胜大声的咆哮起来。
<br/>　　“是吗？老匹夫，我看你是寿元临近了，想要自己找死，不过以你的资质，一辈子都没有希望可能踏入生死台阶了，现在早死两年，也没事。”
<br/>　　楚风眠看着司空景胜，毫不客气的说道。
<br/>　　“一个老匹夫，都没有超脱生死，也敢在我的面前说大话！”
<br/>　　听到楚风眠的话，冰漓长老简直是脑袋一昏。
<br/>　　这话，实在是太狂妄了，尤其是司空景胜如今的确是寿元临近，谁也不敢在他面前，提这件事。
<br/>　　如今楚风眠的话，简直是触犯了司空景胜的霉头。
<br/>　　“楚风眠，还不速速道歉，司空兄毕竟是一个前辈，你若是道歉，这件事还能化解。”
<br/>　　冰漓长老急忙的喊道。
<br/>　　司空景胜对她有恩，就算是如今这司空景胜咄咄逼人，她如今也不好帮楚风眠，来对付这司空景胜。
<br/>　　“冰漓长老不用在意，不过是一个老匹夫罢了，我这就送他，跟那司空弘一样，去地上跪着。”
<br/>　　楚风眠口中，轻描淡写的开口道。
<br/>　　“这样一个不成器的老匹夫，我还没有放在眼中。”
<br/>　　“可恶！小畜生！拿命来，今日我要以你的血，来洗刷我们司空家的耻辱！”
<br/>　　司空景胜怒喝一声，突然之间猛然暴起，手中凝力成剑，直接便是冲着楚风眠斩了过来。
<br/>　　楚风眠一口一个老匹夫，令司空景胜如何可以忍受。
<br/>　　司空景胜这样的人物，不管是放在那里，都是受人敬仰的强者，什么时候被一个神力境的小子，如此看不起。
<br/>　　“以为得到几分奇遇，你就想要跟老夫抗衡？笑话，让你见识一下什么叫做真正的剑术！”
<br/>　　司空景胜的一剑斩出，似乎是天地都要微微变色，正片天地，都要被切割开来一样。
<br/>　　一剑出，天地裂。
<br/>　　“裂空剑法！这是司空家的裂空剑法！那是足以媲美天级武技的剑术，司空家的绝技！”
<br/>　　冰漓长老震撼到。
<br/>　　这裂空剑法，就算是她，如今晋升碎命之境，想要抵挡起来，都是麻烦无比。
<br/>　　在她没有晋升之前，这一剑，冰漓长老甚至都没有自信可以抵挡得住。
<br/>　　同为生死境巅峰，都无法抵挡的住的一剑，如今楚风眠只是一尊神力境武者，又如何抵挡？
<br/>　　“小子，死吧！”
<br/>　　裂空一剑，从天而降，瞬息向着楚风眠斩杀而去，要将楚风眠的身躯，给直接斩断。
<br/>　　这足以就连天空都撕裂的一剑，向着楚风眠斩杀过去。
<br/>　　这一剑，那是司空景胜含怒出手，威力没有丝毫留手，要将楚风眠，直接斩杀。
<br/>　　“就这剑术，也好意思在我的面前嚣张？算了，我就让你见识见识，什么叫做真正的剑术！”
<br/>　　楚风眠冷哼一声，他手持那司空大罗剑，猛然一剑落下。

<br/>　　一张大网，猛然笼罩过去。
<br/>　　这裂天一剑，被这一张剑气大网，直接给团团围困起来。
<br/>　　“没有任何的意义，就你这低等的剑术，也想要拦住这裂天一剑？给我破！”
<br/>　　司空景胜脸上露出几分狰狞笑容，冷哼一声。
<br/>　　这裂天一剑的力量，突然更甚，从这剑气大网中，挣扎的，几乎是随时都要脱困而出。
<br/>　　“死吧！”
<br/>　　轰隆！
<br/>　　裂天一剑跟这剑气大网，猛然碰撞在了一起。

<br/>　　“没有实力，还敢来嚣张？”
<br/>　　楚风眠一边开口，一边心神猛然一动。
<br/>　　这剑气大网，在楚风眠的心神之下，被彻底激发出来，其中的一道道剑气，突然变得锋利无比。

<br/>　　这裂天一剑，被团团包围，突然之间，其中上面，出现了无数裂痕。

<br/>　　这司空景胜的实力，若是踏上了生死台阶，达到碎命之境，楚风眠对付起来，倒是有些棘手。
<br/>　　但是一个生死境巅峰的剑修？
<br/>　　在剑道上，还没有任何人，可以超越的了楚风眠。
<br/>　　这司空景胜跟楚风眠比拼剑道，就算是他的剑道领悟在高深一百倍，都不是楚风眠的对手。
