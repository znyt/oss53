　　第681章
<br/>　　山野荒林之中，一队人跑得气喘吁吁，不时回头张望一眼，然后抬头继续赶路。
<br/>　　这一队人正是从凉州城逃脱的宋伯康一行，除了宋伯康的家小妻女外，还有凉州城的文武官员。
<br/>　　事实上，在逃出凉州城时，宋伯康并不是不想逃到金城、武威、浩甲三座城池中的某一座，重整旗鼓，把自己失去的土地夺回来。
<br/>　　原因是他怕，凉州城恰恰处于凉州和京州交界处，如果自己深入腹地，就有可能被程大雷的人马抓住。那么自己的下场，一定不会好……
<br/>　　于是，在重整旗鼓和逃生之间，宋伯康还是选择了后者。
<br/>　　天大地大，自己的性命要紧，若是丢掉性命，那便什么都没有了。
<br/>　　一队人踏上逃亡的路，这自然不会是一件令人愉悦的事情。一路跋山涉水，还要担心身后有追兵赶过来。
<br/>　　恐惧、饥饿、病痛……这些在折磨着宋伯康这群贵人。即便在凉州这样的苦寒之地，他们也是养尊处优，那里受过这种苦。所以一路上叫苦连天，时不时抱怨几句。
<br/>　　“王上，王上……”陆洛连唤几声，道：“属下实在走不动了，咱们歇一歇吧。”
<br/>　　宋伯康也是筋疲力尽，他四周望了望，道：“也好，原地歇一歇，等下继续赶路。”
<br/>　　众人听到这句话，有些直接倒地，可想而知他们一路上遭受了怎样的折磨。
<br/>　　宋伯康坐在一块大石头上，心情自然不如何好受。在绕过三水关后，他们现在已算是进入京州。前面的路还很长，还不知要受多少苦，方才能抵达长安。
<br/>　　一想到这点，他的心情就更加糟糕。
<br/>　　而回到长安，也并不表示一切事情就都结束了。自己毕竟丢了凉州，当今天子会放过自己，恐怕崔相都不会饶恕自己。
<br/>　　不过，这些事问题倒是不大，宋伯康早已想好说辞。完全可以将这次战败的责任推给其他人。说自己是如何如何英勇，敌人是如何如何狡猾，同伴是如何如何愚蠢。
<br/>　　他在成为凉州王之前，在长安城也混了许多时间，做这些事情还是蛮有经验的。
<br/>　　真正令他不愉快的，还是这次战败。
<br/>　　怎么稀里糊涂，就将凉州城丢了？本来自己有很大机会一统凉州的好不好……
<br/>　　一路吃了这么多苦头，堂堂凉州王被人赶得像丧家犬，他也不是没有想过，自己这一战为何会落败。
<br/>　　想来想去，总还是觉得自己并没有做错什么。谁能想到程大雷会夜袭凉州城，而且会攻破城门。城门被攻破自己不逃跑能做什么，难道伸着脖子让程大雷杀……
<br/>　　每一个选择都是有理由的，每一步都没有错，可为何一步步正确，却将自己推到今天这个结局？
<br/>　　这是宋伯康想不通的地方之一。
<br/>　　看着面前的同伴，他们都是凉州城的贵人，身后大都有世家在撑腰。平日人人锦衣玉食，但今天却唉声叹气，好无骨气。
<br/>　　宋伯康想，如果自己真的有错，便是错用了这些人。世家子弟，拥有更多的资源，到关键时刻却如此无用。
<br/>　　宋伯康心里突然冒出个想法：也许这样一个帝国，早就该结束了。
<br/>　　想这些时他并不知道，就在此地，不久前也有人这样想过。
<br/>　　前方山林中忽然有了动静，一群人宛若惊弓之鸟，人人露出慌张神色。
<br/>　　该不会来了什么豺狼虎豹吧，大家筋疲力尽的走到此处，难道就做了豺狼的食物。
<br/>　　幸运或者说不幸的是，来的并非是野兽，而是人。
<br/>　　一群凶神恶煞的大汉从密林中窜出，将宋伯康一行人围了个水泄不通，就像看到小白兔的大灰狼一般，露出桀桀笑容。
<br/>　　宋伯康总算还镇定，拱了拱手道：“诸位好汉，你们要钱要银子尽可取走，只是不要伤我们性命。”
<br/>　　一人骑着黑马分开人群走进来，与宋伯康的目光撞在一起。宋伯康脸色瞬间大变，苍白如纸再无半点血色。
<br/>　　“宋大人，好久不见。”
<br/>　　宋伯康两眼一黑，仰面倒地，在昏迷之前，口中吐出三个字：
<br/>　　“百里胜。”
<br/>　　……
<br/>　　今天的凉州城是百废待兴，但在一连串事情后，总算暂且稳住了目前的形势。
<br/>　　高力士与徐灵儿从落玉寨被接了回来，高力士开始着手处理凉州城的事物。目前，落玉寨的存在价值已不大，还不如直接放弃，发挥高力士的能力，让他帮忙处理一些事。
<br/>　　就连苏樱也从蛤蟆城接到凉州城，发挥她的能力，帮着做一些事。有意无意，程大雷已经将自己的重心放在凉州城，蛤蟆城毕竟太小，朔方城还未发展起来，相比之下，凉州城更适合作为主基地。
<br/>　　如此，苏樱倒是多了几分精气神，平时被当成金丝雀一般养着，对她来说的确有些无聊。
<br/>　　就连樊梨花都帮着关一些事情，可以说程大雷手下能用的人手都用上了。
<br/>　　在集中所有事情后，一件件杂乱无章的事情大多得到解决。
<br/>　　由和珅安抚城中居民，程大雷下了严令，凡是敢骚扰平民的一律严惩。而对那些趁机生事的作奸犯科之徒也绝不轻饶。城中每天都有治安队巡逻，如此严令之下，暂时稳住了平民的情绪。
<br/>　　之后，和珅便回了朔方城，那里也是有很多事情要处理，还要担心别人攻打朔方城。
<br/>　　刘悲处理军务，降兵放走一批，留下一批，大概有一万有余的贫民子弟愿意留下。
<br/>　　这些人刚刚归附，目前还不算程大雷的兵，将他们真正变成自己人，还需要一些时间。
<br/>　　屈指便是十几日时光，在过去的十几天内，程大雷忙得不可开交，到现在才终于可以喘口气。
<br/>　　如今，和珅管理朔方城，杨国忠管理蛤蟆城，程大雷刘悲等人坐镇凉州城。
<br/>　　而令程大雷奇怪的是，在过去的这十几天时间内，武威三座城一点动静也没有。
<br/>　　既没有兴兵攻打，也没有派人过来表示和平。
<br/>　　程大雷将兵力分散，秦蛮镇守蛤蟆城，关鱼镇守朔方城，剩余兵力聚集在凉州城。
<br/>　　到现在，如果他们三家想生事，程大雷也不需要太过担忧。
<br/>(本章完)
