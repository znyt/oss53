　　温馨的早餐过后，慕清霄就与小医仙离开小阁楼，踏上内院宽敞的道路上。
<br/>　　路上老生有许多，都在为了修习而努力，只要是路过的，都会瞧上两人几眼，特别是看到小医仙后，眼中不免都闪过惊艳之色。
<br/>　　道路上商店林立，与其说内院是一所学院核心，倒不如说是一座城市，只不过生活在这里的，都不是普通人。
<br/>　　走进玲琅满目的商店，小医仙恢复以往的活泼好动，这里看看，那里瞧瞧。
<br/>　　慕清霄则悠哉悠哉的跟在她身后，静静的望着她。
<br/>　　商店中，有着各种各样的灵药，只不过，越是高级的灵药，需要的火能就越多，往往四品以上灵药，根本就不是普通老生能负担起的。
<br/>　　小医仙微微摇头，她并不是炼药师，这些东西也只能养养眼。
<br/>　　不过，当她见到满意的东西后，居然还会特地的砍价，真跟逛街似得。
<br/>　　慕清霄跟在身后，看到管辖区域的商人，脸上那精彩的表情后，差点笑出声来。
<br/>　　半个小时后，小医仙拉着他的手，美眸弯成两轮月牙，开心的说道：“走吧，我们现在去哪里？”
<br/>　　“顺路去北边看看吧。”
<br/>　　慕清霄指了指北边方向，陨落心炎在那里，暂时封印厄难毒体的方法也在那里...
<br/>　　陨落心炎能提高他的实力，暂时封印厄难毒体，小医仙也能减轻许多负担。
<br/>　　虽然嘴上不曾提起，但慕清霄能看出来，她依旧在担心害怕，让自己的女人担心受怕，简直太不称职了。
<br/>　　“走吧。”
<br/>　　慕清霄牵起小医仙的手，脚尖轻点就落在屋顶上，随后以极快的速度，向着北边掠去，令得来往老生目露诧异。
<br/>　　...
<br/>　　...
<br/>　　内院面积大的有些出奇，由于身处内院，慕清霄也不好暴露太多，速度也压制在一般大斗师左右。
<br/>　　沿途中，倒是经常能够看到，一些老生互相切磋的战圈，在圈子外，一堆老生指指点点，议论纷纷。
<br/>　　半刻钟后，慕清霄与小医仙两人，轻轻落在一颗颇为高大的树木上，在两人的视线中，已经出现一片黑压压的学员。
<br/>　　学员们排序整齐，脸上都挂着不耐烦的神色，却一个都没有离开的意思。
<br/>　　而且，就在不远处的几颗大树上，皆是站着一些老生，这些老生神色坚毅，气息雄浑，最高的已经是四星斗灵。
<br/>　　“看来，我们似乎来早了一些。”
<br/>　　站在粗壮的树杆上，居高临下的俯视着周围的场景，视线跃过黑压压的人群，映入眼中的是一处凹陷的地形。
<br/>　　在凹陷的地形中，有一座极其庞大黑塔，深埋入地底，只在地面上，露出一截塔尖与一道黑色的入口。
<br/>　　哪怕是露在地面上的一截，都颇为高大。
<br/>　　这种极为奇异的场景，使得慕清霄眼中都闪过一丝惊异，道：“塔埋地底，这就是传说中的天焚炼气塔......”
<br/>　　目光在四周扫视一圈，自天焚炼气塔的四周，隐隐有些扭曲的皱褶。
<br/>　　皱褶不是很明显，但修炼至高心法的他，感应与眼力是何等的敏锐，哪怕有一丝皱褶，都会感到违和感。
<br/>　　没想到，天焚炼气塔的周围，居然布置下了空间结界，只不过，结界似乎在渐渐衰退。
<br/>　　转过头来，看着小医仙，慕清霄道：“在这里面修炼，估计对压制厄难毒体有些帮助，我们也进去试试吧。”
<br/>　　说罢，再次拉起玉手，身形一闪就已经混进人群中，最后跟着涌动的人群，来到了距离天焚炼气塔最近的地方。
<br/>　　此时，两人的前面已经是空荡荡的一片，而在空地对面十几米处，便是深埋地底的神秘黑塔。
<br/>　　在宽敞的地面上，紧致有序的划分成了好几个区域，每个区域中，都是盘坐着一群闭目的身影，其中有少数几位也是斗灵级别。
<br/>　　小医仙看了一眼盘坐在空旷地区的身影，道：“他们的气息都好强，而且胸口左上方都佩戴着不同的徽章，估计是内院几大势力的人吧。”
<br/>　　慕清霄收回目光，道：“他们也只是暂时比你强，仙儿，你的天赋不错，相信很快就能超越他们。”
<br/>　　听到他的夸赞，小医仙心里喜滋滋的，微微颔首。
<br/>　　“哐当！”
<br/>　　等待片刻后，周围空间忽然响起一阵古老的钟声，钟声回荡在空中，而随着钟声的响起，周围的喧闹声，也戛然而止。
<br/>　　“开塔！”
<br/>　　待钟声消散，一道苍老的声音似乎自塔内传出。
<br/>　　苍老的声音刚刚落下，只见天焚炼气塔上的黑色大门，伴随着一阵嘎吱作响声，缓缓开启。
<br/>　　随后，一股炽热的气息渗透而出，使得周围空间的温度在一瞬间提高不少。
<br/>　　黑门开启，随着炽热的气息涌出，慕清霄双眼一眯，气息更加明显了，这种恐怖的温度，奇妙的感觉，也只有异火了。
<br/>　　“进塔！”
<br/>　　苍老的声音，再度响起。
<br/>　　随后，盘坐在空旷地区的人，几乎是同时睁开双眼，豁然起身，伴随着一阵破风声，那一道道身影，铺天盖地的向塔门内暴射而去。
<br/>　　“我们也进去瞧瞧吧。”
<br/>　　说着，慕清霄就带着小医仙，身影一闪，凭空消失在原地，眨眼间已经出现在了塔门口。
<br/>　　站在塔门口，慕清霄不免有些惊奇，这黑塔露在外面的一小截，虽只是冰山一角，却相当于地球上的三层楼那么高。
<br/>　　难以想象，隐藏在地底的塔身，究竟是何等的磅礴！
<br/>　　接近塔身后，慕清霄在厚重的塔身上感受到了一股冰寒的气息，黑塔估计是某种器物，就是不知道，是用何种材料炼制的。
<br/>　　前脚刚踏进黑塔，慕清霄就隐隐感觉，心跳开始加速。
<br/>　　眼眸深处，闪过一丝黑色火焰，嘴角牵起一丝若有若无的笑意，九霄琉璃焱在兴奋。
<br/>　　这种感觉，就像是小孩童，在家中寻找到了美味的糖果，有些迫不及待的想要一口吞下去。
<br/>　　压制住蠢蠢欲动的神火，慕清霄看了一眼身旁的小医仙。
<br/>　　此时，这小妮子柔美的脸颊，已经变得赤红，犹如火炭一般，并且有一股淡淡的灰色气息，自她的体内渗透而出。
<br/>　　毒气！
