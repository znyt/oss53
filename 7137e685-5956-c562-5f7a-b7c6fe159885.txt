　　阴森煞气，无边血海。
<br/>　　血海之中，元始在大肆搜寻。但这血海除了无边阴煞之气外，也就只剩下了这血气腥腻的众生鲜血了。他找了大半年时间，也什么都没发现，这还是他一寸一寸地搜寻，什么都没放过的情况之下。
<br/>　　这一天，元始在血海深处看到了一座巨大的红色莲花，莲台绽放，散发着黑色业火，灼烧着这无边的煞气。莲台之上，一个身穿血袍，满头赤发的英俊青年盘腿坐下，他正在修炼。
<br/>　　在青年的头顶庆云之中，一柄蓝色小旗摇摇摆摆，散发着无量玄光，护住青年元神。在他的身边，两柄黑色杀剑，散发着无边杀气，一柄上刻着“元屠”二字，另一柄上刻有“阿鼻”二字。
<br/>　　显然，这个英俊的青年便是冥河了。
<br/>　　看了一眼冥河，元始面上并没有什么变动，只是随意扫了一眼，便继续向着血海深处潜去。毕竟，早在来之前，他就知道此行定会遇到冥河，现在看到了，也没什么奇怪的。
<br/>　　百万里血海压力无边，但这点压力对于元始来说，却可以无视。百万里听着很多，但其实并没有多少，但其深度却是很深，元始这半年不仅知识搜寻血海表面，他甚至连血海底都到过，但并没有发现什么。
<br/>　　这一天，元始感应到周边有一种奇怪的引力，这种引力无缘无故出现，却好像是在指引这什么。元始顿时心中有了一丝异样，朝着引力方向赶去。
<br/>　　不久，元始来到一个血海海眼之处，只见海眼漆黑无比，仿佛一个巨大的妖兽张开血盆大口，等待着某人进入，里面阴暗森森，死气弥漫。
<br/>　　海眼周围布满了先天禁制、大阵，危机四伏，准圣来了，都不一定能够安全出入，若是大罗金仙到此，一个杀机，便能够让他灰飞烟灭。
<br/>　　元始无畏，一步踏出，却是避开各种禁阵，跳进了血海之眼。
<br/>　　血海海眼很深，穿透海底，也不知多少里，越来越黑，幽暗之中传来阵阵呜呜声，似乎那太古的怨魂在哭泣。
<br/>　　到了后来，各种虚影浮现，凄厉的大叫着，张牙舞爪，在黑暗的海眼之中长啸，犹如森罗地狱。
<br/>　　终于，不知降了多少里，元始坠落了下来，平稳落地。
<br/>　　洞底有些光亮，绿油油，像是鬼魂的眼睛，下面的世界阴森森的，一条隧道横向通向远方，元始一路向前，感觉阵阵幽森。
<br/>　　一直不知走了多远，不久，元始停了下来。因为在他眼前出现了一条河流，一条流淌着昏黄污浊的水的河流。
<br/>　　这便是传说中的黄泉，传说可以腐蚀灵魂的黄泉水！
<br/>　　元始试了一下，这东西对他只有一丁点的影响，几乎可以忽略不计。但，他还是收了一些。这东西虽对他没有影响，但，并不代表着对其他人没有影响啊。要知道，他的元神修为已经达到《神元九转》第六转圆满了，马上就可以突破第七转了，元神堪比混元强者。而其余人却不一定，因为并不是每个人的灵魂都与他一般。
<br/>　　这黄泉水对他没有什么效果，可低于混元大罗金仙的其余修炼者若是沾上，元神定会受损，这可谓是一种毒药。
<br/>　　况且，这黄泉圣水虽然不是十大真水之一，但其功效却又并不比十大真水低。是个不可多得的好东西。
<br/>　　若是他用这种黄泉水炼制灵宝，说不定可以伤人灵魂。这可是阴人必备的灵器啊。
<br/>　　收集一些黄泉圣水，元始再次踏步向前，不久，他便再次停下，因为，他又看到了一个了不得的东西。
<br/>　　在他的面前，一条若隐若现的惨白色石拱桥横贯于此，大桥之上，“奈何桥”三个潦草大字横立于空。
<br/>　　这是传说中鬼魂所要经过的奈何桥。
<br/>　　奈何桥上，白森森一片，无数阴风混合这血气在桥边吹过，死气弥漫，阴森恐怖。
<br/>　　元始若有所思，眼前的奈何桥还没有形成实质，恐怕是那幽冥界还没有彻底完善的原因。
<br/>　　默默感受一番这奈何桥上的规则，元始便不再停留，沿着奈何桥，继续向前。
<br/>　　不久，元始再次停下，因为他的眼前出现了一个黄色高台，高台回望过去，高高在上，横立路边。
<br/>　　望乡台！
<br/>　　看见这高台的第一眼，元始心中便已知道了这座高台的名字。这座高台名为望乡台。也是，在幽冥界之中，也只有望乡台了。传说人死之后，最后一眼回望故乡，斩断过去，迎接未来。
<br/>　　只是，眼前的望乡台依旧如同先前的奈何桥一般，若隐若现，还没有凝聚成实质。
<br/>　　伫立良久，看过一眼望乡台，元始便继续向前走去。
<br/>　　不久，他又停了下来，因为他眼前再次出现了一条河，河水漆黑无比，死气滚滚，阴暗漆森。
<br/>　　忘川河！
<br/>　　看过忘川河后，元始便再次上路了。这一次，他却是没有再停下来了。一直到了尽头，无处可走。
<br/>　　到了路的尽头，元始发现，在他面前有一个十几丈高的石头，石头之上刻有三道神秘神纹。
<br/>　　“难道这就是三生石？”元始若有所思道。
<br/>　　在他话音刚落之时，只见那石头之上，闪烁一下便显现出“三生石”三个大字，之后，再次消失不见。
<br/>　　“还真是三生石！”元始惊讶道。
<br/>　　“那么，六道轮回又在何处？”元始看了看四周，发现此地并没有什么六道轮回。他并不甘心，继续搜寻。
<br/>　　“轮回，是世界大道的重要组成部分之一，一方天地的根本所在，极为重要！
<br/>　　天地人三才大道，轮回便是这其中的地道，可与统揽日月，总理阴阳的天道，亿万苍生汇聚的人道比肩，轮回的力量可见一斑！
<br/>　　而，像洪荒这般顶尖的大千世界，要开辟出轮回，至少需要有混元后期的修为的牺牲才可以，而祖巫后土，根本连混元都不是，区区一个准圣，又怎么能够身化轮回，执掌地道，与鸿钧合天道相比？”元始眼瞳泛出紫金光芒，看向幽冥深处，沉声道。
<br/>　　“所以，定是此处早已有了轮回，而，后土身上也一定有某种特殊之处，她只是一个契机，一个能够让轮回现世的契机。”元始猜测道。
<br/>　　“那么，真正的轮回，如今又藏在何处？”
