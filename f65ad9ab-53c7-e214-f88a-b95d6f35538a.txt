　　罗昊跟舒磊和韩梅梅客气了几句，看向云岳，他知道云岳突然答应让舒磊和韩梅梅同车一起去安全屋，肯定是有意为之。
<br/>　　云岳接受到罗昊疑惑的眼神，微微一笑，说道：“这件事情还是让两位国安九局的同志来回答，更加合适。”
<br/>　　“情况是这样的。”
<br/>　　韩梅梅向罗昊解释道：“‘crystal-ball’事情被曝光后，我们发现除了在杭山市，我们国内还有几个城市也有‘crystal-ball’成瘾者，其中东海市就是其中之一。因为这件事情跟‘毒皇后’玛丽科娃有关系，上级要求我们一定要严查，争取找到这些城市中是谁在偷偷出售‘crystal-ball’，他们又跟‘毒皇后’有什么关系。”
<br/>　　舒磊从自己西装口袋中拿出一张照片递到罗昊面前，说道：“照片上的男人叫王凯源，我们怀疑他就是‘毒皇后’在东海市的代理人。”
<br/>　　“王凯源，王凯源。”罗昊手中拿着照片，看着照片穿着花衬衫，脖子上挂着一根金链子，戴着墨镜，一副社会混混样子的王凯源，抬头看向舒磊，问道：“难道这个王凯源跟刚才那个特警支队副支队长王荣浩有什么联系？”
<br/>　　“聪明！”舒磊打了一个响指，说道：“这个王凯源是王荣浩的远方表亲。罗队长就根据一张照片和一个名字就能猜出其中的关键，你如果来做我们情报这行，也绝对会是传说级高手。”
<br/>　　“哎哎哎，你这是当着我的面挖我的墙角，信不信我去第九局找你们老大去？”云岳对着舒磊说道。
<br/>　　“哪能啊，云参谋长。”舒磊笑着说道：“你这个墙角我们就是把手指给刨断了也挖不动啊，您的乘龙快婿有谁能挖的动？”
<br/>　　“这还差不多。”云岳对着舒磊警告道：“要不然我肯定跑去老耿那边打你们的小报告，我这个人可是很小心眼的。”
<br/>　　罗昊把照片还给舒磊问道：“那你们是怀疑王荣浩是王凯源背后的保护伞？还是说王荣浩跟‘毒皇后’有直接的联系？”
<br/>　　舒磊收起照片，轻轻摇了摇头，回答道：“我们查了王荣浩的经济状况，包括他所有银行的账户和资产情况，并没有任何可疑之处，没有任何直接证据证明王荣浩跟‘毒皇后’有关系！不过，我们通过对王凯源的监控，发现他跟王荣浩之间的联系比较频繁，一个贩毒的跟一个特警支队副支队长有密切联系，这就很可疑，所有我们有理由怀疑王荣浩即便没有涉足王凯源干的那些事情，也肯定知道王凯源干的好事。”
<br/>　　“王荣浩这个人很有野心，现任支队长钱伟民很快就要被调入省厅，特警支队支队长的这个位置就会被空出来，王荣浩盯着这个位置有很长时间，所以他绝对不会允许自己有任何违法行为，即便他跟王凯源之间的联系被曝光，他也最多就是个知情不报，大不了就是降职记过反省，而不会被送入大牢。”
<br/>　　“王荣浩可能跟‘毒皇后’没有关系，但是王凯源百分之百跟‘毒皇后’有联系。”
<br/>　　云岳这个时候接过话，继续说道：“无常后面会出境并且会试着跟‘毒皇后’的人接触，以雇佣者的身份进入‘黑蛇’监狱接近Derrick，‘毒皇后’这个女人警戒心很高，她不会把自己不信任的人派出去接近Derrick，从Derrick身上得到‘crystal-ball’的配方，如果无常要进入‘黑蛇’监狱就必须要先得到‘毒皇后’的信赖，尽管我们做了假的资料证明无常已经叛变，可是‘毒皇后’这么多年来这样的手段屡见不鲜，是不会轻易相信的，以她的谨慎，一定会让自己还算信得过的人进行调查……”
<br/>　　“所以你才会把行动初始地点放在东海市。”罗昊这个时候彻底明白了过来，说道：“你就是打算利用王凯源这个点，向‘毒皇后’传递无常已经‘叛变’的事实，而王凯源唯一能依靠的就是自己这个做特警支队副支队长的远房表亲，我们刚才这么一套戏其实就是做给王荣浩看的，为的就是给无常做证明！”
<br/>　　“你说得不错！”云岳对罗昊说道：“反应挺快啊。”
<br/>　　“暗影，你这个实在是太烧脑了。”罗昊摇头道：“这么多的弯弯绕，想的我脑瓜疼。”
<br/>　　“如果没有这么多弯弯绕，又怎么能骗过‘毒皇后’这个女人？”云岳说道：“这个女人实在太狡猾，太谨慎了，不然的话，国际刑警组织打了她那么多次念头，她还能活得这么潇洒！我们的机会只有这一次，如果错失这次机会，再想要从‘毒皇后’身上打开突破口基本就不可能了。”
<br/>　　“可是‘毒皇后’如果还是没有相信无常，那该怎么办？”罗昊对云岳问道。
<br/>　　“那么无常会死。”云岳神色沉重的回答道：“如果无常不能取得‘毒皇后’的信任，那么他就一定会死，‘毒皇后’是绝对不会留下活口的。这也是我们犹豫了很长时间要不要执行‘追魂’行动的原因，即便是我们把准备做到了最充足，还是会存在风险点，而这些风险点又全都是我们所不可控的，所以野狼才会在行动开始前，前来征求无常的意见，如果无常摇头拒绝这次行动，我们就会取消这次行动。”
<br/>　　“这次无常会在没有支援的情况下，独自进行行动，在他没有找到‘毒皇后’老巢之前，你们只能待在国内，当做什么事情都没有发生，进行正常训练。”
<br/>　　“可是佟千雪那边要怎么办？”罗昊对云岳问道：“佟千雪已经失去了孩子，如果这次无常真的发生什么意外，佟千雪肯定会疯掉的。”
<br/>　　“佟千雪现在所面临的状况，就跟小熙当时面临的状况相似。”云岳说道：“我知道这样很残忍，但是有些事情总是要有人去牺牲，无常做出了他的选择。我会把千雪接到安全屋来，然后跟她说明情况，在她得知情况后，她会被送到龙穴。”
