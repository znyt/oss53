　　第25章
<br/>　　“大当家，咱们在做什么？”
<br/>　　黄三元不解的问，他今天很早就被程大雷叫起来，陪着他围着蛤蟆岭转圈。
<br/>　　“看山！”程大雷很装X的吐出两个字。
<br/>　　“看……什么山？”黄三元更加困惑。
<br/>　　程大雷没有回答这个问题，打量着面前的巨石，想要爬上去。
<br/>　　“黄管家，你见多识广，如果咱们山寨想要扩充人口，你觉得可以从哪里招募？”
<br/>　　“这个估计很难。”
<br/>　　“怎么个难法？说来听听。”程大雷试了试，还是没办法爬上大石。
<br/>　　“帝国的子民一般可以分为四等，这个说法大当家有没有听过？”
<br/>　　程大雷顿了顿：“什么四等，怎么分的？”
<br/>　　“第一等自然就是王族，大武李氏立国一百二十年，享有无上权威，他们不必劳作，由帝国供奉；第二等是贵族，这一等级包括世家，城主，富商；第三等是平民，包括地主，乡绅，商户；第四等是贱民，包括农户，流民，家奴等等。”
<br/>　　“那山贼属于哪一等？”程大雷不解的问。
<br/>　　黄三元有些为难：“山贼没有等级，也可以说他们排在最低等，帝国根本不承认山贼是帝国的子民，比流民，奴隶，乞丐还要卑贱。普通人如果走投无路，有些宁愿去死也不愿做山贼。如果一个家庭里出了山贼，那将是整个家族的耻辱，会让祖宗蒙羞的。”
<br/>　　说到这里，黄三元瞅了瞅程大雷，他现在摸不准程大雷脾气，还生怕这些话触怒了他。
<br/>　　程大雷并没有生气，他终于爬上那块巨石，站在最高处居高临下俯视整个蛤蟆寨。
<br/>　　“山贼，原来是这么卑贱的存在呐！”
<br/>　　程大雷忽然笑了笑，道：“黄管家，我准备在蛤蟆寨外围布一道围墙，加固一下原由的防御设施。”
<br/>　　“这样好是好，可我们没有人手啊。”
<br/>　　“这个问题我已经考虑过了，而且我已经想到了。”程大雷丢给黄三元一份图纸：“看我设计的防御设施怎样？”
<br/>　　黄三元缓缓将图纸拉开，眼睛瞬间睁大：“这都是大当家亲自设计的？”
<br/>　　“嗯呐。”程大雷毫无心理障碍的承认这件事。
<br/>　　不怪黄三元如此震惊，在这个识字率不到10%的时代，程大雷能拿出这样一份图纸，简直就可以被视为天才。
<br/>　　他望向程大雷的目光竟有几分崇拜，只见程大雷站在巨石上，衣襟飘飘真有几分高深莫测的感觉。
<br/>　　程大雷正打量着蛤蟆寨的地形，从这里望过去蛤蟆岭像一只蹲在地上的癞蛤蟆。蛤蟆头冲着南方，其下是深不可测的悬崖，两侧是崇山峻岭，唯一上山的道路是北方一片长满参天大树的山坡，蛤蟆寨就位于蛤蟆脑袋上一块平坦地势。
<br/>　　程大雷要做的就是加固山寨原有的防御，再利用山石大木补足空隙处，最后再在山寨外围布上陷阱。
<br/>　　“大当家，可按你的计划我们的人手远远不够。”黄三元举着图纸道。
<br/>　　“我已经想好办法了。”程大雷从大石上跃下，道：“没有人，我们就从别的山寨挖人。”
<br/>　　说着，程大雷在黄三元耳边低语了几句。黄三元听着听着，眼睛睁大：“大当家，这样不好吧。”
<br/>　　“有什么不好的，按我们商量好的办，实在不行山寨还剩些火药呢。”
<br/>　　黄三元走后，程大雷继续在山寨转，今天还是他第一次观察整个蛤蟆寨的建筑布局。老实说，没什么布局，看来原先建设这座山寨的人一定是想到哪里建哪里，缺什么建什么。
<br/>　　而这些都将在这次改造计划中全部推翻重建。
<br/>　　嘟，探索蛤蟆寨成就达成。
<br/>　　随着脑海中的提示消息，一副地图在程大雷脑海中缓缓打开。蛤蟆寨的每一处建筑都出现在地图上，无比清晰。
<br/>　　程大雷心中震惊不已，这是不是表示，以后自己探索任何地方，就能获得那里的地图。探索青牛山，就能获得青牛山的地图，探索十三州，就能获得整个帝国的地图。
<br/>　　在这个时代，地图无比重要，拥有一份精确的地图就能知道该在何地进攻，何地防守。要知道，当年荆轲刺秦王，就是用一份地图表达诚意，才换的面见秦王的机会。如此，足见地图的重要性。
<br/>　　……
<br/>　　“好消息，好消息！”
<br/>　　哐！铛！
<br/>　　“好消息，好消息！”
<br/>　　哐！铛！
<br/>　　飞虎寨，高飞虎正在大树下纳凉，这声音忽然从山下传过来。高飞虎激灵一下坐起，有上次的经历，他脑海中已有不好的预感。
<br/>　　果然，当走出山寨的时候，他看到山寨附近的大树上又一次贴满白纸……是叫宣传单什么的嘛。此刻一堆人围在大树下，盯着上面的字。
<br/>　　“大当家，快给我们念念，上面写的是什么。”
<br/>　　高飞虎已经有不好的预感，根本不想看上面的内容，更不想念给其他人听。
<br/>　　“来，大家不要急，我念给大家。”
<br/>　　这是谁呀，这么不长眼。高飞虎扭过头来，看到自己弟弟大摇大摆的走过来，一脸得意。
<br/>　　“好消息，好消息！”
<br/>　　高飞豹清了清嗓子，大声朗读着上面的内容。
<br/>　　“少侠，你还在每天忍受野菜杂粮嘛！在这里你每天都可以吃到白面烙饼。
<br/>　　你还在为没有兵器而困扰么！这里有精致的兵器供你选择。
<br/>　　你还在为自己的未来而迷茫嘛！这里给你一份锦绣前程。
<br/>　　好消息，好消息！
<br/>　　蛤蟆寨正在招贤纳才！
<br/>　　梦想从这里起航，人生在这里被重新决定！
<br/>　　我们的宗旨是：骑最烈的马，睡最美的女人。
<br/>　　加入蛤蟆寨，一切不是梦想！”
<br/>　　在白纸的最下面，还画着一副人像，肩膀上扛着一只大斧头，一只手伸出，食指指着下面四个血红的朱砂大字。
<br/>　　我！需！要！你！
<br/>　　高飞虎脸上的肌肉剧烈颤抖，一股不可遏制的怒火在胸腔中泛滥。
<br/>　　“骑最烈的马，睡最美的女人。”高飞豹啧啧称赞：“这句话写的真好，书法也不错。”
<br/>　　“混蛋！”高飞豹气得扑上去把上面的内容撕得稀巴烂，愤怒的发出一声咆哮：“我从来没见过这么厚颜无耻之人。”
<br/>　　
<br/>(本章完)
