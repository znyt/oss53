　　“他对我来说，是一位亦师亦友的长辈，他求我帮他一个忙，让我来寻找他的后人，帮他证道，之后带他去太阴山，斩杀无极鬼王……我也是为了帮助他完成愿望，后来才逃出太阴山，一路被人追赶……最后才认识了老大。”
<br/>　　大伙听他说到这里，惊骇不已。谁也没有想到，原来瓜瓜身上还背负着这么大的秘密……
<br/>　　“你不是……因为那个女明星养小鬼的事情，跟少阳意外认识的吗？”谢雨晴惊讶地问道。
<br/>　　“那这是我用的一个手段啦。我不能太明显地接触老大，不然会让他怀疑的。”
<br/>　　“那你为什么不直接跟他说？”
<br/>　　瓜瓜叹了口气，道：“一开始吧，我是听从叶天师……嗯，叶法善的嘱咐，先观察老大的情况，叶法善跟我说过，让我找老大，先不要跟他说这些，免得给他压力，让他自己成长起来，拥有足够强大的实力之后，再去太阴山，不然去了也是送死，所以我一直就在老大身边呆着，我也没想到，老大后来成了我的老大，我十分犹豫，不想告诉他真相了……”
<br/>　　老郭回忆，说道：“他怕他去送死？”
<br/>　　瓜瓜点点头。“无极鬼王，连斩三尸的叶法善都对付不了，更不要说他手下还有那么多的强者，老大虽然在人间无敌，不过就连右君都打不过的，去了绝对是送死。但如果我跟他说了这件事，以他的性格，将来说不定真的会去太阴山，所以，我把这个秘密瞒了下来，今天也就是告诉你们，你们可千万别说出去。”
<br/>　　谢雨晴赞道：“你做的对，我才不管什么太阴山太阳山的，少阳是人，他绝不能去送死。”
<br/>　　其余在场的人也纷纷点头，虽然他们都是听不怕死的，不过太阴山那个传说中的地方……从来没有一个活口能从那里面逃脱，他们当然也不想叶少阳去送死。
<br/>　　老郭沉吟了一下，说道：“等一下啊，那个，二当家，叶法善让你找他子孙去太阴山搞事，为什么一定要是他的子孙呢，他就没想过，他子孙如果是个普通人呢？”
<br/>　　瓜瓜笑道：“你当我傻吗，我认识叶法善很久了，我为什么早不出来，偏偏等到老大成年再来，叶法善说，他当初在轮回道中留下一滴骨血，历经轮回，重塑肉身，投胎的就是老大，叶法善算准了他是先天灵体，也是应天劫的人选，资质冠绝天下。
<br/>　　换句话说，老大能有这修道的超级天赋，其实是叶法善早就安排好的，不过老大能有今天，不光是因为这个，他个人的性格也有很大原因么，这个就不是叶法善所能掌控的了，只能说是天命所归，更是证明了老大其实就是应劫之人。叶法善没有找错。”
<br/>　　叶法善是两千年前的人，居然能为数千年之后的子孙埋下伏笔，做好一些安排，并且算出了叶少阳的生辰，让瓜瓜这个时候来找他……不愧是斩三尸的大罗金仙。老郭等人想到这一点，心中也是无限感慨。
<br/>　　老郭想了想道：“你逃出鬼域之后，太阴山派了很多人来追杀你，是不是就是为了这件事？”
<br/>　　瓜瓜点头道：“就是这个，无极鬼王生怕老大真的有一天会去杀了他，所以不愿意让我留在老大身边，这才三番五次派人来杀我，不过后来战事一开，他也顾及不上我了，而且他肯定以为我已经对老大说了真相，把我抓回去也没用了，所以现在不管我了。”
<br/>　　老郭道：“也就是说，你知道怎么进太阴山？”
<br/>　　瓜瓜摇头。“太阴山只能出不能进，但是叶法善说，只要老大能到浮屠海，他有办法让老大感知到太阴山的位置，然后用几件东西，就可以有办法进去……”
<br/>　　“什么办法？”
<br/>　　“郭大侠，你就别问了，你知道这个完全没用，对自己没好处的。”
<br/>　　老郭一想也是，干脆也不问了。
<br/>　　谢雨晴接过话头，问道：“那这里有个问题，既然无极鬼王觉得少阳是个隐患，为什么不直接杀了他，这样不是更加容易，以绝后患？”
<br/>　　瓜瓜摇了摇头，道：“无极鬼王不能杀老大。”
<br/>　　“为什么？”不止一个人提出这个问题。
<br/>　　瓜瓜微微低头，说道：“我刚才不是说，有两个秘密吗，第一个秘密现在说完了。你问的这个，算是第二个秘密了。”
<br/>　　谢雨晴上前拎着耳朵骂道：“在我们这还装起来了，什么第一个秘密第二个秘密的，赶紧全都说出来！”
<br/>　　“哎呀，疼疼疼，这……真的不能说了，我也只是隐约知道一些，如果我说了，我将陷入万劫不复。”瓜瓜很是为难。
<br/>　　“为什么不能说，谁也不会说出去！”谢雨晴揪着耳朵不放，瓜瓜只好施法弹开她的手，揉着耳朵，愁眉苦脸地说道，“不是怕你们说出去，我当时立下毒誓，这件事绝对不说出去，以此设了一个诅咒，一旦说出去，诅咒达成，我会修为尽失，打回原形。”
<br/>　　“有这么严重？”谢雨晴懵懂地看着他，“有没人知道，怎么会突然被打回原形什么的？”
<br/>　　老郭说道：“这个你不懂，这是诅咒之力，一旦违背誓言，诅咒就会生成，自动降下诅咒。”转头瞟了瓜瓜一眼道，“不能说就别说了，你拣能说的说，自己想想。”
<br/>　　瓜瓜认真想了一下，道：“大家可能都错了，关于转世鬼童……嗯，不是你们想的那样。”
<br/>　　“那是什么样？”
<br/>　　“那就没法说明了，其实我也是才知道不久，正是因为这个秘密，无极鬼王才不敢动老大的。”瓜瓜吐了吐舌头，“关于这个，就别问了，我没法再说了，你们自己慢慢想，猜测真相，如果猜到了，不算我说的，不过现在别猜，你们猜不到的。”
<br/>　　老郭道：“你就先不猜了，还说第一个秘密，这个秘密，跟龙华会大伙要开除叶少阳的名籍有什么关系？”
