　　楚风眠双眼睁开，脸色凝重无比。
<br/>　　在读取了这沧海都圣子的记忆之后，楚风眠发现，如今的情况比起楚风眠想象之中的还要更糟。
<br/>　　这三大势力联合起来对付圣龙秘境，不是一朝一夕的事情了，早在是数年前，就在谋划，只是一直没有找到机会。
<br/>　　如今九域开启，一片混乱，正是让三大势力意识到了机会，打算是从这九域之中，将圣龙秘境的圣子，一举歼灭。
<br/>　　而且不止如此，如今圣龙秘境的处境，不止是有着外患，更是有着内忧，根据这沧海都圣子的记忆之中。
<br/>　　这一次三大势力出手，还是跟圣龙秘境内部的许多圣子，都给联合了起来，六绝宗，圣院宗，五灵宗的许多圣子都是他们拉拢的对象。
<br/>　　虽然到底是拉拢了多少的圣子，现在还不知道，但是只有几个内患，都要比起这三大势力还要更加棘手，难以对付。
<br/>　　圣龙秘境毕竟是有着他的底蕴，想要被外力毁灭还没有那么容易，怕的就是内部冲突。
<br/>　　现在楚风眠也有些担心五灵宗圣子一行人了。
<br/>　　炎撼世他们如今离开，多半也是遇到了一些事，一旦是五灵宗的圣子也都被三大势力歼灭的话，那楚风眠可就要落到一个孤立无援的地步了。
<br/>　　幸好的就是，楚风眠提前是将消息已经是交给了炎撼世，现在炎撼世他们应该是已经提前做好了一部分的准备，不会遭遇突然袭击。
<br/>　　不过现在三大势力，还没有开始全面的对付圣龙秘境，赤红大陆，整个归墟天的宝藏，还是他们提前的目标。
<br/>　　不过因为楚风眠在圣龙秘境太过耀眼，以至于三大势力的内部，已经是下了命令，只要是能够抓住，或者杀了楚风眠的人，都会得到大量的赏赐。
<br/>　　现在碰到三大势力的强者，楚风眠必须要小心行事。
<br/>　　“走，去收取更多的宝藏！”
<br/>　　楚风眠目光闪烁，身形一闪，冲天而起，化为了一道遁光向着赤红大陆深处飞去。
<br/>　　归根结底一切的麻烦，想要化解只有一点，便是实力，只有足够的实力才可以化解这些麻烦。
<br/>　　现在楚风眠刚刚突破境界，正是需要大量的诸天奇物宝藏，来稳定住境界，一旦是稳定住境界，楚风眠的实力还足以再强几分。
<br/>　　虽然楚风眠突破之后，还没有跟九劫古帝真正的交过手，可现在楚风眠自己感觉到，在面对九劫古帝的时候，楚风眠也有着自信，有着一战之力了。
<br/>　　要是楚风眠的实力足以轰杀九劫古帝，甚至是可以达到半步天人境那个级别的话，管他什么三大势力的阴谋，楚风眠以一人之力都足以化解。
<br/>　　归根结底，需要的还是实力，在绝对的实力面前，一切的阴谋都没有任何的意义。
<br/>　　楚风眠身形闪烁，穿梭在赤红大陆之中。
<br/>　　赤红大陆之中的宝藏，已经是开始纷纷出世，这一次九域剧变，大量的宝藏纷纷出世。
<br/>　　这赤红大陆也不例外，一眼望去，天空之上弥漫起无数霞光，这每一道霞光，都代表着一座宝藏的出世。
<br/>　　“走！”
<br/>　　楚风眠看了一眼距离最近的霞光，身形一闪直接冲了过去，一眼望去便是看见了在地上的一株血红色的圣药，这血红色之上，还带有着几分脉络纹路。
<br/>　　“脉血草，好东西！”
<br/>　　楚风眠看到这血红色的圣药，楚风眠眼神一喜。
<br/>　　脉血草，是一种灵力无比精粹的圣药，其中的灵力温和，不适合修炼，却是适合恢复伤势，巩固境界之用，对楚风眠来说正好。
<br/>　　楚风眠身形一闪，直接向着那脉血草冲了过去，出手摄取这脉血草。
<br/>　　“有圣药！”
<br/>　　突然之间，就在远方到来了三道遁光，冲了过来，看到这收取脉血草的楚风眠，突然之间一道刀光，直接斩向楚风眠的手臂。
<br/>　　直接出手，没有任何的言语。
<br/>　　在这赤红大陆之中，抢夺宝藏是无比正常的一件事，每一座宝藏，都要沾染无数的鲜血。
<br/>　　这三名遁光，看到楚风眠只有一人，便是毫不犹豫直接出手，手中刀锋斩向了楚风眠。
<br/>　　“死！”
<br/>　　楚风眠眼神冰冷无比，敢在他的手中抢夺宝藏，这三个人是在找死，这三人才不过只是七劫古帝罢了。
<br/>　　楚风眠现在杀八劫古帝，就跟捏死一只蝼蚁一样的简单，更何况是对付区区几名七劫古帝。
<br/>　　楚风眠手心一番，化为了一道龙爪当即轰然落下。
<br/>　　这向着楚风眠轰击而来的三道遁光，当即是被轰的粉碎，随后这三人的身躯，也在战龙之爪下，被无情轰杀。
<br/>　　“吞噬！”
<br/>　　楚风眠运转起吞天秘术，将这三人身上的精血全部吞噬，来巩固楚风眠的境界之用。
<br/>　　同时另一边，那脉血草也被楚风眠摄取到了手心之中，旋即楚风眠没有丝毫的犹豫，一把将这脉血草丢入到了口中。
<br/>　　楚风眠的身躯之中，吞天秘术便是当即被运转了起来，开始炼化这脉血草的药力，滚滚精粹的药力，涌入到了楚风眠的身躯之中，开始稳定楚风眠的境界。
<br/>　　这脉血草虽然只是中古级别的古圣药，但是的确是稳定境界最好之物，被楚风眠吞服炼化下去。
<br/>　　如今楚风眠身上的力量明显是变的稳定了许多，这就是脉血草的效果。
<br/>　　刚刚突破的古帝，身上的力量都会不稳定，所有无法施展出全力。
<br/>　　突破一重境界便是如此，尤其是楚风眠可是从准帝境界，直接突破到了八劫古帝的境界。
<br/>　　一下子突破了足足八重的境界，楚风眠的境界正是不稳定，这脉血草对他稳定境界，有着大用。
<br/>　　不过一株脉血草，还远远不足以让楚风眠彻底稳固境界。
<br/>　　他手心一动，那三名武者的空戒当即打开，打开空戒，琳琅满目无数至宝便是落入到了楚风眠的目光之中。
<br/>　　这三名武者，看来是在赤红大陆之中得到了不少的宝藏，楚风眠也看见了一些对他现在巩固境界有用的至宝，一把将其从空戒之中摄取了出来，被吞天秘术所吞噬。
